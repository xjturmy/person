# 个人投资判断优化 README

> 记录时间：2026-07-04  
> 当前状态：Dashboard 功能已较丰富，后续优化重心从“更多页面/更多展示”切换为“帮助个人做出更清楚、更克制、更可复盘的投资判断”。

## 核心定位

这个工具不是给别人看的投研展示页，而是个人投资副驾。

后续优化应优先回答：

- 方法用得对不对？
- 当前判断是否站得住？
- 当前价格是否符合这个投资方案？
- 当前操作是否合理？
- 什么条件出现后要承认判断失效？

少做装饰、少堆指标、少扩页面。多做结论、约束、复盘和操作校验。

## 当前共识

1. 每套投资方法只回答自己的核心问题，不能互相串味。
2. 林奇法的价格区间应放在最后综合判断中，并且只按林奇方案计算。
3. 格雷厄姆、林奇、芒格等方法可以并存，但不能在同一个最终结论里互相污染。
4. 评分只能辅助，不能代替操作判断。
5. 对个人投资来说，“是否应该买/卖/持有”比“页面是否漂亮”更重要。

## 后续优先级

### 1. 公司研究最终结论

每家公司要能一屏看清：

- 当前动作：买入 / 观察 / 持有 / 减仓 / 卖出
- 核心理由：最多 3 条
- 当前价格：便宜 / 合理 / 偏贵 / 不适合该方法判断
- 操作建议：现在做什么，不做什么
- 失效条件：什么出现后原判断作废
- 下次复查：按季度、年报或价格触发

### 2. 方法页最终判断

每个方法页的最后一页只做“方法内判断”。

- 林奇法：故事是否成立、增长是否延续、价格是否匹配增长
- 格雷厄姆法：是否足够便宜、是否有安全边际、是否适合防守型买入
- 芒格法：是否是真正优质业务、是否有长期复利能力、是否值得长期跟踪

如果方法不适合这家公司，要明确提示“此方法只能辅助，不应作为主判断”。

### 3. 操作合理性检查

每次买卖前，工具应强制帮助检查：

- 是按计划买，还是追涨？
- 是逻辑坏了卖，还是情绪恐慌？
- 当前仓位和确定性是否匹配？
- 如果继续下跌 20%，原理由是否仍成立？
- 如果上涨 30%，是否有减仓计划？
- 当前操作是否违反之前写下的卖出条件？

### 4. 复盘系统

决策记录要能回头对照：

- 当初为什么买？
- 当初认为不会发生什么？
- 现在事实是否证伪了判断？
- 盈亏来自判断正确、运气、估值变化，还是操作纪律？

## 暂缓事项

以下事情暂时不作为优先级：

- 新增更多视觉样式
- 新增更多大师名字
- 继续拆更多小 tab
- 做展示型大屏
- 为了“看起来完整”增加低价值指标

## 验收标准

一次优化是否值得做，用以下标准判断：

- 能否减少一次犹豫不决？
- 能否暴露一个方法误用？
- 能否阻止一次冲动交易？
- 能否让买入、卖出、持有理由更明确？
- 能否让 3 个月后的复盘更容易？

如果答案是否定的，就不是当前阶段的重点优化。
