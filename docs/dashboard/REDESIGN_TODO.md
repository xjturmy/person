## 📦 备选方向(暂存)

### 方向 A — 紧凑信息密度(雪球 + 东财式)

**核心元素**:顶部固定 ticker 条 + 7 横 tab + 每 tab 内"左大图(2/3)+ 右指标表(1/3)"

**优点**:
- Streamlit 复刻最简单(★★★★★),原生 `st.tabs` + `st.columns([2,1])`
- 工作量最小(1-2 天)
- 信息密度高,适合老股民/工程师

**示意布局**:


### 方向 C — 多面板拖拽(OpenBB + TradingView 式)

**核心元素**:左侧公司列表 + 主区可拖拽 grid widget(K线/分位带/对比矩阵/F-Score 时间线),全 widget 按 ticker 联动

**优点**:专业终端风格,适合重度玩家
**缺点**:
- Streamlit 原生**不支持** widget 拖拽 grid
- 需引入 `streamlit-elements` 或干脆换 Dash / Panel
- 工作量 1-2 周,偏离 Streamlit 习惯路径
