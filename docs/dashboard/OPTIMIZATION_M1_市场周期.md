## 🎯 优化目标

把 L1 市场周期 Tab 从"5 张图 + 1 个分位带"升级为**结论先行 + 三信号合成 + 自带说明**的择时驾驶舱,回答"现在是好时机吗?"

核心新增三块:
1. **综合结论 banner**(顶部三信号合成)
2. **康波周期定位卡**(萧条期中后段 · 防御主)
3. **格雷厄姆指数计算**(动态算 + 历史参照)

并对原有 5 项宏观时序加**含义说明 + 红绿灯阈值**。

---

## 🩺 当前痛点

| 现状 | 问题 |
|---|---|
| 顶部温度计 5 个徽章只有数字 + 红绿圆圈 | 用户看不懂"M2 同比 7.6% 🟢" 是什么意思 |
| 段 1 五张时序小图 | 标题只有"M2 同比",没说"是什么 / 为什么看 / 高低含义" |
| 段 2 A50 PE 分位带 | ✅ 做得好(红绿灯 + 当前分位 + 80/50/20 横线) |
| 段 3 行业热力图 | ✅ 颜色编码清晰 |
| **缺**:康波周期定位 | 知识库写了"2026 萧条期中后段",前端零展示 |
| **缺**:格雷厄姆指数 | 知识库有完整公式 + 五档评级表,前端零展示 |
| 段头标题 "段 1/2/3" | 序号无信息量,用户记不住 |
