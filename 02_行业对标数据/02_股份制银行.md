# 股份制银行

## 行业概览

股份制银行(招商/兴业/浦发/民生/中信/光大/平安/华夏/浙商等 12 家全国性)是中国银行业第二梯队,资产规模 5-10 万亿区间。核心商业模式是"息差 + 中间业务"双轮驱动 — 资产端做对公贷款/零售房贷/消费金融,负债端拼活期存款占比;ROE 12-17% 远超大行(8-12%)。零售银行典型(招行/平安)享受估值溢价。

## 周期特性

- **周期类型**:价值(林奇 bank 范畴 — 独立评分体系)
- **康波位置**:萧条期防御金融核心 — 高股息 + 低 PB 安全垫
- **典型周期长度**:跟随宏观信用周期 3-5 年 — LPR/MLF 利率传导 + 房地产景气度联动
- **现在哪个阶段**:萧条期息差承压 + 不良温和上行,但拨备充足 + 估值历史底部

## 龙头格局

- **CR3**:招商银行 / 兴业银行 / 浦发银行(总资产合计占股份行 ~45%)
- **TOP 5**:招商银行(600036,零售之王)/ 兴业银行(601166,同业之王)/ 浦发银行(600000)/ 中信银行(601998)/ 民生银行(600016)
- **格局演变**:零售银行(招行/平安)估值溢价持续;对公银行进入数字化转型期;部分中小股份行不良率上升,弱者恒弱

## 关键观察指标

- **净息差 NIM**(招行 2.0%+ 优秀,行业平均 1.7-1.9%,< 1.5% 预警)
- **不良贷款率**(< 1.5% 健康,> 2% 谨慎;对公 vs 零售结构看)
- **拨备覆盖率**(招行 400%+ 极厚,行业 250-300% 充足,< 200% 警惕)
- **核心一级资本充足率 CET1**(> 10% 健康,充足资本支撑分红)
- **LPR / 房贷利率走势**(资产端定价基准,降息周期息差压力)
- **零售/对公业务结构**(零售占比 > 50% 给估值溢价)

## 投资逻辑

**适合买入的时点**:PB < 0.7 + 股息率 > 5% + 不良企稳 + 拨备充足 + 经济见底预期。格雷厄姆 + 彼得林奇 bank 框架要求安全垫(PB 安全边际)+ 资产质量(不良/拨备)+ 分红可持续性。

**回避时点**:不良率连续 2 季度上升 + 拨备覆盖率快速下降 + 房地产风险敞口高 + LPR 大幅下调(息差收窄)。鲁政委萧条期框架将银行纳入"防御金融",但弱质银行(高不良 + 高地产敞口)是价值陷阱。
