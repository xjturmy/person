# 中药

## 行业概览

中药是中国独有的政策保护型高毛利消费医药赛道,核心商业模式是"独家品种 + 老字号品牌 + 中药资源稀缺性"三重护城河。中药饮片/中成药 OTC 龙头(片仔癀/云南白药/同仁堂)毛利率 40-80%,ROE 长期 15-25%,具备奢侈品定价权(片仔癀单粒提价 20 次)。本质是"含药效的文化消费品",受益于国家中医药保护政策(《中医药法》+ 医保倾斜)与人口老龄化。

## 周期特性

- **周期类型**:防御(林奇 stalwart 范畴)
- **康波位置**:萧条期消费防御 + 老龄化长逻辑 — 必选医药消费 + 政策免疫 + 强现金流
- **典型周期长度**:3-5 年(政策周期 + 原料涨价周期);2017-2019 OTC 提价潮、2021-2023 配方颗粒集采压制、2024-2026 创新中药+基药目录催化
- **现在哪个阶段**:参见 Dashboard 实时判定(估值分位 + 政策催化 + 原料价格趋势)

## 龙头格局

- **CR3**:片仔癀 + 云南白药 + 同仁堂(品牌中药 OTC 三强)
- **TOP 5**:片仔癀(600436,奢侈品定价)/ 云南白药(000538,口腔+创可贴)/ 同仁堂(600085,老字号集团)/ 华润三九(000999,感冒灵+OTC 王者)/ 东阿阿胶(000423,阿胶补益)
- **格局演变**:独家品种+保密配方龙头集中度持续提升;配方颗粒(中国中药/红日)受集采压制利润;创新中药(以岭/天士力)受益审批加速

## 关键观察指标

- **片仔癀终端零售价**(760 元/粒 = 中药奢侈品景气温度计,跌破出厂价警惕)
- **中药材价格指数**(康美·中国中药材价格指数,原料涨价压制毛利)
- **国家基药目录与医保谈判**(进基药=放量起点,2025 版重点关注)
- **配方颗粒集采进度**(已 3 省试点,降幅 30-40%,影响中国中药/红日)
- **OTC 渠道库存**(连锁药店动销 + 电商占比,Q1+Q4 旺季)
- **政策催化**(《中医药振兴发展重大工程实施方案》+ 医保倾斜)

## 投资逻辑

**适合买入的时点**:估值分位 < 30%(PE-TTM < 25 倍) + 原料价格企稳 + 政策催化(基药/医保) + 提价预期。林奇 stalwart 框架要求 PE 与 EPS 增速接近(PEG ~1),片仔癀历史 PEG 中枢 1.0-1.5,云南白药/华润三九 PEG 0.8-1.2 更具性价比。

**回避时点**:估值分位 > 80%(片仔癀曾 PE 90 倍) + 配方颗粒类集采扩面 + 原料持续暴涨 + 销售费用率失控(同仁堂历史教训)。鲁政委萧条期框架将品牌中药纳入"防御核心",但需警惕"奢侈品叙事"破灭风险。
