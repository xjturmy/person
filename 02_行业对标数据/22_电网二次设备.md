# 电网二次设备

## 行业概览

电网二次设备包括继电保护/变电站自动化/调度自动化/配网自动化/电能计量等,是电网"神经中枢"。国电南瑞作为国网系绝对龙头(国网信通+南瑞集团双注入)市占 30%+,四方/许继/平高电气分守细分领域。三大成长极:① 特高压加速建设(国网+南网 2024-2027 年 5800 亿投资计划),② 配网智能化(国网"双高"配电网三年行动),③ 储能调度+虚拟电厂(新型电力系统刚需)。央企背景+股息率 2-3% 提供防御属性。

## 周期特性

- **周期类型**:成长(林奇 fast_grower,叠加 stalwart 防御属性)
- **康波位置**:第六次康波(AI+新能源)电力基础设施核心,与电网设备/新能源车并列三大成长极
- **典型周期长度**:5 年(电网规划五年期)— "十四五"(2021-2025)聚焦特高压,"十五五"(2026-2030)聚焦配网+储能调度
- **现在哪个阶段**:参见 Dashboard 实时判定(国网投资额+在手订单+招标节奏)

## 龙头格局

- **CR3**:国电南瑞+南瑞继保(未上市)+许继电气 合计 50%+ 二次设备份额
- **TOP 4**:国电南瑞(600406)绝对龙头综合平台 / 四方股份(601126)继电保护民营龙头 / 许继电气(000400)特高压直流换流阀龙头 / 平高电气(600312)高压开关+GIS 龙头
- **格局演变**:国网系(南瑞/许继/平高)央企背景持续受益国网投资;民营(四方)在配网+海外突围;储能调度+虚拟电厂催生新玩家

## 关键观察指标

- **国网/南网年度投资计划**(2024 国网 6000 亿+ 创新高,直接驱动设备订单)
- **特高压招标节奏**(直流换流阀/GIS 中标公告,核心订单前瞻)
- **配网投资占比**(国网配网投资占比从 50% 升至 60%+,智能化提速)
- **储能/虚拟电厂订单**(新型电力系统催化,二次设备增量市场)
- **在手订单/合同负债**(领先营收 2-3 季度,景气前瞻指标)

## 投资逻辑

**适合买入的时点**:国网投资同比加速 + 特高压招标密集落地 + 在手订单创新高 + 估值分位 < 40%(PE-TTM < 25 倍)+ 股息率 > 2%。林奇 fast_grower 框架要求营收 15%+/PEG < 1.5,南瑞历史 PEG 中枢 1.0-1.3,叠加央企股息防御属性。

**回避时点**:国网投资同比放缓 + 招标节奏延迟 + 估值分位 > 80% + 股息率 < 1.5%。第六次康波长期看好新型电力系统建设,电网二次设备是"卖铲人"逻辑,确定性高于发电侧。
