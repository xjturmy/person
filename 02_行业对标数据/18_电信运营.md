# 电信运营

## 行业概览

电信运营是国民经济基础设施,核心商业模式是"个人移动业务 + 家庭宽带 + 政企云业务"三轮驱动。运营商已走出 4G→5G 巨额资本开支高峰(资本开支/收入比从 25% 降至 18%),5G ARPU 企稳 + 算力网络(IDC + 云)成第二增长曲线;移动云、天翼云收入年化 +50%,中国电信云已进国内 IaaS 前四。本质是"类公用事业 + 数字基建" — 现金流稳定、分红率 70-75%、股息率 5-7%,低利率环境下类债资产典范。

## 周期特性

- **周期类型**:防御(林奇 slow_grower 范畴,收入 +5% / 利润 +8% 稳定增长)
- **康波位置**:康波萧条/复苏期防御核心 — 类公用事业 + 高股息 + 政策友好
- **典型周期长度**:7-10 年(技术代际周期 3G→4G→5G→6G)— 2019-2021 5G 建设资本开支期、2022 起进入收获期、2026+ 6G 预研启动
- **现在哪个阶段**:参见 Dashboard 实时判定(ARPU 同比 + 资本开支/收入 + 股息率)

## 龙头格局

- **CR3**:中国移动 + 中国电信 + 中国联通(三足鼎立,100% 行业份额)
- **TOP 5**:中国移动(600941)/ 中国电信(601728)/ 中国联通(600050)/ 中兴通讯(000063)/ 紫光股份(000938)
- **格局演变**:三大运营商共建共享 5G 基站节省千亿资本开支;移动云 + 天翼云成关键看点(政企数字化主战场);中国移动 A+H 双重估值差从 50% 收敛到 25%,A 股折价空间收窄

## 关键观察指标

- **移动 ARPU**(移动 51 元 / 电信 46 元 / 联通 44 元,环比+0.5% 健康,负增长警报)
- **5G 套餐渗透率**(移动 80% / 电信 75% / 联通 72%,渗透率提升期 ARPU 自然抬升)
- **资本开支/收入比**(从 5G 高峰 25% 降至 18% 是核心利润释放逻辑,< 15% 自由现金流爆发)
- **政企云业务收入增速**(移动云 + 天翼云 + 联通云 +30~50% 是第二曲线,占收入 10%+)
- **股息率 vs 10 年国债利差**(中移动股息率 5.5% vs 国债 2.0% = 利差 3.5pp,类债配置价值)

## 投资逻辑

**适合买入的时点**:股息率 > 6% + PB < 1.2 + 资本开支同比下降 + 云业务增速 > 30%。林奇 slow_grower 框架对运营商最贴切 — 不追求高增长,追求"稳定分红 + 估值便宜",PE 10-13 倍区间是黄金买点。中移动 H 股股息率长期 7-8%,可叠加港股通配置增厚收益。

**回避时点**:6G 资本开支启动预期 + ARPU 负增长 + 互联网应用替代加速(微信通话替代基础语音)。鲁政委防御框架将电信运营归为"萧条期持仓核心",但需警惕 AI 算力投入带来的资本开支再扩张可能(2025 三家 AI 算力投入合计 800 亿)。
