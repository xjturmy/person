# 白酒

## 行业概览

白酒是中国特色高毛利消费品,核心商业模式是"品牌 + 渠道 + 老酒库存"三重护城河。高端白酒(茅台/五粮液/泸州老窖)毛利率 85%+,ROE 长期 25%+,自由现金流极强;次高端(汾酒/古井/水井坊)受益消费升级与全国化扩张。本质是"含酒精的奢侈品",定价权来自品牌历史与稀缺产能。

## 周期特性

- **周期类型**:防御(中长期 stalwart 林奇范畴)
- **康波位置**:萧条期防御核心 — 必选消费 + 高 ROE + 强现金流
- **典型周期长度**:5-7 年(渠道库存周期)— 2012-2014 三公消费冲击,2015-2018 复苏,2019-2021 高端化牛市,2022-2025 调整去库存
- **现在哪个阶段**:参见 Dashboard 实时判定(估值分位 + 1y 涨跌 + ROE 趋势)

## 龙头格局

- **CR3**:茅五洋(贵州茅台 + 五粮液 + 洋河)合计 ~50% 行业利润
- **TOP 5**:贵州茅台(600519)/ 五粮液(000858)/ 山西汾酒(600809)/ 泸州老窖(000568)/ 洋河股份(002304)
- **格局演变**:高端化趋势确立,品牌集中度持续上升;次高端 200-600 元价位带竞争激烈;光瓶酒/小酒种细分赛道偶有黑马

## 关键观察指标

- **飞天茅台批价**(2499 出厂 / 2200-2500 终端 = 行业景气温度计,跌破 2200 警惕)
- **渠道库存周转**(库存/月销 < 2 月健康,> 3 月预警)
- **消费税政策**(2017 后未调整,但悬剑;若加税生产环节会压制龙头利润)
- **春节/中秋销售**(白酒两大旺季,Q1+Q3 营收占全年 65%+)
- **高端 vs 次高端价差**(价差收窄=次高端机会,扩大=高端独享)

## 投资逻辑

**适合买入的时点**:批价企稳回升 + 渠道库存出清 + 估值分位 < 30%(PE-TTM < 25 倍)+ 消费政策预期友好。林奇 stalwart 框架要求 PE 与 EPS 增速接近(PEG ~1),茅台历史 PEG 中枢 1.0-1.5。

**回避时点**:批价持续下跌 + 库存堆积 + 政策传闻(消费税/限制三公)+ 估值分位 > 80%。鲁政委萧条期框架将白酒纳入"防御核心",但需警惕"贵就是贵"的估值陷阱(如 2021 年 PE 60 倍)。
