# 汽车零部件

## 行业概览

汽车零部件是汽车产业链中游核心环节,涵盖三电系统、底盘、内外饰、玻璃、轻量化、智驾域控、线控转向等细分。商业模式是"绑定整车客户 + 单车价值量 + 平台化扩张"三重驱动。新能源车单车价值量较燃油车提升 30-50%,叠加智能化(智驾域控/线控底盘)再上一台阶。本质是 fast_grower — 受益新能源车渗透率 + 国产替代 + 智能化三大长逻辑,特斯拉/比亚迪/华为系供应链为黄金赛道。

## 周期特性

- **周期类型**:fast_grower(林奇高速成长股,新能源/智能化细分)+ 复苏期进攻方向
- **康波位置**:第六次康波智能交通+能源革命交汇方向,长期成长
- **典型周期长度**:与新能源车渗透率曲线挂钩(2-4 年小周期);2020-2022 渗透率爆发,2023-2024 价格战压力,2025+ 智能化新一轮β
- **现在哪个阶段**:参见 Dashboard 实时判定(新能源车渗透率 + 龙头单车价值量 + 毛利率趋势)

## 龙头格局

- **CR5**:拓普(特斯拉/华为)/ 华域(上汽集团)/ 福耀(全球玻璃)/ 伯特利(线控制动)/ 保隆(空气悬挂),细分领域龙头格局清晰
- **TOP 5**:拓普集团(601689)/ 华域汽车(600741)/ 福耀玻璃(600660)/ 伯特利(603596)/ 保隆科技(603197)
- **格局演变**:特斯拉/比亚迪/华为系供应链成长性最佳;一体化压铸、线控底盘、智驾域控、空气悬挂等新品类持续扩容;福耀玻璃汽车玻璃全球份额 35%+,稳定成长 stalwart 属性

## 关键观察指标

- **新能源车月度渗透率**(> 50% 进入下半场,智能化接棒电动化)
- **龙头单车价值量 ASP**(拓普 / 伯特利新能源车单车配套金额 YoY)
- **大客户销量**(特斯拉/比亚迪/华为系/理想/小米月度交付)
- **毛利率**(价格战压力下毛利率守住 20%+ 为优质标志)
- **PEG 估值**(fast_grower 框架,PEG < 1 进攻、PEG > 1.5 谨慎)

## 投资逻辑

**适合买入的时点**:新能源车渗透率持续提升 + 龙头单车价值量上行 + 智能化新品(域控/线控)放量 + PEG < 1.2。林奇 fast_grower 框架要求 20%+ 增速且 PEG 合理,拓普/伯特利历史 PEG 中枢 0.8-1.3。绑定特斯拉/华为/比亚迪供应链优选。

**回避时点**:整车价格战传导至零部件毛利率持续下滑 + 大客户销量不及预期 + 单一客户依赖过高(>40%)+ PEG > 2。注意华域等传统燃油车配套商面临转型压力,拆解新能源业务占比。
