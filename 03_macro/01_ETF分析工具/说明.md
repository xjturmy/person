## 📊 ETF 分析的价值

### 为什么用 ETF 做行业分析？

1. **代表性** — ETF 汇集了某行业的头部公司，代表行业走势
2. **实时性** — ETF 价格实时反映市场情绪
3. **简化性** — 避免个股选择困难，关注行业大趋势
4. **资金面** — ETF 成交量反映主力资金流向

### 与个股分析的互补关系


### 分析框架

每个 ETF 分析应包括：

1. **基本信息** — ETF 代码、跟踪指数、主要成分股
2. **行业前景** — 该行业的发展阶段和成长空间
3. **资金面** — ETF 近期资金流向、成交量变化
4. **估值水位** — ETF 所代表行业的估值水位
5. **轮动阶段** — 该行业在周期中的位置
6. **操作建议** — 基于上述分析的建议

## 📡 数据来源与获取

### 理杏仁 API 支持情况

✅ **可用**:
- 指数成分股 API — 获取 ETF 跟踪指数的成分股列表
- 公司基本面 API — 获取成分股的财务指标

❌ **不可用**:
- 基金概况 API
- 基金成分股 API

**详见**: [理杏仁ETF数据支持调查](理杏仁ETF数据支持调查.md)

### 推荐的数据获取流程

1. **维护 ETF 配置** — JSON 格式存储 ETF 基本信息与跟踪指数
2. **获取成分股** — 通过指数代码调用指数成分股 API
3. **获取财务数据** — 通过股票代码调用公司基本面 API
4. **汇总分析** — 整合成分股数据生成 ETF 分析报告

## 🔄 更新维护

### 数据来源
- ETF 实时数据：天天基金、同花顺等
- 行业数据：研究报告、新闻资讯

### 维护流程

2. 跟踪行业动态和政策变化

4. 整理成分股数据到 02_companies/

## 🎯 应用场景

### 场景 1：行业轮动择时
```
1. 查看各主题 ETF 的资金流向
2. 观察 ETF 价格与估值变化
3. 判断该行业是否处于吸纳期、上升期还是高位
4. 决定是否配置该行业
```

### 场景 2：个股选择参考
```
1. 选定投资主题（如新能源）
2. 查看该主题 ETF 的主要成分股
3. 结合 02_companies/ 中对应公司的详细分析
4. 在行业内精选个股
```

### 场景 3：资产配置
```
1. 监控 4 个主题 ETF 的相对强弱
2. 调整持仓配置以适应行业轮动
3. 在不同主题间分散投资风险
```
