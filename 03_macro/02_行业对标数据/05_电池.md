# 电池

## 行业概览

电池行业由动力电池(新能源车)+ 储能电池两大需求曲线驱动,中国占全球电池产能 70%+。锂电池主流技术路线分三元(高镍三元)+ 磷酸铁锂(LFP),中国 LFP 性价比优势已扩散全球。固态电池/钠离子是下一代技术,2027-2030 商业化窗口。商业模式核心是"产能 × 良率 × 海外份额"。

## 周期特性

- **周期类型**:成长(林奇 fast_grower 范畴)+ 周期(产能扩张-去化周期)
- **康波位置**:第六次康波能源革命方向 — 与 AI/生物科技并列三大新动能
- **典型周期长度**:产能周期 3-4 年(扩产 → 过剩 → 去化 → 新平衡)
- **现在哪个阶段**:产能过剩末段 + 价格战进入下半场,龙头份额集中度加速提升

## 龙头格局

- **CR3**:宁德时代 / 比亚迪(动力)/ LG新能源(韩,海外)合计全球份额 65%+
- **TOP 5**:宁德时代(300750,全球第一)/ 比亚迪(002594,垂直整合)/ 中创新航(港股)/ 亿纬锂能(300014)/ 国轩高科(002074)
- **格局演变**:中国厂商海外份额提升(欧洲建厂 + 美国 IRA);储能赛道宁德/比亚迪/亿纬三足鼎立;二三线产能加速出清

## 关键观察指标

- **电池装机量**(月度国内+海外,关注同比/环比)
- **碳酸锂价格**(成本核心,< 8 万/吨利好下游,> 15 万压制利润)
- **储能招标量与价格**(年化 GWh 级别需求 + 报价回升)
- **海外市场份额**(宁德欧洲 + 美国 IRA 友好通道,40% 海外营收)
- **龙头毛利率**(宁德 20%+ 健康,二三线 5-10% 挣扎)
- **固态电池/钠电池产业化进度**

## 投资逻辑

**适合买入的时点**:碳酸锂价格见底 + 海外资本开支启动 + 储能放量 + 估值分位 < 30% + 龙头毛利企稳。林奇 fast_grower 要求 EPS 增速 20%+ 与 PEG < 1.5。

**回避时点**:产能继续过剩 + 海外政策风险(IRA/欧洲 CBAM)+ 价格战恶化 + 中小厂技术替代风险。电池行业波动大,建议聚焦龙头(宁德/比亚迪)或通过电池 ETF 分散个股风险。
