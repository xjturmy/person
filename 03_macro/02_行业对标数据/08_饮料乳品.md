# 饮料乳品

## 行业概览

饮料乳品行业是必需消费防御核心,乳品双寡头格局(伊利/蒙牛)合计市占 50%+。核心商业模式是"品牌 + 渠道 + 上游奶源"三位一体。常温白奶基本盘稳固,低温鲜奶 + 高端有机奶 + 婴幼儿配方是增长引擎。原奶价格周期 3-4 年(牛群养殖周期),奶价低位时下游毛利改善;包装饮料(矿泉水/茶饮)消费升级带动农夫山泉/东鹏特饮等龙头崛起。

## 周期特性

- **周期类型**:防御(林奇 stalwart 范畴)
- **康波位置**:萧条期防御消费核心 — 必选消费 + 稳定现金流 + 高股息
- **典型周期长度**:原奶周期 3-4 年(牛群养殖 + 牧场扩产-去化)
- **现在哪个阶段**:原奶价格底部 + 渠道库存出清 + 高端化深化

## 龙头格局

- **CR3**:伊利 / 蒙牛 / 光明(乳品三大全国性品牌)
- **TOP 5**:伊利股份(600887,乳品龙头)/ 蒙牛乳业(02319,港股)/ 光明乳业(600597)/ 新乳业(002946)/ 妙可蓝多(600882,奶酪)
- **格局演变**:常温白奶寡头格局稳定;低温鲜奶区域品牌(光明/三元/燕塘)突围;婴幼儿配方国产替代(飞鹤/伊利/蒙牛 vs 海外品牌);奶酪/特殊乳品细分赛道高增

## 关键观察指标

- **原奶价格**(月度国家统计局,< 3.5 元/kg 利好下游,> 4.2 元压制毛利)
- **高端白奶/低温奶占比**(伊利金典/蒙牛特仑苏占营收 40%+ 高端化典范)
- **渠道库存与终端动销**(尼尔森数据,渠道周转 30 天健康)
- **出生率**(婴幼儿配方关键变量,2016 全面二孩高峰后持续下降)
- **海外乳企并购整合**(伊利澳优 + 蒙牛贝拉米布局)
- **包装饮料新增长点**(矿泉水/无糖茶饮/即饮咖啡)

## 投资逻辑

**适合买入的时点**:原奶价格底部 + 渠道库存出清 + 高端化提速 + 估值分位 < 40% + 股息率 > 3%。林奇 stalwart 框架要求稳健增长(EPS 8-12%)+ 合理估值(PEG ~1)+ 现金流稳定。

**回避时点**:原奶价格快速上涨 + 出生率持续下滑(婴幼儿配方逆风)+ 海外品牌价格战 + 估值过高(PE > 25)。乳品行业格局已稳定,适合作为防御层底仓长期持有 + 股息再投资,但需警惕食品安全黑天鹅(三聚氰胺历史教训)。
