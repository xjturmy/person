# 通信设备

## 行业概览

通信设备行业当前由两条主线驱动:① **AI 算力网络硬件**(光模块/交换机/CPO)— 海外 hyperscaler 资本开支拉动 800G/1.6T 光模块出货爆发,中际旭创/光迅/新易盛全球份额 50%+;② **5G/卫星互联网**(基站/射频/天线)— 国内运营商集采 + 卫星互联网新机遇。核心商业模式是"技术迭代 + 供应链卡位 + 客户绑定"。

## 周期特性

- **周期类型**:成长(林奇 fast_grower 范畴)+ 周期(资本开支周期)
- **康波位置**:第六次康波 AI 算力基础设施 — 第六次康波核心驱动力之一
- **典型周期长度**:资本开支周期 2-3 年,产品迭代周期 1-2 年
- **现在哪个阶段**:AI 算力资本开支爆发期(2024-2026)+ 800G 放量 + 1.6T 蓄势

## 龙头格局

- **CR3**:中际旭创 / 新易盛 / 光迅科技合计 800G 光模块全球份额 60%+
- **TOP 5**:中际旭创(300308,800G 全球龙头)/ 新易盛(300502)/ 光迅科技(002281)/ 中兴通讯(000063,综合通信)/ 烽火通信(600498)
- **格局演变**:光模块向 1.6T/CPO 演进 + 国产硅光突破;5G 基站集采价格触底;卫星互联网处早期布局阶段

## 关键观察指标

- **海外 hyperscaler 资本开支**(Meta/MSFT/AMZN/GOOG 季度 capex 同比)— 关键先行指标
- **800G/1.6T 光模块出货节奏**(中际旭创单季出货量 vs 上季)
- **中国移动/电信集采价格**(国内基础设施需求底)
- **国产 GPU/网络芯片渗透**(华为昇腾 / 海光 / 燧原)
- **库存与订单可见度**(在手订单/季度收入比 > 1.5x 健康)
- **毛利率**(中际旭创 30%+ 优秀,行业平均 20-25%)

## 投资逻辑

**适合买入的时点**:海外 capex 上修 + 800G/1.6T 节点切换 + 估值分位 < 50%(AI 板块波动大不好等到 < 30%)+ 国产替代订单落地。林奇 fast_grower 要求 EPS 增速 30%+ 与 PEG < 1.5,光模块龙头 2024-2026 EPS 增速可能超 50%。

**回避时点**:海外 capex 见顶下修 + 中美贸易/出口管制升级 + 1.6T 切换延迟 + 估值过热(PEG > 2)。AI 板块情绪化波动大,需要止盈纪律,建议分批进入或通过通信 ETF 平滑波动。
