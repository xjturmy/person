# 油气开采

## 行业概览

油气开采是全球能源核心,核心商业模式是"上游勘探开发 + 中游管输 + 下游炼化销售"一体化。国内三桶油垄断上游资源(中石油、中石化、中海油),中海油聚焦海上低成本油气(桶油成本 < 30 美元),弹性最大;中海油服等油服公司随资本开支周期波动。本质是"宏观地缘 + 美元 + 全球需求"三因子共振的强周期品种,WTI 油价 1 美元波动 = 中海油 EPS 1.5% 弹性。

## 周期特性

- **周期类型**:周期(林奇 cyclical 教科书品种,大宗商品 beta)
- **康波位置**:康波萧条期资源品避险 + 复苏期需求复苏双重驱动
- **典型周期长度**:5-7 年(资本开支 + OPEC+ 减产周期)— 2014-2016 页岩油冲击大熊、2020-2022 俄乌冲突牛、2024-2025 高位震荡
- **现在哪个阶段**:参见 Dashboard 实时判定(WTI 油价 + 美元指数 + 三桶油 PB)

## 龙头格局

- **CR3**:中石油 + 中石化 + 中海油(合计 ~85% 国内油气产量)
- **TOP 5**:中国石油(601857)/ 中国海油(600938)/ 中国石化(600028)/ 中海油服(601808)/ 海油工程(600583)
- **格局演变**:中海油因低成本 + 高分红(股息率 7-9%)成最受机构追捧品种;中石油加大天然气产能对冲油价;中石化炼化业务受成品油消费见顶压制;民营油服(杰瑞、博迈科)在页岩气服务有差异化机会

## 关键观察指标

- **布伦特/WTI 油价**(70-85 美元 = 平衡区间,< 60 OPEC+ 减产 / > 90 需求破坏)
- **OPEC+ 月度产量决议**(沙特 + 俄罗斯减产意愿 = 油价底部支撑核心)
- **美国页岩油钻机数 + 库欣库存**(钻机 < 500 台 = 供给收缩,Cushing 库存 < 3000 万桶 = 偏紧)
- **三桶油桶油成本 vs 现货价**(中海油 28 美元 / 中石油 50 美元 / 现货 75 美元 = 利润空间)
- **地缘事件**(俄乌、中东、伊朗制裁,事件驱动 20% 短期波动空间)

## 投资逻辑

**适合买入的时点**:WTI 油价 < 65 美元 + OPEC+ 减产预期 + 三桶油 PB < 1.0 + 股息率 > 6%。林奇 cyclical 框架对油气适用 — 价格周期顶部 PE 低反而是卖点(2022Q2 中石油 PE 7 倍是顶);周期底部高 PE 反而是买点(2020Q1)。中海油可作"高股息 + 资源弹性"组合配置。

**回避时点**:WTI > 95 美元持续 + 需求破坏信号(美国汽油消费下滑)+ 美元指数走强 + 全球衰退预期。鲁政委周期框架将油气归为"复苏期超配、繁荣末期减持",新能源车渗透率年化 +10pp 长期压制成品油需求曲线。
