# 工业自动化

## 行业概览

工业自动化是制造业升级核心赛道,产品涵盖伺服系统/PLC/变频器/工业机器人/控制器,下游覆盖新能源车电驱、3C 电子、锂电设备、光伏设备、机床、半导体等。核心商业模式是"技术 + 渠道 + 行业 know-how"三重壁垒,本质是国产替代驱动的 fast_grower 赛道 — 长期对标西门子/三菱/安川/发那科,国产化率从 30% 向 50%+ 提升。汇川技术为绝对龙头,2018-2024 营收复合增速 35%+,ROE 长期 20%+。

## 周期特性

- **周期类型**:fast_grower(林奇高速成长股) + 鲁政委复苏期进攻方向
- **康波位置**:第六次康波信息技术+智能制造分支,长期成长主线
- **典型周期长度**:与制造业 PMI 同步(2-3 年小周期);2020-2021 爆发,2022-2023 高位震荡,2024-2025 受地产/消费拖累调整
- **现在哪个阶段**:参见 Dashboard 实时判定(PMI + 通用自动化订单 + 龙头营收 YoY)

## 龙头格局

- **CR3**:汇川(通用自动化国内 #1)+ 信捷电气(PLC 国产龙头)+ 雷赛智能(运控国产龙头),通用自动化国产 CR3 ~45%
- **TOP 4**:汇川技术(300124)/ 麦格米特(002851)/ 信捷电气(603416)/ 雷赛智能(002979)
- **格局演变**:汇川全品类拓展(伺服→PLC→机器人→新能源车电驱);新能源车电驱业务已成第二增长极;人形机器人/服务机器人为长期期权;国产替代外资份额持续提升

## 关键观察指标

- **制造业 PMI**(> 50 扩张 = 自动化订单回暖,< 49 持续收缩需警惕)
- **通用自动化市场 YoY**(MIR/睿工业月度数据,领先龙头营收 1-2 季度)
- **新能源车电驱配套量**(汇川单车价值量 + 新势力客户结构)
- **国产化率**(伺服国产份额 > 50% / PLC > 30% 是关键里程碑)
- **PEG 估值**(林奇 fast_grower 框架,PEG < 1 进攻、PEG > 1.5 谨慎)

## 投资逻辑

**适合买入的时点**:PMI 站上 50 + 通用自动化订单 YoY 转正 + 新能源车电驱放量 + 龙头 PEG < 1.2。林奇 fast_grower 框架要求营收/利润增速 20%+ 同时 PEG 合理,汇川历史 PEG 中枢 1.0-1.5。国产替代+智能制造双逻辑长期不变。

**回避时点**:PMI 持续 < 49 + 制造业资本开支收缩 + 新能源车价格战拖累配套商利润 + PEG > 2。注意 fast_grower 业绩失速即"杀估值+杀业绩"双重打击(类似 2022 工控调整)。
