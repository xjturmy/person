# 证券

## 行业概览

证券业是资本市场核心中介,核心商业模式是"经纪 + 投行 + 自营 + 资管 + 信用"五大业务条线协同。经纪佣金率长期下行(万 2.5 → 万 1),自营成第一大利润来源(占比 35%+),投行受注册制 + IPO 节奏影响波动剧烈。本质是"权益市场的杠杆放大器",ROE 与沪深 300 走势高度同步,典型 beta > 1.3 的高弹性赛道。

## 周期特性

- **周期类型**:周期(林奇 cyclical 范畴,与 A 股市场 beta 强挂钩)
- **康波位置**:复苏 → 繁荣期最佳配置,危机/萧条期回避
- **典型周期长度**:3-5 年(与 A 股牛熊周期同步)— 2014-2015 杠杆牛、2019-2021 慢牛、2024Q4 政策牛启动
- **现在哪个阶段**:参见 Dashboard 实时判定(沪深 300 涨跌 + 日均成交额 + 估值分位)

## 龙头格局

- **CR3**:中信证券 + 华泰证券 + 国泰君安(合计 ~25% 行业净利润)
- **TOP 5**:中信证券(600030)/ 华泰证券(601688)/ 东方财富(300059)/ 国泰君安(601211)/ 招商证券(600999)
- **格局演变**:头部券商通过并购加速集中(国泰君安 + 海通合并 2024 落地);东方财富以互联网模式抢占零售流量,经纪市占率突破 5%;中小券商被挤压向区域 + 特色化求生

## 关键观察指标

- **A 股日均成交额**(> 1 万亿 = 景气高位 / < 6000 亿 = 萧条,直接驱动经纪+自营)
- **两融余额**(> 1.6 万亿 = 风险偏好高 / < 1.2 万亿 = 谨慎,信用业务利息收入晴雨表)
- **IPO 月度发行规模**(注册制+全面注册制后波动放大,30 亿/月以下=投行寒冬)
- **沪深 300 一年涨跌**(beta 1.3 倍放大,> +20% 牛市行情 / < -15% 熊市冰点)
- **PB 估值分位**(行业 PB 1.2-2.5 倍区间,< 1.3 历史低位 / > 2.0 过热)

## 投资逻辑

**适合买入的时点**:沪深 300 PB 分位 < 30% + 日均成交额底部回升 + 政策催化(降印花税/活跃资本市场)+ 券商 PB 分位 < 20%。林奇 cyclical 框架强调"在最难看的时候买入",证券股 PB 1.3 倍以下 + 市场冰点是经典左侧机会(2018Q4、2024Q3)。

**回避时点**:沪深 300 PE 分位 > 70% + 日均成交额 > 1.5 万亿持续 + 券商 PB > 2.5 倍。鲁政委周期框架将证券归为"繁荣期减仓品种" — 行情高潮时业绩兑现后会快速回吐,2015Q3、2021Q1 都是教科书顶部。
