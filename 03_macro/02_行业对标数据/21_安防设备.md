# 安防设备

## 行业概览

安防设备从传统视频监控走向 AIoT 智能物联,商业模式是"硬件+软件+解决方案"一体化。海康+大华双寡头格局稳固,全球市占率超 40%,毛利率 40%+,ROE 长期 15-20%,自由现金流强劲。第一曲线政府/企业级监控趋于饱和增速放缓,第二曲线分拆萤石(C 端智能家居)/海康机器人/海康微影/海康汽车电子打开 AIoT 新空间。海外业务受美国实体清单/印度限制冲击,内销+一带一路成对冲。

## 周期特性

- **周期类型**:防御(林奇 stalwart,稳健成长型蓝筹)
- **康波位置**:萧条期防御补充 — 高 ROE + 强现金流 + 股息率 3-4%
- **典型周期长度**:与政府/企业 IT 资本开支强相关 5-7 年;AIoT 第二曲线进入 3-4 年成长期
- **现在哪个阶段**:参见 Dashboard 实时判定(创新业务营收占比+海外业务恢复+订单节奏)

## 龙头格局

- **CR3**:海康+大华+宇视 合计国内 60%+ 份额
- **TOP 3**:海康威视(002415)绝对龙头市占 30%+ / 大华股份(002236)国内第二全球第四 / 千方科技(002373)智慧交通+ICT 综合
- **格局演变**:从硬件竞争走向"AI+大模型+场景"竞争,海康观澜大模型+大华星汉大模型加速落地;创新业务(萤石/机器人/汽车电子)分拆/独立运营释放估值

## 关键观察指标

- **创新业务营收占比**(海康 萤石+机器人+微影+汽车 占比 25%+,衡量第二曲线)
- **海外业务恢复**(实体清单+印度限制后海外营收同比,2024 起触底回升)
- **政府公共/EBG/SMBG/海外**(海康四大事业群结构,EBG 企业级是 AI 主战场)
- **存货周转**(行业景气温度计,< 120 天健康)
- **股息率**(海康历史 2-3%,大华 3-4%,防御属性指示器)

## 投资逻辑

**适合买入的时点**:政府/企业 IT 资本开支回暖 + 创新业务放量 + 海外业务确认改善 + 估值分位 < 35%(PE-TTM < 20 倍)+ 股息率 > 3%。林奇 stalwart 框架要求 PEG ~1 且业绩稳定,海康历史 PEG 中枢 1.0-1.3。

**回避时点**:海外管制升级 + 政府订单延迟 + 创新业务不及预期 + 估值分位 > 75%。鲁政委萧条期防御框架将高 ROE+高股息蓝筹作为底仓,海康/大华符合标准。
