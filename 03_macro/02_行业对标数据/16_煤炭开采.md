# 煤炭开采

## 行业概览

煤炭是中国能源压舱石,核心商业模式是"资源禀赋 + 长协合同 + 一体化运输"三位一体。动力煤(电煤)走长协价(2024 长协 675 元/吨)、焦煤随钢铁需求剧烈波动;龙头通过煤电铁一体化(神华模式)平滑周期。本质是"高股息防御型周期股"— 资本开支收缩 + 行业产能去化背景下,头部公司现金流极强、分红率 60-75%,在低利率环境成为类债资产。

## 周期特性

- **周期类型**:周期(林奇 cyclical 经典品种,叠加近年高分红防御属性)
- **康波位置**:康波萧条/复苏初期资源品轮动 + 高股息防御双重逻辑
- **典型周期长度**:3-4 年(库存+宏观需求周期)— 2016-2018 供给侧改革牛、2021-2022 能源危机牛、2023-2024 高位震荡
- **现在哪个阶段**:参见 Dashboard 实时判定(秦港 5500 大卡价 + PB 分位 + 股息率)

## 龙头格局

- **CR3**:中国神华 + 陕西煤业 + 中煤能源(合计 ~40% A 股煤炭利润)
- **TOP 5**:中国神华(601088)/ 陕西煤业(601225)/ 中煤能源(601898)/ 兖矿能源(600188)/ 山西焦煤(000983)
- **格局演变**:供给侧改革后产能向五大集团 + 国家能源集团高度集中;"煤电联营"成行业共识(神华、陕煤、中煤纷纷参股电厂对冲);焦煤受钢铁需求拖累,动力煤受电力消费支撑分化加剧

## 关键观察指标

- **秦皇岛港 5500 大卡动力煤价**(长协 675 元 / 现货 800 元 = 平衡,> 1000 元过热 / < 600 元亏损)
- **重点电厂日耗 + 库存可用天数**(夏冬两季高峰日耗 > 700 万吨,库存 < 15 天紧张)
- **股息率 vs 10 年国债利差**(神华股息率 > 国债 + 3pp = 类债配置价值显现)
- **澳洲焦煤进口价 + 蒙煤通关量**(焦煤外贸依存度 30%,蒙煤甘其毛都口岸日通关量重要)
- **碳中和政策 + 电煤长协覆盖率**(长协覆盖率 > 80% = 业绩稳定性高;碳中和长期压制估值上限)

## 投资逻辑

**适合买入的时点**:煤价跌至成本线附近(动力煤 < 600 元)+ PB 分位 < 25% + 股息率 > 7% + 长协价稳定。林奇 cyclical 框架强调"周期股 PE 低反而是顶、PE 高反而是底",但近年龙头(神华)已部分蜕变为类公用事业的"高股息 stalwart",可用 DDM 估值锚定。

**回避时点**:煤价 > 1100 元持续 + 行业 PB > 1.8 + 库存高位 + 进口配额放开预期。鲁政委萧条期框架将煤炭视为"资源轮动配置",但需警惕"碳中和 + 新能源发电占比上升"的长期需求曲线下行风险(光伏+风电年新增装机已超火电)。
