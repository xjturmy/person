# 医疗服务

## 行业概览

医疗服务涵盖连锁专科医院、体检、CXO(医药外包) 三大子赛道,核心商业模式是"标准化复制 + 医生资源 + 规模效应"三重护城河。连锁专科(爱尔眼科/通策医疗)通过分级连锁+合伙人模式实现 30%+ 营收增长;体检龙头(美年健康/瑞慈)受益消费医疗渗透率提升;CXO(药明康德/泰格/凯莱英)是全球医药研发外包核心节点。本质是"老龄化 + 消费升级 + 医保减负"三浪叠加的长坡赛道,集采免疫属性突出。

## 周期特性

- **周期类型**:成长(林奇 fast_grower 范畴)
- **康波位置**:第六次康波生物科技方向 + 老龄化超长周期受益
- **典型周期长度**:连锁扩张周期 5-8 年;CXO 受全球医药研发投入 + 地缘政治周期影响 3-5 年
- **现在哪个阶段**:专科连锁(眼科/口腔)渗透率提升期 + CXO 地缘政治阴霾后的估值修复期 + 体检消费医疗复苏期

## 龙头格局

- **CR3**:爱尔眼科(眼科) / 迈瑞医疗(器械+服务) / 药明康德(CXO 龙头)
- **TOP 5**:爱尔眼科(300015,眼科连锁 700+ 家) / 迈瑞医疗(300760,医疗器械+服务) / 美年健康(002044,体检龙头) / 通策医疗(600763,浙江口腔连锁) / 药明康德(603259,全球 CXO 龙头)
- **格局演变**:专科连锁全国化加速(爱尔/通策);CXO 受《生物安全法案》压制后修复;互联网医疗(平安好医生/京东健康)流量入口分化

## 关键观察指标

- **连锁门店数 + 同店增长**(爱尔眼科分级连锁模式 = 门店扩张 × 同店爬坡双驱动)
- **CXO 在手订单 / 新签订单**(药明康德季度披露,先行指标)
- **集采免疫属性**(医疗服务定价权在医院端,不受药品/耗材集采直接冲击)
- **DRG/DIP 医保支付改革**(影响公立医院,利好民营专科)
- **海外营收占比**(CXO 海外占比 70%+,地缘政治敏感)
- **老龄化数据**(60+ 人口占比、慢病渗透率)

## 投资逻辑

**适合买入的时点**:估值分位 < 30%(CXO PE-TTM < 25 倍 / 连锁专科 PE-TTM < 35 倍) + 在手订单回升(CXO) + 同店增长企稳(连锁) + 地缘政治缓和(CXO)。林奇 fast_grower 框架要求 EPS 增速 25%+ 且 PEG < 1.5,爱尔眼科历史 PEG 1.0-1.5、药明康德 0.8-1.2 是经典 GARP 标的。

**回避时点**:估值分位 > 80%(爱尔眼科曾 PE 130 倍) + CXO 海外订单失速 + 《生物安全法案》落地 + 连锁扩张失控(应收账款/商誉减值)。医疗服务波动大,建议通过医疗 ETF 分散或聚焦头部连锁。
