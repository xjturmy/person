# 白色家电

## 行业概览

白色家电(空调/冰箱/洗衣机)是中国家电行业的"压舱石",已进入成熟期。核心商业模式是"规模 + 渠道 + 品牌"三重护城河,龙头三足鼎立(美的/格力/海尔)合计市占 70%+。海外营收占比已达 40%+(美的最高 ~45%),全球化是第二增长曲线。以旧换新政策周期性托底内销,房地产竣工面积滞后 6-9 月影响装修需求。

## 周期特性

- **周期类型**:防御(林奇 stalwart 范畴 — 中国版可口可乐)
- **康波位置**:萧条期稳定现金流 + 高股息防御
- **典型周期长度**:跟随地产/补贴政策 5-7 年小周期
- **现在哪个阶段**:海外营收占比持续提升 + 以旧换新政策托底内销 + 估值合理偏低

## 龙头格局

- **CR3**:美的 / 格力 / 海尔(三足鼎立),合计市占 70%+
- **TOP 5**:美的集团(000333,全品类全球化)/ 格力电器(000651,空调专家)/ 海尔智家(600690,海外强)/ 海信家电(000921)/ 小天鹅 A(已并入美的)
- **格局演变**:美的多元化 + 海外并购(库卡/东芝白电/Clivet)领先;格力空调主业坚固但多元化承压;海尔海外有机增长 + 卡萨帝高端突破

## 关键观察指标

- **空冰洗内销/出口排产**(产业在线月度数据,内销同比 / 出口同比)
- **原材料(铜/钢/塑)价格**(成本端,占总成本 70%+,铜价 < 7 万/吨利好)
- **房地产竣工面积**(滞后 6-9 月,装修需求驱动新购)
- **以旧换新补贴节奏**(2024-2025 补贴 GWh 级别拉动)
- **海外营收占比与汇率**(美的 45%/海尔 50%+,人民币贬值利好)
- **海外仓 + 跨境电商占比**(直接面对 C 端,毛利提升)

## 投资逻辑

**适合买入的时点**:估值分位 < 40%(PE < 15 倍)+ 股息率 > 4% + 出口数据回暖 + 原材料价格友好 + 以旧换新政策预期。林奇 stalwart 框架要求稳健增长(EPS 5-10%)+ 合理估值(PEG ~1)+ 高股息可持续。

**回避时点**:房地产竣工面积大幅下滑 + 原材料价格快速上涨(铜价 > 8 万/吨)+ 出口受阻(关税/贸易战) + 以旧换新政策退出。家电行业波动相对小,适合长期持有 + 股息再投资。
