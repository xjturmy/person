# 化学制药

## 行业概览

化学制药行业经历从"仿制药为主"向"创新药 + 仿制药双轨"的结构性转型。仿制药受集采压制毛利率(原研药降价 70-90%),创新药通过 License-out(对外授权)+ 自主出海打开估值天花板。研发管线兑现是估值核心,Ⅲ 期临床成功 + 海外 BLA/NDA 获批是关键里程碑。研发投入营收占比 >10% 的企业才有创新药潜力。

## 周期特性

- **周期类型**:成长(林奇 fast_grower 范畴)
- **康波位置**:第六次康波生物科技方向 — 与 AI/能源革命并列三大新动能
- **典型周期长度**:研发周期 10-15 年,临床进展驱动短中期股价
- **现在哪个阶段**:创新药出海加速期(2024-2026 多笔 License-out 大单)+ 仿制药集采压力释放期

## 龙头格局

- **CR3**:恒瑞医药 / 复星医药 / 石药集团(创新药管线 + 营收规模)
- **TOP 5**:恒瑞医药(600276,创新药龙头)/ 百济神州(688235,出海标杆)/ 信达生物(港股,PD-1)/ 复星医药(600196)/ 石药集团(港股)
- **格局演变**:Big Pharma(恒瑞/复星/石药)向创新药全面转型;Biotech(百济/信达/再鼎)聚焦差异化创新;CXO(药明康德/凯莱英)受地缘政治影响波动

## 关键观察指标

- **临床进展**(Ⅱ/Ⅲ 期里程碑,FDA/NMPA 获批)— 关键催化剂
- **License-out 交易首付/里程碑款**(默克 + 百济 EJM 案例)— 出海能力验证
- **集采续约价格**(部分品种已 3 轮集采,降幅边际收窄)
- **医保谈判节奏**(创新药纳入医保是放量起点)
- **研发投入/营收占比**(>10% 健康,恒瑞 ~25%)
- **海外营收占比**(出海 = 估值溢价)

## 投资逻辑

**适合买入的时点**:管线 Ⅲ 期数据读出前 6-12 月布局 + 集采落地后估值底 + License-out 大单催化 + 医保谈判预期。林奇 fast_grower 框架要求 EPS 增速 20%+ 与 PEG < 1.5 (但创新药估值通常用 DCF 或 sum-of-parts)。

**回避时点**:Ⅲ 期临床失败 + 集采降价超预期 + 出海受阻(地缘政治) + 研发投入大幅缩减。化学制药波动大,单一品种风险高,建议通过医药 ETF 分散配置或聚焦管线丰富的 Big Pharma。
