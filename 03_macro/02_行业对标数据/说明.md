## 📊 行业对标的价值

### 为什么需要行业分析？

1. **找出最好的公司** — 在同行业内进行对比，找出行业龙头和优势方
2. **评估估值合理性** — 对标行业平均估值，判断个股估值是否偏低或偏高
3. **理解竞争力** — 通过对标分析，理解公司的竞争优势和劣势
4. **追踪产业趋势** — 观察整个行业的发展阶段和增长空间

## 🎯 分析框架

### 行业对标的四个维度

#### 1. 估值对标
```
对比指标：PE、PB、PS、PEG
问题：
  - 我持有的公司估值在行业中处于什么水位？
  - 与行业龙头的估值差异是合理还是高估/低估？
  - 历史估值水位如何，当前是否处于极端位置？
```

#### 2. 财务对标
```
对比指标：ROE、ROA、净利率、毛利率、资产周转率
问题：
  - 公司盈利能力在行业中排名如何？
  - 财务质量是否优于行业平均水平？
  - 负债率和流动性是否健康？
```

#### 3. 成长对标
```
对比指标：收入增速、利润增速、EPS增速、市场份额
问题：
  - 公司增长速度是否超过行业平均？
  - 市场份额是否在扩大或缩小？
  - 行业处于什么阶段（成长期/成熟期/衰退期）？
```

#### 4. 风险对标
```
对比指标：负债率、流动比率、现金流充分度、营运效率
问题：
  - 公司风险是否低于行业平均？
  - 是否有潜在的财务风险或产业风险？
```

## 🔗 数据来源与获取方式

### 关键指标来源

| 指标类别 | 具体指标 | 数据源 | 更新频率 |
|---------|---------|--------|--------|
| 估值 | PE/PB/PS/股息率 | 理杏仁、Wind | 日度 |
| 财务 | ROE/ROA/利润率 | 公司财报、理杏仁 | 季度 |
| 成长 | 收入/利润增速 | 公司财报 | 季度 |
| 风险 | 负债率、流动比 | 公司财报 | 季度 |

### 获取方式

#### 方法 1：从 02_companies 提取数据
```
1. 打开 02_companies/{公司A}/01_基本面数据/
2. 提取需要对标的指标
3. 在本模块汇总并对比
```

#### 方法 2：从理杏仁批量获取
```
使用 .tools/lixinger-archiver 中的脚本
批量导出行业内多家公司的同一指标
```

#### 方法 3：利用券商研报
```
1. 打开 02_companies/{公司}/04_券商分析/
2. 提取券商对行业的对标分析
3. 整理成本模块的对标数据
```

## 💡 使用场景

### 场景 1：评估个股相对强弱

```
步骤：
1. 选定目标公司（如新华保险）
2. 确定行业同行（中国人保、中国太保等）
3. 对标关键指标（PE、ROE、负债率等）
4. 判断该公司的竞争力水位
5. 决定是否值得配置
```

### 场景 2：发现被低估的机会

```
步骤：
1. 扫描行业内的估值分布
2. 找出 PE/PB 明显低于行业平均的公司
3. 分析低估的原因（是风险还是市场忽视）
4. 结合基本面评估投资价值
```

### 场景 3：追踪行业龙头变化

```
步骤：
1. 定期更新行业内主要公司的财务指标
2. 观察 ROE、市场份额等排名变化
3. 识别龙头地位是否稳固
4. 调整持仓以适应行业格局变化
```

### 场景 4：理解产业风险

```
步骤：
1. 观察整个行业的利润率是否在压缩
2. 监测主要竞争对手的市场份额变化
3. 评估行业是否处于衰退期或转折点
4. 预警潜在的系统性风险
```

## 🎯 实战建议

✅ **做到**：
- 定期更新行业数据，保持对竞争格局的理解
- 将行业分析与个股分析结合，形成完整的投资视角
- 利用对标分析识别被市场忽视的投资机会
- 追踪行业龙头的变化，及时调整持仓

❌ **避免**：
- 只看单一公司数据，忽视行业整体趋势
- 过度依赖估值对标，忽视基本面差异
- 盲目追随龙头，不理解选择理由
- 忽视行业周期，在行业衰退期还在加仓
