# 集成电路

## 行业概览

集成电路是数字经济基石,产业链分设计(Fabless)/制造(Foundry)/封测(OSAT)三大环节。中国大陆 2023 年市场规模超 1.2 万亿元,自给率仅 ~20%,国产替代空间巨大。设计端韦尔/紫光国微/兆易创新在 CIS/特种 IC/存储已具全球竞争力;制造端中芯国际 14nm 量产 / 华虹特色工艺差异化突围;封测端长电/通富/华天进入全球前十。AI 芯片(寒武纪/海光/昇腾系)+ 存储(长江存储/长鑫)成为新增长极。

## 周期特性

- **周期类型**:成长(林奇 fast_grower,设计端波动大,设备/封测相对稳定)
- **康波位置**:第六次康波(AI+半导体)主线,与半导体设备形成产业链上下游协同
- **典型周期长度**:3-4 年(库存周期,硅周期)— 2019-2021 缺芯涨价,2022-2023 去库存,2024 起 AI 拉动新一轮
- **现在哪个阶段**:参见 Dashboard 实时判定(下游需求+库存周转+晶圆厂稼动率)

## 龙头格局

- **CR3**:中芯国际+韦尔股份+紫光国微 合计 ~30% 板块市值
- **TOP 4**:中芯国际(688981)A股+H股 制造端绝对龙头 / 韦尔股份(603501)CIS 全球第三 / 紫光国微(002049)特种 IC 龙头高毛利 / 兆易创新(603986)NOR Flash+MCU 国产领先
- **格局演变**:设计端百花齐放向头部集中;制造端中芯/华虹双龙头;封测从代工向 Chiplet 先进封装升级;AI 芯片寒武纪/海光异军突起

## 关键观察指标

- **下游需求**(手机/PC/汽车/AI 服务器 各占比与同比,AI 是 2024+ 主驱动)
- **晶圆厂稼动率**(中芯/华虹 季度披露,80%+ 健康,60% 以下预警)
- **存储芯片价格**(DRAM/NAND 现货价,周期指示器)
- **设计公司库存周转**(库存天数 < 90 天健康,> 150 天预警)
- **美国出口管制清单**(BIS 实体清单/EAR 修订,影响先进制程获取)

## 投资逻辑

**适合买入的时点**:下游需求复苏 + 库存出清 + 晶圆厂稼动率回升 + 估值分位 < 40% + AI 算力订单确认。林奇 fast_grower 框架要求营收增速 > 25% 且 PEG < 1.5,板块龙头如韦尔/紫光历史 PEG 中枢 1.0-1.3。

**回避时点**:下游消费电子疲软 + 库存堆积 + 稼动率持续下滑 + 估值分位 > 80% + 出口管制升级。第六次康波长期看好但需对短期硅周期保持敬畏,避免在景气顶点追高。
