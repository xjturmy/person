# 光伏设备

## 行业概览

光伏设备是新能源发电核心赛道,产业链由硅料-硅片-电池片-组件-逆变器-辅材构成。一体化龙头(隆基/通威/晶澳/TCL中环)凭借成本与产能优势主导格局,辅以专业化电池片企业与逆变器龙头(阳光电源)。本质是"制造业+周期品",定价权弱、降本周期清晰,技术路线 PERC→TOPCon→HJT/BC 不断迭代。2024-2025 行业进入过剩出清期,主链普遍亏损,优胜劣汰加速。

## 周期特性

- **周期类型**:典型 cyclical(林奇 cyclical 范畴 + 鲁政委复苏/繁荣期进攻品)
- **康波位置**:第六次康波能源革命主线方向,当前处于产能过剩出清期
- **典型周期长度**:3-5 年(产能扩张-过剩-出清循环);2019-2021 大牛市,2022 高景气,2023-2025 过剩底部
- **现在哪个阶段**:参见 Dashboard 实时判定(硅料/组件价格 + 行业 PB 分位 + 龙头开工率)

## 龙头格局

- **CR5**:隆基/通威/晶澳/TCL中环/晶科合计组件出货 >60%
- **TOP 5**:隆基绿能(601012)/ 通威股份(600438)/ TCL中环(002129)/ 晶澳科技(002459)/ 阳光电源(300274)
- **格局演变**:一体化趋势确立,二三线产能加速退出;BC/HJT 技术差异化突围;海外建厂(美国/东南亚/中东)规避关税成为新护城河

## 关键观察指标

- **硅料价格**(N 型致密料 < 40 元/kg = 全行业亏损底部,> 80 元 = 盈利改善)
- **组件价格**(< 0.7 元/W = 二三线亏现金,> 0.9 元 = 龙头盈利修复)
- **一体化龙头开工率**(< 60% 过剩,> 80% 紧平衡)
- **国内+海外装机量**(全球年度装机 YoY,500GW 是 2025 关键门槛)
- **PB 分位**(cyclical 看 PB 不看 PE,PB < 1.5x 接近周期底部)

## 投资逻辑

**适合买入的时点**:硅料/组件价格触底企稳 + 二三线持续退出 + 一体化龙头 PB < 1.5x + 装机数据 YoY 转正。cyclical 框架要求"PE 高时买、PE 低时卖"(底部亏损 PE 失真),用 PB 替代估值锚。康波第六次能源革命方向,长期需求确定性高。

**回避时点**:产能仍在扩张 + 价格继续下行 + 龙头开工率持续下滑 + 海外贸易政策恶化。林奇 cyclical 警告:周期顶部 PE 低反而是卖点(2022 年通威 PE 5 倍即为典型)。
