# 家电零部件

## 行业概览

家电零部件是家用电器大类下的细分赛道,核心是制冷压缩机、热管理阀件(电子膨胀阀/四通阀/截止阀)、电机控制器等关键部件。本质是**家电存量基本盘 + 新能源车热管理第二曲线 + 储能/数据中心液冷第三曲线**的三段式增长。龙头三花智控(002050)与盾安环境(002011)凭借家电时代积累的阀件技术,跨界进入特斯拉 Model S/3/Y 全系热管理供应链,毛利率显著高于家电业务,成为最重要的 alpha 来源。

## 周期特性

- **周期类型**:成长(fast_grower)— 家电存量基础上叠加新能源车 + 储能/液冷新曲线
- **康波位置**:第六次康波能源革命方向受益 — 热管理 = 新能源车 BOM 关键环节
- **典型周期长度**:新能源车渗透 7-10 年长周期 + 储能/液冷 5 年成长期
- **现在哪个阶段**:新能源车热管理渗透率从 50% 向 70% 迈进 + 储能温控/数据中心液冷需求 2025 起爆发,景气向上

## 龙头格局

- **CR3**:三花智控(002050,电子膨胀阀全球第一)+ 盾安环境(002011,四通阀)+ 银轮股份(002126,热交换器)
- **TOP 标的**:三花智控(002050)/ 盾安环境(002011)/ 银轮股份(002126)/ 春风动力(?) / 拓普集团(601689,空气悬挂/热管理一体化)
- **格局演变**:家电业务存量贡献现金流;新能源车热管理 → 单车价值量从 3000 元向 5000-8000 元(800V/液冷加持)提升;储能温控/数据中心液冷打开第三天花板

## 关键观察指标

- **新能源车热管理渗透率**(单车价值量 / 全行业出货量 / 月度跟踪)
- **储能/数据中心液冷需求**(招标量与价格,2025 储能温控市场 ~200 亿)
- **空调销量**(家电基本盘,以旧换新政策托底)
- **特斯拉/比亚迪/华为系车型出货**(直接影响三花/拓普订单)
- **铜价**(原材料成本占 BOM 20%+)

## 投资逻辑

**适合买入的时点**:新能源车热管理订单可见度提升 + 储能温控大订单 + 估值分位 < 30% + 单车价值量提升预期。林奇 fast_grower 框架要求 PE < EPS 增速,三花历史 PE 中枢 30-40 倍,景气底部 25 倍合理买点。

**回避时点**:新能源车价格战压制零部件价 + 储能补贴退坡 + 估值分位 > 80%。鲁政委萧条期框架将能源革命产业链列为"成长配置主力",但需警惕"成长股戴维斯双杀"风险。
