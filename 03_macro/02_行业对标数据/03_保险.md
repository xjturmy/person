# 保险

## 行业概览

保险业核心是"负债 + 资产"双轮 — 负债端通过代理人/银保/经代渠道卖寿险/财险积累保费,资产端配置债券/股票/不动产赚长期收益。寿险定价用 NBV(新业务价值)+ EV(内含价值)替代利润,因为利润释放滞后保单期 20-30 年。中国寿险渗透率 ~3.5%(美国 7%),长期空间大,短期受代理人改革阵痛 + 长端利率下行压制。

## 周期特性

- **周期类型**:价值(林奇 insurance 范畴 — 独立评分体系)
- **康波位置**:萧条期防御金融,但弹性大于银行
- **典型周期长度**:与利率周期+权益市场双联动,5-10 年
- **现在哪个阶段**:代理人改革阵痛末段 + 长端利率底部预期 + 权益市场低位,估值修复机会

## 龙头格局

- **CR3**:中国平安(601318)/ 中国人寿(601628)/ 中国太保(601601)合计市占 60%+
- **TOP 5**:中国平安 / 中国人寿 / 中国太保 / 新华保险(601336)/ 中国人保(601319)
- **格局演变**:平安综合金融 + 寿险改革领先;太保健康险/养老险布局深;新华纯寿险弹性最大;互联网保险(众安)细分突围

## 关键观察指标

- **新业务价值率 NBV margin**(>30% 优秀,行业平均 20-25%)
- **内含价值 EV 增长率**(>10% 健康)
- **10 年期国债收益率**(资产端关键,>3% 利好,<2.5% 利空)
- **代理人留存率与人均产能**(数量 → 质量转型期看人均件均/人均 NBV)
- **综合投资收益率**(>5% 长期目标,权益占比影响波动)
- **退保率与续保率**(< 2% 健康,警惕集中退保)

## 投资逻辑

**适合买入的时点**:PEV < 0.5(平安/新华)+ 长端利率见底 + 代理人队伍质量改善信号 + 权益市场低位 + 大单储蓄险火爆带动 NBV。格雷厄姆 insurance 框架强调 EV/PEV 安全垫 + 投资收益稳定性。

**回避时点**:长端利率快速下行(资产端再投资风险)+ 代理人继续大规模流失 + 权益市场暴跌(权益占比 12-15% 拖累)+ 监管政策收紧。萧条期保险股波动大于银行,需要更长持有期(3-5 年)兑现 EV 内在价值。
