## 📊 ETF 分析的价值

### 为什么用 ETF 做行业分析？

1. **代表性** — ETF 汇集了某行业的头部公司，代表行业走势
2. **实时性** — ETF 价格实时反映市场情绪
3. **简化性** — 避免个股选择困难，关注行业大趋势
4. **资金面** — ETF 成交量反映主力资金流向

### 与个股分析的互补关系

```
个股分析（微观）          ETF 分析（中观）

└─ 01_基本面数据             └─ 成分股、财务数据
   └─ 估值、财务              └─ 行业趋势、资金
```

## 📡 数据来源与获取

### 理杏仁 API 支持情况

✅ **可用**:
- **指数成分股 API** — 获取 ETF 跟踪指数的成分股列表（含权重）
- **公司基本面 API** — 获取成分股的财务指标（PE、PB、ROE 等）

❌ **不可用**:
- 基金概况 API — 理杏仁不提供
- 基金成分股 API — 理杏仁不提供

### 推荐的数据获取流程

1. **维护 ETF 配置** → 在 `etf_config.json` 记录 ETF 与跟踪指数的映射
2. **获取成分股** → 通过指数代码调用指数成分股 API，获取成分股清单
3. **获取财务数据** → 通过股票代码调用公司基本面 API，获取财务指标
4. **汇总分析** → 整合成分股数据生成 ETF 分析报告

**详见**: [理杏仁ETF数据支持调查](理杏仁ETF数据支持调查.md)

## 📝 模板与方法论

### ETF 配置文件格式

每个主题目录下的 `etf_config.json` 记录该主题的 ETF 信息：

```json
{
  "theme": "新能源",
  "etfs": [
    {
      "code": "516030",
      "name": "华夏新能源汽车ETF",
      "index_code": "399417",
      "index_name": "新能源汽车指数",
      "manager": "华夏基金",
      "inception_date": "2021-12-01"
    }
  ]
}
```

### 单个 ETF 分析框架

每个 ETF 的分析应包括：

1. **基本信息** — ETF 代码、跟踪指数、主要成分股
2. **行业前景** — 该行业的发展阶段和成长空间
3. **资金面** — ETF 近期资金流向、成交量变化
4. **估值水位** — ETF 所代表行业的估值水位
5. **轮动阶段** — 该行业在周期中的位置
6. **成分股质量** — 成分股的财务指标和权重分布
7. **操作建议** — 基于上述分析的建议

详见：[ETF分析模板](模板/ETF分析模板.md)

## 🔄 更新维护

### 数据来源
- ETF 实时数据：天天基金、同花顺等
- 成分股数据：理杏仁指数成分股 API
- 财务指标：理杏仁公司基本面 API
- 行业数据：研究报告、新闻资讯

### 更新频率
- **基本信息（etf_config.json）** — 变更时更新
- **成分股列表（constituents.csv）** — 季度更新（指数调整周期）
- **财务数据（fundamentals.csv）** — 月度更新
- **分析评估** — 每月或定期更新

### 维护流程
1. 定期检查 ETF 成分股变化（通过指数 API）
2. 跟踪成分股的财务指标变化（通过公司 API）
3. 监测 ETF 资金流向和估值变化
4. 整理成分股数据到本目录
5. 更新 ETF 分析报告

## 🎯 应用场景

### 场景 1：行业轮动择时
```
1. 查看各主题 ETF 的资金流向
2. 观察 ETF 价格与估值变化
3. 通过成分股权重变化，判断行业结构
4. 决定是否配置该行业
```

### 场景 2：个股选择参考
```
1. 选定投资主题（如新能源）
2. 查看该主题 ETF 的主要成分股及权重
3. 结合 02_companies/ 中对应公司的详细分析
4. 在行业内精选个股
```

### 场景 3：资产配置
```
1. 监控 4 个主题 ETF 的相对强弱
2. 观察成分股权重变化
3. 调整持仓配置以适应行业轮动
4. 在不同主题间分散投资风险
```

### 场景 4：风险评估
```
1. 通过成分股权重分析集中度风险
2. 通过财务指标评估成分股质量
3. 观察权重变化识别风险转移
4. 评估 ETF 的系统性风险
```

## 💡 使用建议

✅ **做到**：
- 定期查看 ETF 走势，感受市场脉动
- 与个股分析结合，提高决策质量
- 跟踪主要成分股在 02_companies/ 的分析
- 利用成分股财务数据评估 ETF 质量
- 监测权重变化识别行业轮动信号

❌ **避免**：
- 只看 ETF 不看基本面
- 盲目追热点概念
- 忽视行业周期
- 过度集中于单个成分股
- 忽视权重变化带来的风险
