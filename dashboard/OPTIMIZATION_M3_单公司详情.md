## 🎯 优化目标

把当前"Hero + 雪花 + 大师矩阵 + 子 Tab + K线 + 文档 + 横向对比"**层级混乱的 9 段堆叠**,重组为**4 个清晰区块 + 提问式标题 + 优势短板/七大师投票自带说明**的单页驾驶舱。

四大改动:
1. **优势/短板 Top3 增加"为什么强/弱"一句话归因**(目前只有维度名 + 分数)
2. **七大师投票增加方法论速读 + 投票口径说明**(目前是 expander 默认折叠 + 哲学速读并列两栏挤)
3. **重排 4 大区块,加视觉分隔(背景色/卡片/带 ▼ 折叠图标的 banner)**,让用户知道"看到哪了"
4. **顺序按"快→深"梳理**:决策结论 → 评分体系 → 数据细节 → 文档归档
