**正常化PE估值（考虑周期）：**

假设制冷剂价格回落至中等水平：
```
情景分析：

- 中性：净利润回落至12亿，合理PE 18倍 → 市值216亿 → 股价35元
```

### 3.3 股息贴现模型（DDM）简化版

**分红数据（估算）：**

| 年度 | 每股分红 | 分红率 | 股息率 |
|-----|---------|-------|-------|
| 2024 | ~0.60 | ~30% | ~1% |
| 2023 | ~0.20 | ~25% | ~0.5% |
| 2022 | ~0.30 | ~30% | ~0.8% |

**简化DDM估值：**

假设：
- 长期分红增长率 g = 3%
- 要求回报率 r = 10%
- 2025年预期每股分红 D1 = 1.20元

```
合理股价 = D1 / (r - g) = 1.20 / (10% - 3%) = 17元

当前股价 = 68元
溢价幅度 = (68 - 17) / 17 = 300%
```

**结论：从股息角度，当前股价严重高估。**
