# 芯片ETF 分析资料

> 本文件夹用于存放芯片ETF的相关分析资料

## 文件结构

- **行业深度分析五步法记录.md** - 使用五步法进行的行业分析
- **成分股共振检查清单.csv** - 成分股基本面跟踪清单
- **资金流向跟踪.csv** - 资金流向数据记录
- **估值跟踪表.csv** - 估值历史数据跟踪

## 数据来源

- 基金信息：基金公司官网、天天基金网
- 估值数据：理杏仁、Wind
- 资金流向：东方财富、同花顺
- 成分股数据：指数公司官网

## 更新频率

- 估值数据：每日更新
- 资金流向：每周更新
- 深度分析：每季度更新
