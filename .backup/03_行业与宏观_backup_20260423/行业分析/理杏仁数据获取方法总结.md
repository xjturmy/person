## 关键代码片段

### 获取所有指标数据

```python
import requests
import csv
import os
from datetime import datetime, timedelta

# API配置
API_URL = "https://open.lixinger.com/api/cn/company/fundamental/non_financial"
TOKEN = "YOUR_TOKEN_HERE"
STOCK_CODE = "000333"
COMPANY_NAME = "美的集团"

# 获取过去一年的数据
end_date = datetime.now()
start_date = end_date - timedelta(days=365)

# 定义所有指标
indicators = {
    "pe_ttm": "PE-TTM",
    "pb": "PB",
    "sp": "股价",
    "mc": "总市值",
    "ta": "成交金额",
    "dyr": "股息率",
    "shn": "总股东人数",
    "tv": "成交量",
    # ... 其他指标
}

# 发送请求
payload = {
    "token": TOKEN,
    "stockCodes": [STOCK_CODE],
    "startDate": start_date.strftime("%Y-%m-%d"),
    "endDate": end_date.strftime("%Y-%m-%d"),
    "metricsList": list(indicators.keys())
}

response = requests.post(API_URL, json=payload, timeout=60)
data = response.json()["data"]
```

### 保存为每个指标单独的CSV文件

```python
def save_individual_indicators(data, company_name, stock_code, target_dir):
    """将每个指标保存为单独的CSV文件"""
    os.makedirs(target_dir, exist_ok=True)
    
    # 按日期排序
    data.sort(key=lambda x: x.get('date', ''))
    
    # 提取基础数据
    base_data = []
    for item in data:
        base_data.append({
            "date": item.get("date", ""),
            "stockCode": item.get("stockCode", "")
        })
    
    created_files = []
    
    # 为每个指标创建CSV文件
    for indicator_code, indicator_name in indicators.items():
        csv_filename = os.path.join(target_dir, f"{company_name}_{indicator_name}.csv")
        
        # 准备该指标的数据
        indicator_data = []
        for i, item in enumerate(data):
            if indicator_code in item:
                row_data = base_data[i].copy()
                row_data[indicator_name] = item[indicator_code]
                indicator_data.append(row_data)
        
        if indicator_data:
            # 保存为CSV文件
            with open(csv_filename, 'w', newline='', encoding='utf-8-sig') as csvfile:
                fieldnames = ["date", "stockCode", indicator_name]
                writer = csv.DictWriter(csvfile, fieldnames=fieldnames)
                writer.writeheader()
                writer.writerows(indicator_data)
    
    return created_files
```

## 常见问题与解决方案

### Q1: API返回401错误
- **原因**：Token无效或过期
- **解决方案**：重新获取Token，确保会员状态有效

### Q2: 数据条数不足
- **原因**：某些指标数据可能不完整（如陆股通数据只有最近几天）
- **解决方案**：检查数据完整性，部分指标可能只有近期数据

### Q3: 网页模式导出失败
- **原因**：页面加载问题或按钮交互问题
- **解决方案**：切换到API模式

### Q4: 文件编码问题
- **原因**：中文Excel需要UTF-8 with BOM编码
- **解决方案**：使用`encoding='utf-8-sig'`保存CSV文件

## 最佳实践

1. **定期更新**：每月检查一次Token有效性
2. **数据验证**：下载后检查数据完整性
3. **分批处理**：对于大量股票，建议分批处理
4. **错误处理**：添加重试机制和错误日志
5. **数据备份**：定期备份下载的数据文件
