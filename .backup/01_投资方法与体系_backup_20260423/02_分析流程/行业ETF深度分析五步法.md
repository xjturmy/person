# 行业ETF深度分析五步法

> 系统化的行业分析框架，适用于ETF投资决策

---

## 第一步：行业生命周期定位

### 1.1 识别行业所处阶段

```
行业生命周期四阶段：

导入期        成长期          成熟期         衰退期
  │            │              │             │
  ▼            ▼              ▼             ▼
┌─────┐     ┌──────┐       ┌──────┐      ┌──────┐
│高风险│     │高增长│       │稳收益│      │低增长│
│高回报│     │高波动│       │低波动│      │高风险│
└─────┘     └──────┘       └──────┘      └──────┘
  │            │              │             │
  ▼            ▼              ▼             ▼
投资方式：   投资方式：     投资方式：    投资方式：
风投/早期    成长股策略     价值+分红     困境反转
```

### 1.2 定位检查清单

| 特征 | 导入期 | 成长期 | 成熟期 | 衰退期 |
|------|--------|--------|--------|--------|
| **增长率** | 0-10% | 20-50% | 5-15% | <0% |
| **竞争格局** | 分散 | 集中度提升 | 寡头垄断 | 退出整合 |
| **利润率** | 负或低 | 快速提升 | 稳定高 | 下滑 |
| **资本需求** | 高 | 高 | 中 | 低 |
| **典型策略** | 观望/小仓位 | 积极配置 | 核心持仓 | 回避/短周期 |

### 1.3 分析方法

**定量指标：**
- 行业收入5年CAGR
- 龙头企业增速
- 资本开支/GDP占比
- 渗透率曲线

**定性判断：**
- 技术成熟度（Gartner曲线）
- 政策支持力度
- 消费者接受度
- 产业链配套成熟度

### 5.3 买卖纪律

**买入条件（满足3条以上）：**
- [ ] PE处于历史30%分位以下
- [ ] 行业增速 > GDP增速
- [ ] 竞争格局改善（集中度提升）
- [ ] 政策支持力度加大
- [ ] 资金面持续流入

**卖出条件（满足2条以上）：**
- [ ] PE升至历史80%分位以上
- [ ] 行业基本面恶化（连续两季增速下滑）
- [ ] 竞争格局恶化（价格战激烈）
- [ ] 重大政策利空
- [ ] 发现更好的替代标的
