## 一、核心分类原则

```
┌─────────────────────────────────────────────────────────────┐
│                    行业ETF分类逻辑                             │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【判断维度】                                                 │
│  1. 盈利稳定性：ROE波动幅度、盈利可预测性                     │
│  2. 经济周期敏感度：是否随经济周期大幅波动                   │
│  3. 成长性：长期增速预期                                     │
│  4. 波动性：历史回撤幅度                                    │
│  5. 股息特征：是否稳定分红                                   │
│                                                              │
│  【分类标准】                                                 │
│                                                              │
│  防御权益 ←────────────────→ 进攻权益                          │
│  稳定盈利    中等波动        高增长预期                       │
│  低敏感度    适度周期        高波动性                         │
│  高股息      中等股息        低股息/不分红                    │
│  必需消费    可选消费        科技成长                         │
│  公用事业    医药/金融        AI/新能源/生物                  │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

## 二、防御权益行业ETF（25-30%）

### 2.1 核心特征
- **盈利稳定性**：ROE波动<5%，连续10年盈利
- **经济周期敏感度**：低（收入不随周期大幅波动）
- **成长性**：增速5-10%（稳健增长）
- **波动性**：年化波动率<20%
- **股息特征**：股息率>3%，连续分红

### 2.2 细分行业及ETF

#### 必需消费（8-10%）⭐防御核心
**投资逻辑**：无论经济好坏，人们都要吃饭、用日用品

| 子行业 | ETF示例 | 配置建议 | 核心特征 |
|--------|---------|----------|----------|
| **食品饮料** | 消费ETF(159928)、酒ETF(512690) | 5-7% | 品牌护城河、定价权 |
| **农林牧渔** | 养殖ETF(159865)、农业ETF(159825) | 2-3% | 周期弱化版、刚需 |
| **日用品** | 消费50ETF(515650) | 1-2% | 稳定现金流 |

**选股标准**：
- 必选消费属性（不看经济好坏都要消费）
- 品牌护城河（茅台、海天、伊利等）
- ROE>15%且波动小
- 股息率>2%

#### 公用事业（5-8%）⭐防御核心
**投资逻辑**：特许经营、稳定现金流、类债券属性

| 子行业 | ETF示例 | 配置建议 | 核心特征 |
|--------|---------|----------|----------|
| **电力** | 电力ETF(159611)、绿电ETF(159669) | 3-5% | 稳定收益、新能源转型 |
| **水务/燃气** | （无单独ETF，选个股） | 1-2% | 特许经营权 |
| **交通运输** | 交通运输ETF(159666) | 1-2% | 高速、铁路稳定分红 |
| **电信运营商** | 通信ETF(515880) | 2-3% | 高股息、低增长 |

**选股标准**：
- 特许经营或寡头垄断
- 稳定的收费模式
- 高股息率（>4%）
- 低增长但低风险

#### 金融（红利属性，3-5%）⭐防御辅助
**投资逻辑**：高股息、低估值、系统重要性

| 子行业 | ETF示例 | 配置建议 | 核心特征 |
|--------|---------|----------|----------|
| **银行** | 银行ETF(512800)、红利低波ETF(512590)含银行 | 2-3% | 高股息、低估值 |
| **保险** | 保险主题基金（场外） | 1-2% | 长期价值、低估值时配置 |
| **券商** | **⚠️偏进攻，不算防御** | 0%或进攻权益 | 周期性强、波动大 |

**选股标准**：
- 大型国有银行优先（工行、建行等）
- 股息率>5%
- 估值处于历史低位（PB<0.8）
- 避开券商（归入进攻权益）

#### 红利低波（8-12%）⭐防御组合
**投资逻辑**：一篮子高股息、低波动股票，天然防御属性

| ETF | 代码 | 配置建议 | 特点 |
|-----|------|----------|------|
| **红利低波ETF** | 512590 | 5-8% | 沪深A股高股息低波动 |
| **中证红利ETF** | 515880 | 3-5% | 沪深100只高股息 |
| **标普中国A股红利** | 场外基金 | 2-3% | 质量更高的红利股 |
| **深证红利ETF** | 159905 | 2-3% | 深市高股息蓝筹 |

**特点**：
- 天然包含大量防御性行业
- 股息率通常>4%
- 波动率低于沪深300
- 适合作为防御权益底仓

### 2.3 模糊地带的处理

| 行业 | 默认分类 | 调整条件 | 说明 |
|------|----------|----------|------|
| **医药** | 看细分 | 传统中药→防御；创新药→进攻 | 传统医药偏防御，CXO、创新药偏进攻 |
| **房地产** | 进攻权益 | 当前周期下行 | 高杠杆、强周期，不属防御 |
| **建筑/基建** | 进攻权益 | 政策驱动、周期性强 | 虽分红但波动大 |
| **煤炭** | 防御权益 | 高股息周期 | 当前高股息，但需警惕周期下行 |

## 三、进攻权益行业ETF（15-25%）

### 3.1 核心特征
- **盈利稳定性**：ROE波动>10%，盈利不确定性高
- **经济周期敏感度**：中高（收入随周期波动大）
- **成长性**：增速预期15%+（高增长）
- **波动性**：年化波动率>25%
- **股息特征**：股息率低（<1%）或不分红，专注成长

### 3.2 细分行业及ETF

#### AI方向（5-8%）⭐进攻核心
**投资逻辑**：第六次康波核心驱动力，高成长高波动

|--------|---------|----------|----------|
| **芯片/半导体** | 芯片ETF(159995)、半导体ETF(512480) | 2-3% | AI算力基础设施 |
| **云计算/软件** | 云计算ETF(516510)、软件ETF(515230) | 2-3% | AI应用平台 |
| **人工智能** | AIETF(515070)、人工智能ETF(159819) | 2-3% | AI产业链综合 |
| **机器人** | 机器人ETF(562500) | 1-2% | AI+制造业 |
| **全球AI** | 纳斯达克100ETF(513100) | 3-5% | 全球AI龙头exposure |

**选股标准**：
- 处于AI产业链核心位置
- 有明确商业化路径
- 估值容忍度较高（但要避免极端泡沫）
- 单只ETF不超过3%

#### 新能源方向（5-8%）⭐进攻核心
**投资逻辑**：第六次康波能源革命，过渡期结构性机会

| **新能源车** | 新能源车ETF(515030)、汽车ETF(516110) | 2-3% | 渗透率持续提升 |
| **碳中和** | 碳中和ETF(159790)、环保ETF(159861) | 1-2% | 综合清洁能源 |

**选股标准**：
- 技术路线清晰
- 行业龙头优先
- 避开产能过剩最严重环节
- 关注产能出清信号

#### 生物科技方向（3-5%）⭐进攻卫星
**投资逻辑**：第六次康波方向之一，早期高风险高回报

| 子行业 | ETF示例 | 配置建议 | 核心特征 |

**选股标准**：

- 现金储备充足（应对烧钱阶段）
- 高度分散（ETF优于个股）
- 单只不超过2%

**选股标准**：
- 国产替代逻辑清晰
- 技术壁垒较高
- 避免纯概念炒作

## 五、配置执行指南

### 5.1 防御权益配置步骤

```
Step 1：建立底仓（60%防御权益仓位）
□ 买入红利低波ETF（5-8%）
□ 买入消费ETF（3-5%）

Step 2：补充公用事业（20%防御权益仓位）
□ 买入电力ETF（2-3%）
□ 买入通信ETF（1-2%）

Step 3：金融配置（20%防御权益仓位）
□ 买入银行ETF（2-3%）
□ 或继续持有红利低波（已含银行）

优先级：红利低波 > 消费 > 电力 > 银行 > 其他
```

### 5.2 进攻权益配置步骤

```
Step 1：核心配置（60%进攻权益仓位）
□ 买入纳斯达克100ETF（3-5%）← 全球AI exposure
□ 买入芯片ETF（2-3%）← AI算力基础设施

Step 2：新能源配置（30%进攻权益仓位）
□ 买入光伏ETF（2-3%）← 当前左侧布局
□ 买入新能源车ETF（2-3%）← 渗透率提升

Step 3：卫星配置（10%进攻权益仓位）
□ 生物科技ETF（2-3%）← 高风险高回报
□ 其他方向小仓位试探（各1-2%）

优先级：纳指100 ≈ 芯片 > 光伏 ≈ 新能源车 > 生物科技 > 其他
```

### 5.3 行业ETF选择检查清单

**选择前必问**：
```
□ 这个行业的盈利是否稳定？（防御权益要求稳定）
□ 这个行业是否随经济周期大幅波动？（周期性强→进攻）
□ 这个行业的股息率是否>3%？（高股息→防御）
□ 这个行业的成长性预期是否>15%？（高成长→进攻）
□ 这个行业的波动性是否<20%？（低波动→防御）

如果以上5题多数答"是" → 防御权益
如果以上5题多数答"否" → 进攻权益
```

## 六、常见误区与提醒

### 6.1 常见分类错误

| 错误认知 | 正确分类 | 理由 |
|----------|----------|------|
| "医药都是防御" | 看细分 | 中药/仿制药→防御；CXO/创新药→进攻 |
| "银行是进攻" | 防御 | 高股息、低波动，属于防御权益 |
| "券商是防御" | 进攻 | 强周期、高Beta，属于进攻权益 |
| "5G通信是进攻" | 偏防御 | 基础设施属性，波动相对较小 |
| "房地产是防御" | 进攻 | 当前周期下行，高杠杆高风险 |

### 6.2 动态调整提醒

```
行业ETF不是一成不变的：

防御权益 → 进攻权益（罕见）：
• 传统消费公司转型成功（如白酒企业做新能源）
• 公用事业公司大幅扩张（如电力公司转型绿电平台）

进攻权益 → 防御权益（常见）：
• 科技行业成熟（如通信行业从3G→5G，增速放缓）
• 周期行业供给侧改革后稳定（如煤炭行业产能出清）
• 行业进入成熟期，开始稳定分红

定期Review（季度）：
• 检查行业属性是否发生变化
• 检查估值是否极端（防御行业估值过高时减仓）
• 检查股息率是否持续（防御行业不分红时剔除）
```
