### 1.2 核心原则

| 原则 | 说明 | 投资应用 |
|------|------|----------|
| **多学科整合** | 从不同学科汲取重要模型 | 不局限于财务分析 |
| **双轨分析** | 理性分析 + 心理分析 | 既要算对，也要理解人性 |
| **能力圈原则** | 只投资自己能理解的 | 不懂不投 |
| **逆向思维** | 先想如何失败 | 避免愚蠢比追求聪明更重要 |
| **检查清单** | 系统性避免遗漏 | 投资决策前的必备流程 |

## 二、关键思维模型

### 2.1 数学与统计模型

#### 复利模型
```
公式：FV = PV × (1 + r)^n
关键洞察：时间是好企业的朋友，坏企业的敌人
```

#### 概率思维
- 决策是基于概率的，不是确定性的
- 寻求期望值 > 0 的机会
- 重视基础概率（不要忽视先验概率）

#### 正态分布与肥尾
- 极端事件比正态分布预测的更常见
- 投资中要留足安全边际应对黑天鹅
- 不要过度依赖历史数据

#### 贝叶斯定理
```
P(A|B) = P(B|A) × P(A) / P(B)

投资应用：根据新证据不断更新对公司的判断
```

### 2.2 物理与工程模型

#### 临界质量
- 某些效应需要达到临界规模才能显现
- 网络效应、规模经济都需要临界质量
- 投资平台型公司需要判断是否已达到临界点

#### 反馈循环
- **正反馈**：成功带来更大成功（飞轮效应）
- **负反馈**：自我调节的稳定性
- 案例：亚马逊的飞轮，苹果的生态系统

#### 冗余备份
- 工程系统需要冗余来保证可靠性
- 投资中的体现：安全边际、分散投资
- 不要把所有筹码押在一个判断上

### 2.3 生物与进化模型

#### 进化论
- 适者生存，不是最强者生存
- 商业环境不断变化，适应能力是关键
- 关注公司的进化能力而非静态优势

#### 生态位
- 每个物种占据特定生态位
- 商业中的细分市场、护城河
- 关注公司在生态系统中的位置

#### 达尔文法则
- 变化是永恒的
- 过度专业化可能是风险
- 多元化能力有助于长期生存

### 2.4 心理学模型

#### 认知偏差清单

| 偏差类型 | 表现 | 防御策略 |
|----------|------|----------|
| 确认偏差 | 只寻找支持自己观点的证据 | 主动寻找反面证据 |
| 可得性偏差 | 过度关注容易想起的信息 | 系统性收集数据 |
| 锚定效应 | 被第一印象过度影响 | 多维度估值 |
| 损失厌恶 | 对损失的恐惧大于对收益的喜悦 | 预先设定规则 |
| 社会认同 | 羊群效应，随大流 | 独立思考 |
| 权威偏差 | 过度信任专家 | 质疑权威 |
| 激励偏差 | 被利益影响判断 | 理解各方利益 |
| 禀赋效应 | 对自己持有的东西估值过高 | 客观重估 |
| 沉没成本 | 因已投入而继续 | 面向未来决策 |

### 2.5 经济学模型

#### 机会成本
- 每个选择都有其代价
- 持有现金的机会成本
- 换仓决策的核心考量

#### 边际效用递减
- 新增投入带来的收益递减
- 规模不经济的风险
- 关注边际ROIC而非总量

#### 比较优势
- 专注自己最擅长的
- 在自己的能力圈内投资
- 承认某些领域不如别人

#### 博弈论
- 考虑对手的反应
- 竞争格局分析
- 纳什均衡在商业中的应用

### 2.6 会计学模型

#### 复式记账
- 每个数字都有对应科目
- 理解三张报表的内在联系
- 关注异常变动

#### 现金流量
- 现金是事实，利润是观点
- 自由现金流的重要性
- 区分会计利润和经济利润

### 2.7 复杂系统模型

#### 涌现性
- 整体大于部分之和
- 企业的文化是涌现现象
- 无法通过分析单一因素预测整体

#### 非线性
- 输入与输出不成比例
- 小因素可能引发大变化
- 杠杆效应和连锁反应

#### 不确定性
- 世界本质上是不确定的
- 概率分布可能比点估计更有用
- 保持谦逊和开放

## 三、投资决策检查清单

### 3.1 买入前检查清单

```
□ 数学模型
  □ 估值是否合理？（安全边际）
  □ 预期收益率是否 > 机会成本？
  □ 风险调整后收益是否有吸引力？

□ 商业模型
  □ 商业模式是否简单可理解？
  □ 护城河是什么？（竞争优势）
  □ 行业格局如何？（博弈论）

□ 财务模型
  □ ROIC是否持续高于WACC？
  □ 现金流是否健康？
  □ 资产负债表是否稳健？

□ 管理模型
  □ 管理层是否理性、能干、诚实？
  □ 激励是否对齐？
  □ 资本配置能力如何？

□ 心理模型
  □ 我是否受到确认偏差影响？
  □ 是否考虑了反面证据？
  □ 是否因为持有而估值过高？（禀赋效应）
  □ 买入是因为价值还是因为他人在买？（社会认同）

□ 系统模型
  □ 公司的生态位是什么？
  □ 是否具备进化能力？
  □ 黑天鹅风险是什么？
```

### 3.2 持有期间检查清单

```
□ 基本面是否恶化？
□ 估值是否变得不合理？
□ 是否有更好的机会？（机会成本）
□ 是否因为沉没成本而继续持有？
□ 是否因为禀赋效应而忽视风险？
```

### 3.3 卖出检查清单

```
□ 买入理由是否不再成立？
□ 是否发现了当初的错误？
□ 估值是否严重高估？
□ 基本面是否发生质变？
□ 是否有更好的替代标的？
```
