## 二、第一类：债券（35-40%）

### 2.1 配置逻辑
- **核心功能**：流动性储备 + 防御本金 + 锁定票息收益
- **配置理由**：
  - 萧条期现金为王，保持充足流动性
  - 当前利率处于相对高位，锁定票息收益
  - 为市场恐慌时的极端机会储备弹药
  - 避免权益市场大幅波动

### 2.2 细分配置

| 子类别 | 配置比例 | 具体标的 | 功能定位 |
|--------|----------|----------|----------|
| **现金及货币基金** | 20-25% | 余额宝、货币基金、短债理财 | 流动性储备，随时可用 |
| **短债基金** | 5-10% | 短债ETF(511360)、短融ETF | 稳健收益，波动小 |
| **国债/国开债** | 5-10% | 国债ETF(511010)、国开债ETF(159650) | 锁定高利率，无信用风险 |

### 2.3 操作要点

│  □ 分散在2-3个货币基金/短债产品                              │

## 三、第二类：黄金（15-20%）

### 3.1 配置逻辑
- **核心功能**：对冲信用货币贬值 + 地缘风险保险 + 抗通胀
- **配置理由**：
  - 康波萧条期典型特征：债务高企、货币贬值压力
  - 地缘冲突加剧时，黄金是"终极避险资产"
  - 黄金与权益资产相关性低，分散风险
  - 周金涛："黄金是康波萧条期的战略配置"

### 3.2 细分配置

| 子类别 | 配置比例 | 具体标的 | 特点 |
|--------|----------|----------|------|
| **黄金ETF** | 15-20% | 华安黄金ETF(518880)、博时黄金ETF(159937) | 流动性好，交易便捷 |
| **实物黄金**（可选） | 0-5% | 金条、金币 | 极端风险对冲 |

### 3.3 操作要点

## 四、第三类：防御权益（25-30%）

### 4.1 配置逻辑
- **核心功能**：稳定现金流 + 低估值安全垫 + 适度增长
- **配置理由**：
  - 萧条期需要"收息资产"提供收益缓冲
  - 防御型资产波动小，心理压力大
  - 高股息再投资，复利效应
  - 优质资产在恐慌时被错杀，提供买入机会

### 4.2 细分配置

| 子类别 | 配置比例 | 具体标的 | 选股标准 |
|--------|----------|----------|----------|
| **红利低波ETF** | 8-12% | 红利低波ETF(512590)、中证红利ETF(515880) | 高股息+低波动 |
| **必需消费** | 8-10% | 消费ETF(159928)或个股（茅台、伊利等） | 品牌护城河+稳定现金流 |
| **公用事业** | 5-8% | 水电、高速、电信运营商个股 | 特许经营+高股息 |

## 五、第四类：进攻权益（15-25%）

### 5.1 配置逻辑
- **核心功能**：布局第六次康波 + 获取长期成长收益
- **配置理由**：
  - 第六次康波（AI+能源+生物）是未来10-20年的主线
  - 当前处于早期，估值波动大，需小仓位试探
  - 2030年前后若回升期确认，可逐步加仓
  - 保持对新技术的 exposure，不错过时代机遇

### 5.2 细分配置

| 方向 | 配置比例 | 具体标的 | 风险等级 |
|------|----------|----------|----------|
| **AI方向** | 5-8% | 芯片ETF(159995)、云计算ETF(516510)、纳指ETF(513100) | 高 |
| **新能源** | 5-8% | 光伏ETF(159857)、新能源车ETF(515030)、储能 | 中高 |
| **生物科技** | 3-5% | 生物科技ETF(159892)、创新药ETF(515120) | 极高 |
| **价值核心** | 2-5% | 护城河个股（ROE>20%的龙头） | 中 |

### 5.3 各方向投资逻辑

#### AI方向（5-8%）
```
投资逻辑：
• 第六次康波的核心驱动力（周金涛）
• 已从概念进入商业化早期
• 2030年AI生产率数据验证是关键节点

配置比例：
• 基础配置：5%
• 2030年验证后：可加仓至15-20%

具体标的：
• 芯片/算力：3%（AI基础设施，最先受益）
• 云计算/平台：2%（AI应用层）
• 纳指ETF：3%（全球AI龙头exposure）

风险控制：
• 单只ETF不超过3%
• 不追高涨幅>100%的标的
• 设置止损：跌幅>40%时review
```

#### 新能源方向（5-8%）
```
投资逻辑：
• 第六次康波的"基础"（能源革命）
• 过渡期结构性机会（光伏、储能、电动车）
• 当前产能过剩，处于周期底部，适合布局

配置比例：
• 基础配置：5-8%
• 产能出清信号出现后：可加仓至10-12%

具体标的：
• 光伏ETF：3%（等待产能过剩缓解）
• 新能源车ETF：3%（渗透率持续提升）
• 储能：2%（高成长性，但波动大）

风险控制：
• 精选龙头，避开二线厂商
• 关注产能周期，避免高位买入
• 技术路线风险（如锂电vs氢能）
```

#### 生物科技方向（3-5%）
```
投资逻辑：
• 第六次康波方向之一
• 基因编辑、细胞治疗等前沿技术
• 早期阶段，高度分散投资

配置比例：
• 基础配置：3%
• 重大临床突破后：可加仓至5-8%

具体标的：
• 生物科技ETF：2%
• 创新药ETF：1%
• 医疗器材：1%

风险控制：
• 单只持仓不超过1%
• 高度分散（ETF优于个股）
• 关注医保政策风险
```

## 六、三种风险偏好配置方案

```

适合人群：
• 临近退休或已退休
• 无法承受>20%回撤
• 追求稳定现金流

配置详情：
债券45%：现金30% + 短债10% + 国债5%

```

```
债券：35%    黄金：15%    防御权益：30%    进攻权益：20%

适合人群：
• 有3-5年投资经验
• 能承受20-30%回撤
• 追求稳健增值+适度成长

配置详情：
债券35%：现金25% + 短债5% + 国债5%
黄金15%：黄金ETF 15%
防御权益30%：红利低波12% + 必需消费10% + 公用事业8%
进攻权益20%：AI 6% + 新能源 6% + 生物科技 4% + 价值核心 4%
```

### 6.3 积极型（适合年轻投资者）

```
债券：30%    黄金：15%    防御权益：25%    进攻权益：30%

适合人群：
• 年轻，有10年以上投资期限
• 能承受30-40%回撤
• 追求长期复利最大化

配置详情：
债券30%：现金20% + 短债5% + 国债5%
黄金15%：黄金ETF 15%
防御权益25%：红利低波10% + 必需消费8% + 公用事业7%
进攻权益30%：AI 10% + 新能源 10% + 生物科技 5% + 价值核心 5%

注意：当前萧条期，建议先用平衡型，2030年确认回升期后再转积极型

## 七、动态调整规则

### 7.1 基于估值的再平衡

| 市场状态 | 沪深300 PE | 调整动作 | 目标配置 |
|----------|------------|----------|----------|
| **极度低估** | <10倍 | 债券→权益 | 债券25% + 防御35% + 布局30% |
| **适度低估** | 10-12倍 | 小幅加仓权益 | 债券30% + 防御32% + 布局23% |
| **合理区间** | 12-16倍 | 维持平衡型 | 债券35% + 防御30% + 布局20% |
| **适度高估** | 16-20倍 | 减仓权益 | 债券40% + 防御25% + 布局15% |
| **极度高估** | >20倍 | 大幅减仓权益 | 债券50% + 防御20% + 布局10% |

### 7.2 基于时间的动态（2030节点）

```
2026-2029年（当前）：
• 维持平衡型或保守型配置
• 进攻权益小仓位试探（15-20%）
