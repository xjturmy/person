## 一、核心投资逻辑（为什么这样配置）

### 1.1 双周期框架

```
┌─────────────────────────────────────────────────────────────┐
│                    投资决策双层框架                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【第一层：康波周期 —— 50-60年长周期】                        │
│  ├── 当前阶段：第五次康波萧条期（2020-2030）                    │
│  ├── 战略配置：进攻权益 15-25%（萧条期上限）                   │
│  └── 调整频率：年度评估，重大转折时调整                        │
│                                                              │
│  【第二层：格雷厄姆指数 —— 短周期估值指标】                     │
│  ├── 计算公式：(1/PE-TTM) - 十年期国债收益率（差值法）        │
│  ├── 当前数值：2.49%（2026-04-14，显著低估）                   │
│  └── 调整频率：每周记录，月度评估，季度执行                    │
│                                                              │
│  【核心纪律】                                                 │
│  • 进攻权益始终在 10-30% 区间（萧条期约束）                    │
│  • 债券类始终 ≥25%（保持流动性底线）                          │
│  • 单季度调整 ≤±10%（避免过度交易）                          │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 1.2 为什么采用四分类配置

| 分类 | 核心功能 | 配置理由 |
|------|----------|----------|
| **债券类 25-35%** | 流动性储备 + 防御本金 | 萧条期现金为王，为极端机会储备弹药 |
| **黄金 15%** | 对冲货币贬值 + 地缘风险 | 康波萧条期典型特征，周金涛推荐配置 |
| **防御权益 25-30%** | 稳定现金流 + 低估值安全垫 | 高股息资产提供收益缓冲，优质资产恐慌时错杀 |
| **进攻权益 15-25%** | 布局第六次康波 + 长期成长 | AI+能源+生物是未来10-20年主线，当前小仓位试探 |

## 五、动态调整规则

### 5.1 基于格雷厄姆指数的战术调整

```
每月评估：
• 格雷厄姆指数 >3.0%：进攻权益→30%（上限）
• 格雷厄姆指数 2.0-3.0%：进攻权益→25-30%（当前所处）
• 格雷厄姆指数 1.0-2.0%：进攻权益→20%（基准）
• 格雷厄姆指数 0.5-1.0%：进攻权益→15-20%（减仓）
• 格雷厄姆指数 <0.5%：进攻权益→10%（下限）

触发条件：
• 格雷厄姆指数变化>0.3%：关注
• 格雷厄姆指数变化>0.5%：评估调整
• 格雷厄姆指数突破关键阈值（2.0%或3.0%）：执行调整
```

### 5.2 2030年战略调整预案

```
【情景1：回升期确认】
触发：AI生产率>1.5%/年，核聚变工程突破
调整：进攻权益 25-30% → 40-45%，债券类 25-30% → 20%

【情景2：萧条期延长】
触发：AI商业化进度低于预期，技术迟滞
调整：进攻权益维持25-30%，债券类增至35-40%，黄金增至20%

【情景3：金融危机】
触发：全球债务危机，股市>50%下跌
调整：进攻权益 25-30% → 10-15%，债券类增至45-50%，等待极端买点
```

## 六、核心纪律清单

### 6.1 约束条件（红线）

| 约束项 | 下限 | 上限 | 理由 |
|--------|------|------|------|
| 进攻权益 | 10% | 30% | 萧条期纪律，防风险+不错过机会 |
| 债券类 | 25% | 50% | 保持底线流动性 |
| 黄金 | 10% | 25% | 基础保险10%，地缘风险时增至25% |
| 单季度调整 | - | ±10% | 避免过度交易 |

### 6.2 操作纪律

**建仓纪律**：
- [ ] 现金比例不低于20%
- [ ] 单只标的不超过5%
- [ ] 分批买入，至少分3批
- [ ] 不追高涨幅>50%的标的

**持有纪律**：
- [ ] 防御权益以收息为目的，股息再投资
- [ ] 进攻权益以3-5年为持有单位
- [ ] 除非基本面恶化，否则不卖出防御个股

**卖出纪律**：
- [ ] 进攻权益单只跌幅>40%时强制review
- [ ] 防御个股基本面恶化（ROE连续下滑）时卖出
- [ ] 达到目标配置比例后不再追加
