# 彼得·林奇成长投资策略

> 基于彼得·林奇的投资理念，关注公司成长性、业务可理解性和PEG估值

---

## 策略概述

| 属性 | 内容 |
|------|------|
| **策略名称** | 彼得林奇成长策略 |
| **版本** | 1.0 |
| **核心思想** | 投资你了解的、寻找十倍股、PEG估值、分类管理 |
| **适用场景** | 成长股投资、业务可理解的标的选择 |

---

## 核心原则

1. **投资你了解的**：只投资业务简单可理解的公司
2. **寻找十倍股**：关注有巨大成长空间的标的
3. **PEG估值**：将PE与增长率对比，寻找PEG<1的标的
4. **分类管理**：根据公司类型（快速/稳健/周期/困境）采用不同策略

---

## 公司六分类体系

### 1. 缓慢增长型 (Slow Growers)

| 项目 | 内容 |
|------|------|
| **增长率** | 0-10% |
| **特征** | 大型成熟公司、稳定增长、高股息 |
| **示例** | 公用事业、银行股 |
| **策略** | 主要为了股息，增长预期较低 |

### 2. 稳健增长型 (Stalwarts)

| 项目 | 内容 |
|------|------|
| **增长率** | 10-20% |
| **特征** | 稳健增长、行业龙头、可预测性强 |
| **示例** | 消费龙头、医药公司 |
| **策略** | 10-15%仓位，稳定增长收益 |

### 3. 快速增长型 (Fast Growers)

| 项目 | 内容 |
|------|------|
| **增长率** | 20-50% |
| **特征** | 高速增长、规模扩张、再投资需求高 |
| **示例** | 科技成长股、新兴行业龙头 |
| **策略** | 5-8%仓位（风险较高），寻找PEG<1的标的 |

### 4. 周期型 (Cyclicals)

| 项目 | 内容 |
|------|------|
| **增长率** | 周期性波动 |
| **特征** | 需要择时、高风险高回报 |
| **示例** | 汽车、钢铁、航空 |
| **策略** | 5-10%仓位，在行业低谷买入，高峰卖出 |

### 5. 困境反转型 (Turnarounds)

| 项目 | 内容 |
|------|------|
| **增长率** | 困境反转 |
| **特征** | 高风险、需要深入研究 |
| **示例** | 重组公司、破产重整 |
| **策略** | 2-3%仓位（高风险），深入研究后参与 |

### 6. 隐蔽资产型 (Asset Plays)

| 项目 | 内容 |
|------|------|
| **特征** | 隐蔽资产、价值未被市场发现 |
| **策略** | 寻找账面价值被市场低估的资产 |

## 估值标准

### PEG比率（核心指标）

| 等级 | 目标值 | 说明 |
|------|--------|------|
| **目标** | < 1.0 | PEG = PE / 增长率 |
| **理想** | < 0.5 | 极具吸引力的成长股 |

### 其他估值指标

| 指标 | 最低要求 | 理想范围 | 说明 |
|------|----------|----------|------|
| **增长率** | 15% | 20-50% | 最低15%增长 |
| **PE范围** | < 40 | 20-30 | 成长股PE不超过40倍 |
| **ROE** | 15% | > 20% | ROE不低于15% |

## 基本面检查清单

### 盈利增长

| 时间维度 | 要求 | 说明 |
|----------|------|------|
| **最近季度** | > 20% | 季度盈利同比增长 |
| **过去12个月** | > 20% | 年度盈利增长 |
| **一致性** | 过去3年持续增长 | 增长稳定性 |

### 收入增长

| 指标 | 要求 | 说明 |
|------|------|------|
| **最近季度** | > 15% | 季度收入增长 |
| **加速趋势** | 增速在加快 | 而非放缓 |

### 盈利能力

| 指标 | 要求 | 说明 |
|------|------|------|
| **ROE** | > 15% | 净资产收益率 |
| **净利率** | 稳定或提升 | 盈利质量 |
| **经营现金流** | > 净利润 | 现金创造能力 |

### 资产负债表

| 指标 | 要求 | 说明 |
|------|------|------|
| **负债权益比** | < 50% | 财务安全性 |
| **现金状况** | 充裕或可控 | 流动性 |

---

## 研究六问

1. **这家公司是做什么的？**（业务简单性）
2. **这个业务有吸引力吗？**（市场前景）
3. **是什么让这家公司与众不同？**（竞争优势）
4. **增长来自哪里？**（增长驱动因素）
5. **现在估值合理吗？**（PEG<1？）
6. **机构是否过度持有？**（避免拥挤交易）

## 危险信号（Red Flags）

- [ ] 热门行业中的热门股（避免追高）
- [ ] 被吹捧为"下一个XX"的公司
- [ ] 多元化经营进入陌生领域（多元化折扣）
- [ ] 客户/供应商过度集中
- [ ] 应收账款增长快于收入增长
- [ ] 存货积压
- [ ] 高管离职或大量减持
- [ ] 利润率持续下滑

## 仓位管理

| 公司类型 | 建议仓位 | 说明 |
|----------|----------|------|
| **稳健型** | 10-15% | 稳定增长，可重仓 |
| **快速型** | 5-8% | 风险较高，控制仓位 |
| **周期型** | 5-10% | 需要择时，灵活调整 |
| **困境型** | 2-3% | 高风险，轻仓参与 |
| **单一标的上限** | 5% | 任何单一标的不超过5% |

---

## 卖出规则

### 基本面变化

- [ ] 增长率放缓至低于15%
- [ ] PEG升至1.5以上
- [ ] 核心业务出现衰退
- [ ] 发现更好的投资机会

### 估值过高

- [ ] PE超过40倍且PEG>1.5
- [ ] 市值达到合理估值上限

### 投资逻辑失效

- [ ] 投资逻辑被证伪
- [ ] 竞争优势消失
- [ ] 管理层重大失误

---

## 组合管理

| 管理维度 | 频率 | 内容 |
|----------|------|------|
| **关键指标跟踪** | 每月 | 增长率、PEG、财务数据 |
| **深度复盘** | 每季度 | 全面评估持仓 |
| **仓位再平衡** | 偏离目标时 | 涨幅导致仓位偏离时调整 |
| **现金管理** | 持续 | 保持5-10%现金用于机会捕捉 |
