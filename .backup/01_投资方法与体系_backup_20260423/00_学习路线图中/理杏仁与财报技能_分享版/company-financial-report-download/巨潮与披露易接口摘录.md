# 巨潮资讯（A股）与港交所披露易（港股）接口摘录

> **声明**：巨潮与港交所页面、接口可能变更；以下内容依据当前知识库内下载脚本整理，**以各平台实时规则与法律条款为准**。仅用于个人研究、合规频率内的资料整理。

---

## 1. A 股：巨潮资讯网（cninfo）

### 1.1 公告列表查询（POST）

| 项目 | 内容 |
|------|------|
| URL | `http://www.cninfo.com.cn/new/hisAnnouncement/query` |
| Content-Type | `application/x-www-form-urlencoded`（与浏览器一致即可） |
| 建议 Header | `X-Requested-With: XMLHttpRequest` |

**常用表单字段（脚本/浏览器）**

| 字段 | 说明 |
|------|------|
| `pageNum` | 页码，从 1 开始 |
| `pageSize` | 每页条数，如 30 |
| `column` | 市场板块：`szse`（深市）、`sse`（沪市）等 |
| `tabName` | 常为 `fulltext` |
| `stock` | **`{6位代码},{orgId}`**，英文逗号拼接；`orgId` 来自股票清单 JSON |
| `category` | 公告分类 key，财报类见下表 |
| `seDate` | 公告日期区间 `YYYY-MM-DD~YYYY-MM-DD` |
| `isHLtitle` | 如 `true` |

**定期报告类 `category`（深沪合并后缀 `_szsh`）**

| category | 用途 |
|----------|------|
| `category_ndbg_szsh` | 年度报告 → 档案命名常对应 **Q4** |
| `category_bndbg_szsh` | 半年度报告 → **Q2** |
| `category_yjdbg_szsh` | 一季度报告 → **Q1** |
| `category_sjdbg_szsh` | 三季度报告 → **Q3** |

更全分类 key 可参考（若链接变更请在站内查找同路径）：  
`http://www.cninfo.com.cn/new/js/app/disclosure/notice/history-notice.js`

**响应 JSON**：常见字段含 `announcements` 数组；元素含 `announcementTitle`、`announcementTime`（毫秒时间戳）、`adjunctUrl`（附件相对路径）等。

### 1.2 附件下载（GET）

若 `adjunctUrl` 为非空相对路径：

```text
http://static.cninfo.com.cn/ + adjunctUrl
```

保存为 PDF（或按响应 `Content-Type` 处理）。

### 1.3 A 股 `orgId` 清单（GET）

| URL | `http://www.cninfo.com.cn/new/data/szse_stock.json` |
|-----|-----------------------------------------------------|
| 用法 | 解析 `stockList`：`code` 为证券代码，`orgId` 为机构 id；代码建议 **左补零 6 位** 再匹配 |

## 2. 港股：披露易 HKEXnews

### 2.1 人工检索

- 首页（中文）：`https://www.hkexnews.hk/index_c.htm`  
- 通过「上市公司公告」标题检索，按公司名或股份代号查找 `Annual Report`、`Interim Report`、`Results Announcement` 等。

### 2.2 可复用检索 URL 参数（示例）

披露易支持带参数打开检索结果，常见参数：

| 参数 | 说明 |
|------|------|
| `lang` | `EN` 或 `ZH` |
| `market` | 主板常见 `SEHK` |
| `stockId` | 披露易内部公司 ID（比仅用股票代码稳定） |
| `category` | 如 `0` 表示标题检索默认分类 |

**示例（具体 `stockId` 需按公司在网页检索后获得）**

```text
https://www1.hkexnews.hk/search/titlesearch.xhtml?lang=EN&market=SEHK&stockId=1000249228&category=0
```
