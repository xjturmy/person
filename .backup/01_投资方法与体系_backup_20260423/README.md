## 四大模块说明

### 01_策略框架

**定位**：核心理论框架，建立投资哲学

| 文件 | 内容 | 应用场景 |
|------|------|----------|
| `芒格多元思维模型.md` | 跨学科思维模型 | 决策框架、认知升级 |
| `彼得林奇成长策略.md` | 成长股六分类、PEG估值 | 成长股筛选、公司分类 |
| `格雷厄姆深度价值策略.md` | 安全边际、定量分析 | 价值股筛选、防御配置 |
| `格雷厄姆投资法/` | 格雷厄姆式五步分析（商业模式→盈利→财务→估值→整合，PDF + 抽取 Markdown） | 个股深度分析、流程模板 |
| `彼得林奇分类与评估体系/` | 六类判定、战略/定量/复盘/总结（美的案例 PDF + 抽取 Markdown） | 个股深度分析、模板参考 |

### 02_分析流程

**定位**：操作手册，执行层面的具体方法

| 文件 | 内容 | 应用场景 |
|------|------|----------|
| `个股定量评估与决策体系.md` | 个股分析标准流程 | 公司研究、买入决策 |
| `公司出招全流程应对手册.md` | 应对不同市场情况 | 持仓管理、风险应对 |
| `行业ETF深度分析五步法.md` | 行业ETF选择方法 | 行业配置、ETF投资 |

### 03_市场认知与纪律

**定位**：投资纪律、心态管理、风险控制

| 文件 | 内容 | 应用场景 |
|------|------|----------|
| `0_投资体系框架与纪律准则.md` | 整体投资纪律 | 规则制定、行为约束 |
| `格雷厄姆指数估值区间速查表.csv` | 估值参考数据 | 快速估值判断 |
| `牛市中的风险.md` | 风险防范意识 | 市场过热时警示 |

### 04_投资书籍

**定位**：系统化阅读经典，构建知识体系

详见 `04_投资书籍/README.md`

## 使用路径

### 新手入门路径

1. **从纪律开始** → 阅读 `03_市场认知与纪律/0_投资体系框架与纪律准则.md`
2. **学习基础策略** → 阅读 `01_策略框架/格雷厄姆深度价值策略.md`
3. **掌握分析流程** → 阅读 `02_分析流程/个股定量评估与决策体系.md`
4. **持续学习** → 进入 `04_投资书籍/` 系统阅读经典

### 进阶深化路径

1. **策略对比** → 对比格雷厄姆与彼得林奇策略
2. **思维升级** → 学习 `01_策略框架/芒格多元思维模型.md`
3. **实操精进** → 反复练习 `02_分析流程/` 中的方法
4. **心态修炼** → 定期回顾 `03_市场认知与纪律/牛市中的风险.md`
