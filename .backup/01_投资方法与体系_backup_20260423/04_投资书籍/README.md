# 投资书籍学习模块

> 系统性阅读投资经典，构建完整的知识体系

---

## 模块目标

1. **建立知识框架**：通过经典书籍掌握投资的核心原理
2. **学习方法论**：培养深度阅读和批判性思考能力
3. **实践转化**：将书中理论转化为可操作的投资体系
4. **持续进化**：形成个人化的投资哲学

## 使用建议

### 对于初学者
1. 先阅读《阅读方法论.md》，掌握三层阅读法
2. 从 [01_书籍清单/00_书单总览.md] 的"初学者路径"开始
3. 每本书阅读后必须完成阅读分析文档

### 对于进阶者
1. 直接查看 [02_阅读分析/00_分析索引.md] 获取已完成分析的精华
2. 根据需要选择特定主题深入
3. 关注经济周期分析，用于战略资产配置

### 对于实践者
1. 重点关注各分析文档中的"检查清单转化"部分
2. 将可操作原则应用到实际投资决策
3. 定期回顾和更新阅读分析
