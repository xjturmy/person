# 价值投资大师理念关系图谱

> 基于《价值投资之道》及多本经典的投资理念传承与演变分析

---

## 一、核心图谱：师承与演变

```
格雷厄姆（奠基者）
    │
    ├──→ 巴菲特（学生+进化者）←── 费雪（并行影响）
    │         │
    │         └──→ 芒格（思想伙伴）
    │               │
    │               └──→ 现代价值投资体系
    │
    └──→ 彼得·林奇（集大成者+简化者）
```

## 二、五位大师一句话定位

| 大师 | 核心贡献 | 一句话定位 |
|------|----------|------------|
| **格雷厄姆** | 安全边际、定量分析 | "投资的安全基础"——教你如何不输 |
| **费雪** | 成长股15要点、闲聊法 | "成长的慧眼"——教你识别未来的伟大 |
| **巴菲特** | 护城河、买企业 | "价值与成长的融合"——以合理价格买伟大 |
| **芒格** | 多元思维、误判心理学 | "决策的智慧"——教你理性思考、避免愚蠢 |
| **彼得·林奇** | 六类分类、PEG | "散户的利器"——把复杂体系简化为可操作工具 |

## 三、理念演变的三大主线

### 主线1：从定量到定性

| 阶段 | 代表人物 | 核心特征 |
|------|----------|----------|
| 定量分析 | 格雷厄姆 | 清算价值、资产负债表、"烟蒂股" |
| 过渡 | 早期巴菲特 | 低PE/PB、账面价值、"便宜货" |
| 定性分析 | 后期巴菲特+芒格+费雪 | 护城河、管理层质量、商业模式、"伟大企业" |

### 主线2：从分散到集中

| 阶段 | 持股数量 | 核心逻辑 |
|------|----------|----------|
| 格雷厄姆 | 30-100只 | 统计套利、机械规则 |
| 现代价值 | 15-20只 | 深入研究、质量筛选 |
| 巴菲特/芒格 | 10-15只 | 深度理解、高质量集中 |

### 主线3：从短期到长期

| 阶段 | 持有期 | 卖出标准 |
|------|--------|----------|
| 格雷厄姆 | 2-3年 | 反弹即卖出 |
| 价值投资 | 3-5年 | 基本面恶化才卖出 |
| 巴菲特/芒格/费雪 | 永久 | 除非基本面质变 |

## 四、关键分歧与融合

### 格雷厄姆 vs 费雪/巴菲特

| 维度 | 格雷厄姆 | 费雪/巴菲特 |
|------|----------|-------------|
| 选股标准 | 低PE、低PB、定量为主 | 好公司、护城河、定性为主 |
| 安全边际 | 极度折扣（50%+） | 合理价格+企业质量 |
| 分散度 | 极度分散 | 适度集中 |
| 持有期限 | 2-3年反弹后卖出 | 永久持有 |

**巴菲特的融合**："以合理价格买优秀公司，比以极低价格买平庸公司更好。"

### 建议2：分层投资框架

| 层级 | 仓位 | 理念来源 | 核心策略 |
|------|------|----------|----------|
| **核心层** | 60-70% | 巴菲特+芒格 | 护城河+安全边际，长期持有 |
| **成长层** | 20-30% | 彼得·林奇+费雪 | 六类分类+PEG，3-5年持有 |
| **机会层** | 10% | 格雷厄姆+林奇 | 极度低估+困境反转，事件驱动 |

### 建议3：买入前检查清单

**理性分析（轨道A）**
- [ ] 能一句话解释商业模式？（能力圈）
- [ ] 有明确的经济护城河？（多尔西）
- [ ] ROE>15%持续5年？（巴菲特）
- [ ] 管理层德才兼备？（费雪）
- [ ] FCF/收入>5%？（多尔西）
- [ ] DCF估算内在价值？
- [ ] 安全边际>20%？（格雷厄姆）
- [ ] PEG合理？（林奇，成长股）

**心理检查（轨道B）**
- [ ] 是否喜欢？列出3个缺点
- [ ] 是否因别人推荐关注？
- [ ] 是否急于决定？→等待24小时
- [ ] 是否想弥补亏损？
- [ ] 是否使用杠杆？→绝对避免

**逆向思考**
- [ ] 什么情况下会失败？
- [ ] 护城河5年内会被侵蚀吗？
- [ ] 最大损失是多少？
- [ ] 有更好的替代机会吗？
