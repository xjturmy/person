## 使用说明

### 新增PDF存档流程

1. **确定分组**：根据阅读分析的分类，确定PDF应归入哪个目录
   - 价值投资 → `02_价值投资/`
   - 成长投资 → `03_成长投资/`
   - 经济周期 → `04_经济周期/`
   - 黄金投资 → `05_黄金投资/`

2. **移动文件**：将PDF从 `01_外部资料/` 移动到对应的分组目录

3. **更新索引**：在 `02_阅读分析/00_分析索引.md` 中标记"已读"并添加PDF来源链接

4. **更新本文件**：在本README的表格中添加新的对应关系

## 文件命名规范

建议的PDF文件命名格式：
```
《书名》副标题_作者.pdf
例如：《聪明的投资者》第4版_格雷厄姆.pdf
```

或对于研报类文件：
```
[机构]报告标题_日期.pdf
例如：[兴业证券]黄金投资框架研究_202312.pdf
```
