## 一、北京市场的特殊性

### 1.1 北京 vs 全国市场的差异

```
┌─────────────────────────────────────────────────────────────────┐
│                  北京市场 vs 全国市场                           │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│   维度              全国市场              北京市场               │
│   ────────────────────────────────────────────────────────       │
│   城市能级          三四线下跌            一线核心，抗跌性强      │
│   人口流向          流出/停滞             持续流入（疏解后趋稳）  │
│   政策敏感度        一般                  极高（限购最严）      │
│   教育资源          分散                  高度集中              │
│   地段稀缺性        一般                  极度稀缺              │
│   价格基数          已回调20-30%          回调幅度较小           │
│   流动性            差                    相对较好               │
│   租金回报率        1-2%                  1.5-2%                 │
│                                                                  │
│   【核心差异】                                                   │
│   全国处于"下行通道"，北京处于"高位震荡+结构性分化"             │
│   北京优质资产仍具保值属性，但需精选                             │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2 北京市场的周期位置（基于PDF框架）

**全国定位**：第五次康波萧条期 + 库兹涅茨周期下行筑底（2026年）

**北京定位**：**一线城市率先企稳窗口期**

| 指标 | 全国状态 | 北京状态 | 领先/滞后 |
|------|---------|---------|----------|
| **政策底** | 逐步放松中 | 限购仍严，边际放松 | 滞后 |
| **价格调整** | 回调20-30% | 回调5-15%（远郊），核心区微跌 | 抗跌 |
| **市场情绪** | 悲观 | 观望，但核心区仍有需求 | 分化 |
| **成交量** | 低位 | 核心区稳定，远郊冷清 | 分化 |

**关键判断**：
- 北京不会缺席全国周期，但调整幅度更小、时间更短
- 核心区（东西城、海淀、朝阳核心）可能已进入"震荡筑底"阶段
- 远郊区（房山、门头沟、延庆等）仍处于"下行通道"

## 五、个人行动计划（北京生活）

### 5.1 立即执行（本月内）

- [ ] **盘点现状**：明确当前房产情况、贷款余额、可动用资金
- [ ] **设定目标**：刚需/改善/投资，明确需求优先级
- [ ] **建立监控**：关注北京住建委月度成交数据、房贷政策变化
- [ ] **开始看房**：即使不立即买，也要了解市场（重点关注目标区域）

### 5.2 短期准备（3-6个月）

- [ ] **资金筹备**：如需贷款，提前查询征信、准备流水
- [ ] **区域研究**：深度研究2-3个目标板块，建立价格感知
- [ ] **房源筛选**：在贝壳/链家建立筛选条件，定期查看新增房源
- [ ] **谈判练习**：看10套以上，了解房东心态和议价空间

### 5.3 中期决策（6-18个月）

- [ ] **时机判断**：根据指标满足度决定是否入场
- [ ] **果断行动**：一旦决定，2-3个月内完成交易
