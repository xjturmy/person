# 权益调整决策树

> 基于 **格雷厄姆指数** + 市场信号，动态调整权益类持仓比例。

---

## 🎯 核心决策：格雷厄姆指数主导

### 第 1 层：格雷厄姆指数判断（必看）

**计算公式：**

```
格雷厄姆指数 = (1 / 沪深300 PE) ÷ 10年期国债收益率
```

**基准配置表：**

| 格雷厄姆指数 | 权益配置基准 | 动作方向 |
|------------|------------|---------|
| **> 2.5** | 75-85% | 🟢🟢 激进加仓 |
| **2.0 - 2.5** | 65-75% | 🟢 积极加仓 |
| **1.5 - 2.0** | 55-65% | 🟡 标准配置 |
| **1.0 - 1.5** | 40-55% | 🟡 保守配置 |
| **< 1.0** | 20-40% | 🔴 减仓防守 |

> 详见：[04_格雷厄姆指数.md](04_格雷厄姆指数.md)

## 🔧 完整决策流程

### 第 2 层：估值确认（辅助验证）

**加仓信号强烈：**

```
IF 格雷厄姆指数 > 2.5
   AND 沪深300 PE < 12
   AND PE 历史分位 < 20%
→ ✅ 积极加仓
```

**减仓信号强烈：**

```
IF 格雷厄姆指数 < 1.0
   AND 沪深300 PE > 16
   AND PE 历史分位 > 80%
→ ⚠️ 积极减仓
```

### 第 3 层：情绪修正

| 格雷厄姆指数 | 市场情绪 | 实际调整 |
|------------|---------|---------|
| **> 2.5 + 极度悲观** | 📉📉📉 | **基准 +10%**（逆向加仓） |
| **> 2.5 + 正常** | 📊 | 按基准配置 |
| **> 2.5 + 乐观** | 📈 | 按基准，不追高 |
| **< 1.0 + 极度乐观** | 🚀🚀🚀 | **基准 -10%**（逆向减仓） |
| **< 1.0 + 正常** | 📊 | 按基准配置 |
| **< 1.0 + 悲观** | 📉 | 按基准，不割肉 |

### 第 4 层：技术面执行（择时）

**确认买点（加仓时机）：**

- ✅ MA200 支撑有效
- ✅ 连续 3 天止跌
- ✅ 成交量放大回暖

**确认卖点（减仓时机）：**

- ⚠️ MA200 失守
- ⚠️ 连续放量下跌
- ⚠️ 创出新低

## 📋 执行规则

### 硬性规则

- ✅ 每月复核一次格雷厄姆指数
- ✅ 单次调整幅度不超过 **10%**
- ✅ 必须格雷厄姆指数 + 至少 2 个辅助信号确认
- ✅ 记录每次调整的决策和原因
- ✅ 分批执行，不 all-in 或 all-out

### 禁止行为

- ❌ 单凭短期情绪决策
- ❌ 忽略格雷厄姆指数盲目跟风
- ❌ 不设止损，扛到底
- ❌ 加杠杆追高
- ❌ 在极端情绪日做决策
