# 彼得林奇投资法 · A 股公司分类校正版

> 目的:把彼得林奇六类公司判断从美股原始口径,校正为更适合 A 股公司的分类入口。
> 核心原则:先判断“是否适用林奇通用法”,再判断生命周期类型。误判类型会直接误导 PEG、财务护栏和卖出条件。

---

## 一、先判断是否适用林奇通用法

| 公司/行业 | 通用林奇六类是否适用 | 正确处理 |
|---|---|---|
| 银行 | 不适用 | 转银行价值评估:PB、ROE、息差、不良率、拨备、分红 |
| 保险 | 不适用 | 转保险价值修复:PB/PEV、EV/NBV、偿付能力、投资收益率、股息 |
| 券商 | 不适用普通成长法 | 转券商周期估值:PB 分位、净资本、自营/经纪/投行业务周期 |
| 强周期资源/化工/航运/钢铁/煤炭 | 适用“周期型”,不适用 PEG | 主看 PB、库存、产能、商品价格、盈利拐点 |
| 新能源/汽车/科技制造 | 部分适用,常有双特征 | 主次分类:周期型 + 快速增长型,或快速增长型 + 周期型 |
| 白酒/家电/成熟消费 | 适用稳健/缓慢增长 | 主看 ROE、现金流、股息、估值分位,不追求高 CAGR |
| 医药 | 分子行业处理 | 创新药偏快速增长,仿制药/集采压力偏稳健或困境修复 |

---

## 二、A 股分类顺序

1. **行业适配**
   - 金融/保险/银行先排除通用林奇。
   - 强周期行业先归周期型。
   - 新能源、汽车链、科技制造允许主次类型。

2. **生命周期**
   - 高增长兑现:快速增长型。
   - 成熟龙头:稳定增长型。
   - 高股息低增长:缓慢增长型。
   - 经营/利润明显下滑且有修复迹象:困境反转型。
   - 资产价值显著被低估:隐蔽资产型。

3. **主次类型**
   - A 股很多公司不应只有一个标签。
   - 示例:比亚迪 = 周期型主 + 快速增长辅;宁德时代 = 快速增长主 + 周期辅;新华保险 = 不适用通用林奇,转保险专属。

## 三、分类参数校正

| 类型 | A 股判断口径 | 估值锚 | 关键误区 |
|---|---|---|---|
| 快速增长型 | 3 年/5 年营收 CAGR ≥20%,或先进制造/科技类 15%-20% 且 ROE/现金流能支撑 | PEG 分档 + PE/PS 分位 | 高景气行业容易周期化,不能只看增长 |
| 稳定增长型 | 消费/家电/成熟医药等龙头,CAGR 5%-15%,ROE 稳定 | PE 分位 + PEG + 股息 | 成熟龙头低增长不等于坏公司 |
| 缓慢增长型 | 增速低,但现金流和分红稳定 | 股息率 + PE/PB 分位 | 低增长但不分红,不是真正缓慢增长投资标的 |
| 周期型 | 行业供需、价格、库存主导盈利 | PB 分位 + 周期位置 | 低 PE 往往是周期顶部陷阱 |
| 困境反转型 | 利润/现金流明显受损,但出现修复信号 | PB 折价 + 现金流转正 | 不要把结构性衰退当反转 |
| 隐蔽资产型 | 现金、土地、牌照、股权等真实资产被低估 | NAV/PB 折价 | 资产不可变现会变成价值陷阱 |

## 四、增长连续性数据原则

1. **披露记录数不等于季度数**
   - A 股季报完整时,单季 YoY 可按 Q1/Q2/Q3/Q4 逐季统计。
   - 港股、新上市公司、部分年度披露不完整公司,常只有半年报和年报;不能把“2 条披露”简单等同于“2 个季度”。

2. **累计口径要按覆盖季度数计权**
   - H1 半年报覆盖 Q1-Q2,按 2 个季度计入连续性样本。
   - 年报若能用全年累计 - 半年累计拆出 H2,则 H2 覆盖 Q3-Q4,按 2 个季度计入。
   - 单季报覆盖 1 个季度。
   - 缺上一期累计值时,不强行把年报累计值当单季值。

3. **样本不足不判铁律成败**
   - 有效覆盖季度数 < 6/8 时,不能判定“快速增长铁律达标”,也不能判定“增长属性丧失”。
   - 评分应按已有样本命中率给临时分,并封顶。例如快速增长型:
     - 2/2 覆盖季度均 >20%,且最新仍强:可给较高观察分,但不满分。
     - 1/2 >20%:中等观察分。
     - 0/2 但最新仍为正:低观察分。
     - 0/2 且最新为负:低分。

4. **页面文案必须标注口径**
   - 若使用半年报/年报派生,应写明“覆盖期 YoY / 有效覆盖季度”,不要写成“单季 YoY 铁律已失败”。
   - 对蜜雪集团这类港股,增长连续性应优先解释为“样本不足 + 覆盖期增速强弱”,而不是机械套 8 季规则。

## 五、典型公司校准

| 公司 | A 股校正版分类 | 说明 |
|---|---|---|
| 新华保险 | 不适用通用林奇 | 保险看 EV/NBV、投资收益率、偿付能力,不是 PEG |
| 招商银行 | 不适用通用林奇 | 银行看 PB、ROE、息差、不良率、拨备 |
| 贵州茅台 | 稳定增长型 | 成熟高质量消费龙头,看 ROE、现金流、PE 分位 |
| 山西汾酒 | 稳定增长型 | 次高端白酒成熟龙头,不能因历史 CAGR 高就外推为快速增长 |
| 泸州老窖 | 稳定增长型 | 成熟白酒龙头,低增速阶段仍应看 ROE、现金流、分红和估值分位 |
| 美的集团 | 稳定增长型 | 家电龙头,负债率按家电口径放宽 |
| 伊利股份 | 缓慢/稳定边界 | 成熟消费,高股息和现金流权重提高 |
| 迈瑞医疗 | 稳定增长型 | 医疗器械平台型龙头,阶段性低增速不等于缓慢增长 |
| 三美股份 | 周期型 | 化工周期属性强,不用 PEG 判断买卖 |
| 中国中车 | 周期/资产属性 | 轨交设备偏订单周期和资产折价,不宜当普通成长股 |
| 比亚迪 | 周期型主 + 快速增长辅 | 汽车周期和新能源成长同时存在 |
| 宁德时代 | 快速增长主 + 周期辅 | 高成长,但电池价格/产能周期不可忽略 |
| 立讯精密 | 快速/稳定边界 | 消费电子周期 + 科技制造成长,需看客户集中和景气周期 |
| 恒瑞医药 | 稳定/快速边界 | 医药集采和创新药管线要分开看 |
| 科大讯飞 | 困境反转主 + 成长故事辅 | AI/软件空间仍在,但收入和利润兑现不足,先看修复验证 |
| 宁沪高速 | 缓慢增长型 | 高速公路偏成熟现金流和分红资产,不按交通运输一级行业粗分为周期 |
| 上海机场 | 困境反转型 | 机场枢纽更像客流/免税修复资产,主线是客流和商业合同恢复 |

## 六、落地规则

1. 前端先显示“是否适用通用林奇”。
2. 不适用时,不要再展示 PEG、通用负债率护栏等误导项。
3. 支持“主类型 + 次类型 + 权重”。
4. 筛选页和公司页统一读取同一分类器结果。
5. 每次新增公司时,先按本文件确认行业适配,再进入六类判断。
6. 增长连续性必须按“有效覆盖季度数”统计,样本不足时只给观察分,不判铁律成败。
