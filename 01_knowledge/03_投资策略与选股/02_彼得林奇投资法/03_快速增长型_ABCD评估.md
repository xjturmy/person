# 03 · 快速增长型 🚀 ABCD/12345 评估

> 公司质量 A/B/C/D × 价格吸引力 1/2/3/4/5 双维评分体系 — 适用于追求"十倍股"赔率的高速成长公司。
>
> **核心目标**:在合理价格押注高增长(3y CAGR ≥ 20-25%),追求"十倍股"赔率;严格防范泡沫(PEG > 2 即卖出)。
>
> **林奇语录**:"如果你不能用三句话讲清楚为什么买它,就别买。"

**核心:评估增长的动能、质量和财务安全。**

|------|----------|------------------|--------------|

|  | 增长连续性 | 15 分 | 过去 8 个季度单季收入同比增速:连续增长且 >20% 得 15 分;基本连续增长得 10 分;波动 >3 个季度得 3 分 |
|  | 市场空间/份额 | 10 分 | 公司当前营收占行业总潜在市场空间:**<1%** 得 10 分(蓝海);1%-5% 得 7 分;5%-10% 得 4 分;>10% 得 1 分 |
| **2. 商业模式与壁垒(30 分)** |  |  |  |
|  | 盈利模式可复制性 | 15 分 | 已验证的成功模式(店效/区域)可系统化复制:复制路径清晰且已验证得 15 分;有路径未验证得 8 分;无复制能力得 0 分 |

|  | 资产负债率 | 10 分 | 资产负债率:**<30%** 得 10 分;30%-40% 得 7 分;40%-50% 得 4 分;>50% 得 0 分(高度风险)|

**核心:为高增长支付合理溢价,但严格防范泡沫。**

| 类别 | 评价指标 | 评分标准(满分)| 量化评分细则 |
|------|----------|------------------|--------------|
| **1. 增长估值锚(70 分)** |  |  |  |
|  | PEG 估值 | 45 分 | PEG = 动态市盈率 / 未来 1-2 年预期盈利增长率。**PEG ≤ 1.0** 得 45 分;1.0-1.3 得 35 分;1.3-1.7 得 20 分;1.7-2.0 得 5 分;**> 2.0 得 0 分(泡沫警戒线)**|

(**基础分合计:100 分**)

## 第二部分:快速增长股专属调整因子

### 一、"好公司"调整因子(±15 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 增长加速 | **+8 分** | 最新季度营收/盈利增速环比显著提升(如从 25% 升至 40%+),且可持续 |
|  | 跨界成功验证 | **+5 分** | 成功复制模式到新区域或新产品线,首个完整季度数据大幅超预期 |
|  | 新产品/技术突破 | **+3 分** | 推出革命性产品或获得关键专利,可能开辟全新增长曲线 |
| **减分项** | 增长失速 | **-15 分** | 连续两个季度营收增速下滑 >10 个百分点,或利润增速下滑 >15 个百分点 |
|  | 单店/单模型效率恶化 | **-10 分** | 扩张中新业务的核心效率指标(ROI、坪效)持续显著低于成熟业务 |
|  | 客户/产品过度集中 | **-8 分** | 单一客户/产品收入占比 >40% 且存在流失风险 |

### 二、"好价格"调整因子(±10 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 非基本面错杀回调 | **+10 分** | 股价因市场恐慌/行业政策误读等非基本面问题,1 个月内下跌 >30%,公司基本面未变 |
|  | 估值滞后于业绩 | **+5 分** | 发布业绩大幅超预期后,股价涨幅显著低于盈利增幅,导致 PEG 被动下降 |
| **减分项** | 市场狂热与过度交易 | **-12 分** | 股价短期暴涨(如 1 个月内 >50%),成交量创历史天量,同时被主流媒体广泛报道为"下一个巨头" |
|  | 内部人/机构高位减持 | **-8 分** | 在估值高位(市盈率 > 历史 80% 分位),重要股东或长期机构投资者公告计划减持 |

## 第三部分:等级划分标准

### 一、公司质量最终等级(A/B/C/D)

| 等级 | 调整后总分范围 | 定性描述 |
|------|----------------|----------|
| **A 级** | **85 - 120 分** | **卓越增长**:增长强劲、模式清晰、财务稳健,具备了成为"十倍股"的主要潜质 |
| **B 级** | 70 - 84 分 | **良好增长**:增长不错,能见度较高,但可能在某些方面(如竞争壁垒、现金流)存在短板,潜力与风险并存 |
| **C 级** | 55 - 69 分 | **一般增长**:增长有故事,但或有明显缺陷(如高负债、增长依赖单一因素),未来不确定性大 |
| **D 级** | < 55 分 | **风险增长**:增长质量差,或存在重大风险点(如增长已失速、模式被证伪),需坚决回避 |

### 二、价格吸引力最终等级(1/2/3/4/5)

| 等级 | 调整后总分范围 | 定性描述 |
|------|----------------|----------|
| **1 级** | **85 - 120 分** | **高度低估**:增长与估值匹配度极佳,PEG 优势显著,市场关注度低,安全边际与上涨空间都很大 |
| **2 级** | 70 - 84 分 | **适度低估**:为增长支付了相对合理的价格。价格不算特别便宜,但若增长能持续,仍具有良好回报潜力 |
| **3 级** | 55 - 69 分 | **合理估值**:估值已充分反映市场对公司的增长预期。价格偏高,需要未来持续的超预期增长来支撑 |
| **4 级** | 40 - 54 分 | **估值偏高**:价格已部分透支未来 1-2 年的增长。安全边际薄,对增长的可证性要求极高 |
| **5 级** | < 40 分 | **严重高估**:估值泡沫化,无论什么故事都已过度定价,是彼得·林奇警告的"坚决卖出"或"回避"区域 |

## 第四部分:二维矩阵投资决策

**横轴**:公司质量(A/B/C/D 级)
**纵轴**:价格吸引力(1/2/3/4/5 级)

| 公司质量 \\ 价格 | 1 级(高度低估)| 2 级(适度低估)| 3 级(合理估值)| 4 级(估值偏高)| 5 级(严重高估)|
|------------------|-----------------|------------------|------------------|------------------|------------------|
| **A 级**(卓越增长)| **全力出击** 可配置战略性仓位,是最理想的投资机会 | **积极建仓** 核心机会,适合建立主要仓位并长期跟踪 | **持有/谨慎新增** 已有仓位可保留,等待估值消化或以小仓位试探 | **停止买入/减仓** 不再增加风险暴露,考虑部分获利了结 | **坚决卖出** 清仓离场,绝不犹豫 |
| **B 级**(良好增长)| **重点配置** 风险收益比优秀,可作为组合的重要部分 | **适度配置** 卫星仓位,需要密切关注其短板是否恶化 | **持有/跟踪** 可持有,但不宜追高。若增长证实可保持,否则减仓 | **考虑减仓** 性价比已低,应开始减持 | **卖出** 建议卖出 |
| **C 级**(一般增长)| **小仓位试探** 赔率吸引但胜率不高,必须严格控制仓位(如 <5%)| **少量试探** 仅在有明确催化剂时小量参与,并设严格止损 | **观望** 缺乏吸引力,不建议投入时间和资金 | **回避** 不参与 | **坚决回避** 不参与 |
| **D 级**(风险增长)| **投机买入** 保费级别高风险,仅适合极小仓位娱乐性参与 | **不参与** | **不参与** | **不参与** | **不参与** |

## 第五部分:与稳健增长股评估体系的差异对比

| 评估维度 | 稳健增长股体系 🛡️ | 快速增长股体系 🚀 | 核心差异 |
|----------|--------------------|---------------------|----------|
| **增长关注点** | 稳定性、连续性(5 年 CAGR 10-15%)| 爆发性、加速度(3 年 CAGR >20-25%)| 从"慢而稳"转向"快而猛",容忍短期波动,追求非线性增长 |
| **财务关键指标** | 低负债、强现金流、稳定股息 | 极低负债、现金流未必要正但需可控、无股息要求 | 对负债要求更苛刻(<40% vs <30%),对现金流的评价更动态 |
| **估值核心** | PEG ≤ 1,低股息率是缺点 | PEG 是唯一金标准,容忍 PEG ≤ 1.3,无视股息 | 对 PEG 的容忍度略高,但泡沫警戒线(PEG > 2)更严格 |
| **调整因子** | 侧重内部人、回购、治理、行业情绪 | 侧重增长加速/失速、模式验证、市场情绪、关注度 | 调整因子与增长驱动力和风险点直接相关 |
| **投资逻辑** | 在合理价格买入好资产,获取稳定复利 | 在合理价格押注高增长,追求巨大赔率 | 目标回报不同,因此风险偏好和评估侧重点不同 |

## 第六部分:核心纪律

### 1. 持有纪律

- **不止盈,看故事在不在轨**:涨 100% 不重要,关键看 PEG 是否仍 ≤ 1.5,基本面是否仍在加速
- **每季度复盘**:看故事兑现度,4 条卖出触发是否亮起

### 2. 4 条卖出触发(任一即严肃评估)

| # | 条件 | 阈值 | 含义 |
|---|------|------|------|
| ① | PEG > 2.0 | 泡沫警戒 | 估值反转 |
| ② | 连续 2 季单季 YoY < 20% | 增长断档 | 高速期可能终结 |
| ③ | 资产负债率 > 50% | 林奇铁律失守 | 财务恶化 |
| ④ | 故事兑现度 < 60% | 故事破裂 | 模式被证伪 |

### 3. 仓位控制

- A 级 + 1 级 → 战略性仓位(可达组合 10-15%)
- B 级 + 2/3 级 → 卫星仓位(3-5%)
- C 级 + 任何价格 → 试探仓位(< 3%)
- D 级 → 不参与

### 4. 林奇关键引语

> "在沃尔玛达到 1000 倍前,它一直是一只 fast grower。"
>
> "你需要做的就是检查它的故事是否在轨 — 如果是,就继续持有;如果不是,就立刻卖出。"
