# 02 · 稳健增长型 🛡️ ABCD/12345 评估

> 公司质量 A/B/C/D × 价格吸引力 1/2/3/4/5 双维评分体系 — 适用于年化 10-20% 可预测复利的"组合压舱石"。
>
> **核心目标**:挑选行业龙头中"可预测的复利机器",以合理价格(PEG ≤ 1.0)长期持有,获取稳定 10-20% 复合回报。
>
> **林奇语录**:"在合理价格买入好资产,获取稳定复利。涨 30-50% 就要轮动。"

**核心:评估增长稳定性、护城河耐久性、财务韧性。**

|  | 增长连续性 | 15 分 | 过去 8 季单季 YoY:连续 ≥10% 得 15 分;6/8 季 ≥10% 得 10 分;波动 >3 季得 3 分 |
|  | 净利率稳定性 | 10 分 | 5y 净利率标准差 / 均值:<10% 得 10 分;10-20% 得 6 分;>20% 得 0 分(波动太大不属稳健)|

### 二、"好价格"评分表(基础分 100 分)

**核心:为可预测的增长支付合理价格,PEG ≤ 1.0 是金标准。**

| 类别 | 评价指标 | 评分标准(满分)| 量化评分细则 |
|------|----------|------------------|--------------|
| **1. 增长估值锚(60 分)** |  |  |  |
|  | PEG 估值 | 35 分 | PEG = 动态 PE / 未来 1-2 年盈利增速。**PEG ≤ 0.8** 得 35 分;0.8-1.0 得 28 分;1.0-1.3 得 18 分;1.3-1.7 得 8 分;> 1.7 得 0 分(估值反转预警)|
|  | PE 历史分位 | 15 分 | 当前 PE-TTM 在自身 5y 历史分位:**< 30%** 得 15 分;30-50% 得 11 分;50-70% 得 6 分;>70% 得 0 分 |
|  | PB 与 ROE 匹配 | 10 分 | 用 ROE/PB 看资本效率:ROE/PB ≥ 5 得 10 分(性价比佳);3-5 得 6 分;<3 得 0 分 |
| **2. 股息与情绪(40 分)** |  |  |  |
|  | 股息率 | 15 分 | **≥ 3%** 得 15 分;2-3% 得 10 分;1-2% 得 5 分;<1% 得 0 分 |
|  | 股息率历史分位 | 10 分 | 当前股息率在自身 5y 分位:**> 70%** 得 10 分(历史高位 = 价格相对低);40-70% 得 6 分;<40% 得 0 分 |

(**基础分合计:100 分**)

## 第二部分:稳健增长股专属调整因子

### 一、"好公司"调整因子(±10 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 连续 5 年 ROE 稳定 ≥ 20% | **+8 分** | 5y ROE 均值 ≥ 20% 且无单年 < 15%;表明定价权稳固 |
|  | 持续股票回购(非债务驱动)| **+5 分** | 近 3 年回购金额 ≥ 净利润 5%,且自由现金流覆盖 |
|  | 行业龙头地位强化 | **+3 分** | 市占率从 N 升至 N+5pp,或竞争对手出清 |
| **减分项** | 增长停滞向缓慢增长滑落 | **-15 分** | 营收 CAGR 连续 2 年 < 5%,且 ROE 持续走低 |
|  | 主业被技术替代 | **-10 分** | 核心产品被新技术冲击(如 PC → 移动 / 燃油车 → EV)|
|  | 重大商誉减值 / 财务造假预警 | **-15 分** | 商誉减值 > 净资产 10%,或外部审计变更 |

### 二、"好价格"调整因子(±10 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 非基本面错杀 | **+10 分** | 因行业 panic 或政策误读,1 个月内跌 >20%,基本面未变 |
|  | 业绩超预期但股价滞涨 | **+5 分** | 季报营收/利润超预期 ≥ 10%,但股价 1 个月涨幅 < 增长幅度 |
| **减分项** | PEG 已 > 1.5 | **-12 分** | 估值开始透支未来增长 |
|  | 高位机构减持 | **-8 分** | PE 处于 5y > 70% 分位时,大股东 / 长期机构减持 ≥ 1% 流通股 |

## 第五部分:核心纪律与林奇语录

### 1. 持有纪律

> **林奇:"涨 30-50% 就要轮动。"**

- 稳健增长股不是"永久持有"标的 — 当股价涨幅显著超过盈利增幅(PEG 升至 1.3+)时,即使基本面没变,也应轮动到新的"PEG ≤ 1.0"标的
- 检查频率:每季度业绩出来后复查一次 PEG / 故事 / 4 条卖出触发

### 2. 卖出触发(任一即严肃评估)

| # | 条件 | 阈值 |
|---|------|------|
| ① | PEG > 1.7 | 估值反转 |
| ② | 连续 2 季单季 YoY < 10% | 增长断档 |
| ③ | 资产负债率 > 50% | 护栏失守 |
| ④ | 故事兑现度 < 60% | 故事破裂 |

### 3. 与稳健 vs 快速增长的区分线

| 维度 | 稳健 🛡️ | 快速 🚀 |
|------|---------|---------|
| 营收 CAGR | 10-20%(可预测)| ≥ 20%(高弹性)|
| 估值容忍 | PEG ≤ 1.0(严格)| PEG ≤ 1.5(放宽)|
| 持有周期 | 多年滚动 | 增长高速期(2-5 年)|
| 风险 | 增长停滞 → 缓慢增长 | 增长失速 → PEG 反转 |

### 4. 林奇关键引语

> "在沃尔玛涨 100 倍前,它已经是一只 stalwart 了 30 年。" — 不要小看稳健增长股的复利威力。
>
> "宝洁 / 可口可乐这类公司不会让你致富,但永远不会让你破产。"
