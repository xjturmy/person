# 06 · 隐蔽资产型 💰 ABCD/12345 评估

> 公司质量 A/B/C/D × 价格吸引力 1/2/3/4/5 双维评分体系 — 适用于"市值显著低于资产真实价值"的等待重估标的。
>
> **核心目标**:用 P/NAV(市值/净资产真实价值)< 0.6 的标的承接,等催化剂(分拆 / 出售 / 重估)兑现 60%-200% 折扣。
>
> **林奇语录**:"我最爱的隐蔽资产是不动产 — 在你查找时,它们在公司账上的成本是 30 年前的,但市价已经翻了 30 倍。"

---

## 第一部分:量化评分标准

### 一、"好公司"评分表(基础分 100 分)

**核心:评估"隐蔽资产是否真实存在且可识别",资产质量 + 折让幅度 + 催化剂可见度。**

| 类别 | 评价指标 | 评分标准(满分)| 量化评分细则 |
|------|----------|------------------|--------------|
| **1. 隐蔽资产识别度(40 分)** |  |  |  |
|  | 隐蔽资产类型清晰度 | 15 分 | 单一明确(如核心地段不动产 / 优质子公司股权 / 上市标的)得 15 分;有但模糊(如累积品牌价值)得 8 分;不明确 0 分 |
|  | 资产价值可估算度 | 15 分 | 公允价值有可比交易 / 公开估值报告得 15 分;可估但需假设得 10 分;高度主观得 0 分 |
|  | 财报附注披露 | 10 分 | 资产明细在年报附注详细披露得 10 分;部分披露得 6 分;隐瞒/含糊得 0 分(造假风险) |
| **2. 折让幅度与安全边际(35 分)** |  |  |  |
|  | NAV / 市值 | 20 分 | NAV ≥ 市值 × 2 得 20 分(深度折扣);1.5-2 倍 得 15 分;1.2-1.5 倍 得 8 分;< 1.2 倍 得 0 分(无折扣) |
|  | 现金 / 市值 | 10 分 | ≥ 50% 得 10 分(强下行垫子);30-50% 得 7 分;15-30% 得 3 分;< 15% 得 0 分 |
|  | PB 绝对水平 | 5 分 | < 0.7 得 5 分;0.7-1.0 得 3 分;>1.0 得 0 分(已无账面折扣) |
| **3. 催化剂可见度(25 分)** |  |  |  |
|  | 重组/分拆/出售公告 | 15 分 | 已公告 + 时间表明确 得 15 分;管理层明确表态 得 10 分;市场预期但未确认 得 5 分;无 0 分 |
|  | 大股东 / 管理层动作 | 5 分 | 内部增持 / 私有化要约 得 5 分;无 0 分 |
|  | 政策 / 行业触发 | 5 分 | 政策变化触发资产价值重估(如房产税 / 国企改革) 得 5 分;无 0 分 |

(**基础分合计:100 分**)

### 二、"好价格"评分表(基础分 100 分)

**核心:不看 PE/PEG(EPS 与隐蔽资产无关),看 P/NAV + 资产质量分散度。**

| 类别 | 评价指标 | 评分标准(满分)| 量化评分细则 |
|------|----------|------------------|--------------|
| **1. P/NAV 估值(50 分)** |  |  |  |
|  | P / NAV 倍数 | 35 分 | P/NAV ≤ 0.4 得 35 分(极致折扣);0.4-0.6 得 25 分;0.6-0.8 得 12 分;> 0.8 得 0 分 |
|  | 净流动资产 / 市值(NCAV) | 15 分 | (流动资产 - 全部负债) / 市值 ≥ 0.7 得 15 分(格氏 NCAV);0.4-0.7 得 10 分;<0.4 得 0 分 |
| **2. 资产质量(30 分)** |  |  |  |
|  | 资产流动性 | 15 分 | 主要为现金 / 上市股权 / 核心地段不动产 得 15 分;混合 得 8 分;主要为低流动性资产(品牌/专利)得 0 分 |
|  | 资产分散度 | 15 分 | 多类型隐蔽资产(地产 + 股权 + 现金) 得 15 分;两类 得 10 分;单一类型 得 5 分(集中风险) |
| **3. 等待成本(20 分)** |  |  |  |
|  | 股息率 | 10 分 | ≥ 4% 得 10 分(等待时有现金回报);2-4% 得 6 分;<2% 得 0 分(纯等待催化) |
|  | 主营业务现金流 | 10 分 | 主业 CFO 为正且持续 得 10 分;盈亏平衡 得 5 分;主业亏损吃资产 得 0 分 |

(**基础分合计:100 分**)

## 第二部分:隐蔽资产专属调整因子

### 一、"好公司"调整因子(±15 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 资产估值已被第三方背书 | **+10 分** | 评估机构 / 投行报告 / 同行业可比交易公开 |
|  | 大股东声誉良好 | **+5 分** | 大股东历史上有"主动释放价值"记录,非"长期占着茅坑"型 |
| **减分项** | 资产可能被低价转出 | **-15 分** | 控股股东与上市公司利益不一致,有侵占记录 |
|  | NAV 估算依赖大量主观假设 | **-10 分** | 关键资产无可比交易,价值估算误差区间 ±50% 以上 |
|  | 资产已被注入持续亏损子公司 | **-10 分** | 隐蔽资产被用于补贴亏损主业,价值持续蒸发 |

### 二、"好价格"调整因子(±15 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 资产价格本身在上行周期 | **+10 分** | 不动产 / 商品 / 子公司股权对应行业 PB 历史低位 |
|  | 大宗回购 / 私有化要约出现 | **+10 分** | 已公告或市场可信传闻 |
| **减分项** | 隐蔽资产对应的行业泡沫期 | **-10 分** | 资产价格本身处历史高位(可能 NAV 已透支) |
|  | 长期 PB 大幅折扣未消 | **-5 分** | 折扣已存在 ≥5 年仍无催化剂,可能是"价值陷阱" |

## 第三部分:等级划分标准(调整后总分)

### 一、公司质量最终等级 A/B/C/D

| 等级 | 调整后总分 | 定性描述 |
|------|------------|----------|
| **A 级** | **80 - 115** | 高质量隐蔽资产:类型清晰 + 折扣 ≥50% + 催化剂明确,中长期价值确定性高 |
| **B 级** | 60 - 79 | 良好隐蔽资产:折扣明显,但催化剂时间不确定 |
| **C 级** | 40 - 59 | 一般隐蔽资产:有折扣但披露不足或催化剂模糊 |
| **D 级** | < 40 | 价值陷阱嫌疑 / 隐蔽资产可能不存在或被侵占 |

### 二、价格吸引力最终等级 1/2/3/4/5

| 等级 | 调整后总分 | 定性描述 |
|------|------------|----------|
| **1 级** | **80 - 115** | 极致便宜:P/NAV ≤ 0.4 + 资产高流动性 — 接近"白送"水平 |
| **2 级** | 60 - 79 | 适度便宜:P/NAV 0.4-0.6 |
| **3 级** | 40 - 59 | 合理:P/NAV 0.6-0.8(已部分反映) |
| **4 级** | 20 - 39 | 偏贵:P/NAV > 0.8 |
| **5 级** | < 20 | 已反映:P/NAV ≥ 1.0(无安全边际) |

## 第四部分:二维矩阵投资决策

| 公司质量 \\ 价格 | 1 级(极致便宜)| 2 级(适度便宜)| 3 级(合理)| 4 级(偏贵)| 5 级(已反映)|
|------------------|-----------------|------------------|-------------|-------------|----------------|
| **A 级**(高质量)| **重点配置** 5-10% | 适度配置 3-5% | 持有 / 跟踪 | 减仓 | 卖出 |
| **B 级**(良好)| 适度配置 3-5% | 试探 1-3% | 跟踪 | 不参与 | 卖出 |
| **C 级**(一般)| 小仓位 ≤ 1% | 不参与 | 不参与 | 不参与 | 不参与 |
| **D 级**(陷阱嫌疑)| **不参与** | 不参与 | 不参与 | 不参与 | 不参与 |

**总仓位限制**:隐蔽资产标的累计 ≤ 总仓位 20%(等待时间长,机会成本)。

## 第五部分:核心纪律与林奇语录

### 1. 持有纪律

> **林奇:"隐蔽资产股的关键是'等' — 但你必须有等的资本和耐心。最大的敌人是机会成本。"**

- 介入前必须明确"等什么催化剂" — 写入故事脚本(如"等地产剥离 / 子公司分拆 / 大股东增持")
- 持有上限:**3 年**未见任何催化剂应严肃考虑止损,即使资产折扣还在
- 检查频率:**每半年** 复查 NAV 是否还成立 + 催化剂是否在路上

### 2. 卖出触发(任一即严肃评估)

| # | 条件 | 阈值 |
|---|------|------|
| ① | P/NAV 涨回 ≥ 0.9 | 折扣消失,价值兑现完成 |
| ② | 隐蔽资产被低价转出 | 大股东侵占,故事破裂 |
| ③ | 主业现金流持续亏损 ≥ 4 季 | 资产被吞噬,NAV 持续缩水 |
| ④ | 持有 ≥ 3 年仍无任何催化剂出现 | 价值陷阱兑现 — 不要继续等 |

### 3. 与其他类型的区别

| 维度 | 隐蔽资产 💰 | 困境反转 🔧 | 周期型 🔄 |
|------|--------------|-------------|------------|
| **核心赌注** | 价值重估(NAV → 市值)| 经营改善 | 周期反转 |
| **盈利重要性** | 不重要(资产为锚)| 改善必须 | 周期相关 |
| **估值口径** | P/NAV + NCAV | EV/Sales + 现金 | PB 历史分位 |
| **持有周期** | 等催化剂(1-3 年)| 改善期(2-3 年)| 半周期(2-5 年)|
| **失败模式** | 价值陷阱 / 资产被侵占 | 反转破产 | 周期高位被套 |

### 4. 林奇关键引语

> "Forest 公司在我手上时,它的市值低于它在缅因州拥有的森林市价 — 这是隐蔽资产最简单的版本。"

> "找一个市值低于现金及现金等价物的公司 — 听起来不可能,但确实有。" — Net-Net 思路

> "隐蔽资产里最容易识别的是不动产 — 公司账上是 30 年前的成本,但市价已经翻了 N 倍。"

## 第六部分:操作示例(假设按本框架打分)

### 示例:某老牌国企 YYY(2024 年评估)

**资产盘点**:
- 北京/上海核心地段办公楼(账面成本 1980s,市价估算 80 亿)
- 持有上市子公司 30% 股权(对应市值 30 亿)
- 现金 15 亿
- **NAV 估算 = 80 + 30 + 15 = 125 亿**;当前市值 = 50 亿 → P/NAV = 0.40

**好公司评分**:

| 项 | 数据 | 评分 |
|----|------|------|
| 隐蔽资产类型清晰度 | 不动产 + 上市股权 (清晰) | 15 / 15 |
| 资产价值可估算度 | 不动产有公开评估 + 子公司有市价 | 15 / 15 |
| 财报附注披露 | 详细 | 10 / 10 |
| NAV / 市值 | 125 / 50 = 2.5 倍 | 20 / 20 |
| 现金 / 市值 | 30% | 7 / 10 |
| PB 绝对水平 | 0.65 | 5 / 5 |
| 重组分拆公告 | 国企改革背景下管理层表态 | 10 / 15 |
| 大股东动作 | 无 | 0 / 5 |
| 政策触发 | 国企改革 | 5 / 5 |
| **基础分** |  | **87** |
| 调整(资产已第三方评估 +10) |  | +10 |
| **最终分** |  | **97** → **A 级** |

**好价格评分**:

| 项 | 数据 | 评分 |
|----|------|------|
| P/NAV | 0.40 | 35 / 35 |
| NCAV / 市值 | 0.55 | 10 / 15 |
| 资产流动性 | 上市股权 + 不动产(中等) | 8 / 15 |
| 资产分散度 | 三类(地产 + 股权 + 现金) | 15 / 15 |
| 股息率 | 4.2% | 10 / 10 |
| 主营 CFO | 持平 | 5 / 10 |
| **基础分** |  | **83** → **1 级**(极致便宜)|

**矩阵决策**:A 级 + 1 级 = "重点配置 5-10%"。

## 第七部分:数据来源

| 数据点 | 来源 | 备注 |
|--------|------|------|
| 现金及等价物 | DuckDB `balancesheet`(理杏仁 fs API)| 单独字段 |
| 净流动资产 NCAV | DuckDB `balancesheet`(流动资产 - 全部负债)| 自算 |
| 上市子公司股权 | 财报附注 / 巨潮资讯网 | **手填**到 02_companies/{N}/03_行业分析/隐蔽资产盘点.md |
| 不动产估值 | **手填**(参考贝壳找房 / 安居客 / 评估报告) | 误差大,需多源核对 |
| NAV 综合 | 手算 | 现金 + 股权市值 + 不动产 + 其他 |
| P/NAV | 市值 / NAV | 自算 |
| 主业 CFO | DuckDB `cashflow`(去掉投资/筹资) | 同源 |
