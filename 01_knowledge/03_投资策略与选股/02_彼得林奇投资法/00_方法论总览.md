## 一、五步决策法(全类型通用)

```
┌──────────────────────────────────────────────────────────────────┐
│                   林奇五步成长投资框架                              │
├──────────────────────────────────────────────────────────────────┤
│  ① 公司分类 → 六类之一,定下后续评估口径                            │
│  ② 成长核查 → CAGR 速率 / 季度连续 / 销量驱动(三层证据)          │
│  ③ 财务护栏 → 类型驱动的负债率 / 现金流 / 周转阈值                  │
│  ④ PEG 估值 → 成长合理性(部分类型不适用,见下表)                   │
│  ⑤ 故事更新 → 每季 ping 兑现度 + 4 条卖出触发                      │
└──────────────────────────────────────────────────────────────────┘
```

**核心心法**:
> "如果你不能用三句话讲清楚为什么买它,就别买。" — 彼得·林奇

每只股票必须落到一份**故事脚本**:
- **🎯 一句话**:为什么买
- **✅ 验证证据**:从财报派生的支撑数据
- **❌ 不会发生的事 / 卖出信号**:故事破裂的明确信号

## 三、双维 ABCD/12345 评估体系总览

林奇核心创新:**用同一套决策语言评估不同类型公司**。

### 横轴 — 公司质量等级 A/B/C/D(从评分表得出)

| 等级 | 调整后总分 | 定性描述 |
|------|-----------|----------|
| **A 级** | 85-120 | 卓越 / 周期王者 — 顶配,可重仓 |
| **B 级** | 70-84  | 良好 / 周期强者 — 核心仓位候选 |
| **C 级** | 55-69  | 一般 / 周期跟随 — 缺陷明显,小仓试探 |
| **D 级** | < 55   | 风险股 / 周期挣扎 — 坚决回避 |

### 纵轴 — 价格吸引力等级 1/2/3/4/5(从价格评分表得出)

| 等级 | 调整后总分 | 定性描述 |
|------|-----------|----------|
| **1 级** | 85-120 | 高度低估 / 深度萧条 — 全力出击 |
| **2 级** | 70-84  | 适度低估 / 复苏初期 — 积极建仓 |
| **3 级** | 55-69  | 合理估值 / 景气上升 — 持有/谨慎新增 |
| **4 级** | 40-54  | 估值偏高 / 景气高点 — 减仓 |
| **5 级** | < 40   | 严重高估 / 狂热泡沫 — 坚决卖出 |

### 决策矩阵(通用模板,各类型阅读路径不同)

| 公司质量 \\ 价格 | 1 级(低估) | 2 级 | 3 级 | 4 级 | 5 级(高估) |
|------------------|------------|------|------|------|--------------|
| **A 级** | 全力出击 | 积极建仓 | 持有/谨慎新增 | 停止买入/减仓 | 坚决卖出 |
| **B 级** | 重点配置 | 适度配置 | 持有/跟踪 | 考虑减仓 | 卖出 |
| **C 级** | 小仓位试探 | 少量试探 | 观望 | 回避 | 坚决回避 |
| **D 级** | 投机买入 | 不参与 | 不参与 | 不参与 | 不参与 |

> **关键认知**:不同类型的"价格吸引力 1 级"含义不同:
> - **稳健 / 快速增长**:绝对估值低 + 增长合理(PEG 视角)
> - **周期型**:周期位置在底部(产品价格低位 + 库存出清),与"绝对估值"无关

## 四、三类核心差异对比

| 维度 | 稳健增长 🛡️ | 快速增长 🚀 | 周期型 🔄 |
|------|-------------|-------------|------------|
| **增长关注点** | 稳定性/连续性<br>5y CAGR 10-15% | 爆发性/加速度<br>3y CAGR ≥ 25% | 周期位置<br>不看 CAGR |
| **核心估值锚** | PEG ≤ 1.0(金标准)| PEG ≤ 1.5(容忍更高)+ PS | **PB 历史分位**<br>(PE 反向解读)|
| **PEG 适用?** | ✅ 是 | ✅ 是(金标准)| ❌ 否(EPS 大幅波动) |
| **负债率上限** | ≤ 50% | **≤ 40%(林奇铁律)** | ≤ 55% |
| **现金流要求** | CFO/NI ≥ 0.9 | CFO/NI ≥ 1.0 | 低谷期 ≥ 1.0 |
| **股息要求** | 稳定分红是加分 | 不要求 | 不要求 |
| **卖出阈值** | 估值 + 故事破裂 | PEG > 2.0 / YoY < 20% | 周期高点(产品价格 5y >80%)|
| **持有周期** | 多年(滚动复利)| 增长高速期(2-5 年)| **半周期**(2-5 年随行业)|
| **林奇语录** | "找到能涨几倍但不会破产的安全公司" | "如果不能 3 句话说清楚为什么买,就别买" | "在绝望中买入,在希望中卖出" |

## 六、PEG 估值口径(类型分化)

| 类型 | 是否用 PEG? | 阈值 | 备注 |
|------|--------------|------|------|
| 缓慢增长 🐢 | ❌ 不用 | — | 看股息率(≥ 4%)+ 派息历史(≥ 10 年) |
| **稳健增长 🛡️** | ✅ 是 | **PEG ≤ 1.0 是核心** | 涨 30-50% 即考虑轮动 |
| **快速增长 🚀** | ✅ 是 | **PEG ≤ 1.5 合理 / > 2.0 泡沫** | 容忍度比稳健略高 |
| **周期型 🔄** | ❌ 不用 | — | 用 PB 历史分位 + 产品价格周期位置 |
| 资产隐蔽 💰 | ❌ 不用 | — | 看市值 / NAV(资产真实价值) |
| 困境反转 🔧 | ❌ 不用 | — | EPS 不稳,PEG 失真 |

## 七、4 条通用卖出触发条件

无论什么类型,任一触发即"严肃评估卖出":

1. **PEG > 2.0** — 估值反转(适用稳健 / 快速)
2. **连续 2 季单季 YoY < 阈值** — 增长断档(快速 < 20%,稳健 < 10%)
3. **资产负债率 > 阈值** — 护栏失守(类型驱动,见上表)
4. **故事兑现度 < 60%** — 故事破裂(每季 ping)

## 八、林奇 25 条投资金律(精选 10 条)

1. **赚钱靠胃,不靠脑** — 在恐慌时买入,在贪婪时卖出。
2. **熟悉的最赚钱** — 投资你了解、用过、亲自见过的公司。
3. **股市没有时间表** — 长期持有好公司,不要被短期波动吓跑。
4. **公司决定一切** — 短期股价是投票机,长期股价是称重机。
5. **业绩永远跑赢经济学家** — 关注公司财报,而非宏观预测。
6. **PEG ≤ 1 是甜蜜区** — 这是判断成长股是否便宜的核心。
7. **避开"明日的明星"宣传** — 当所有分析师追捧一只股票,通常是顶部。
8. **不熟不投** — 不懂的行业(如生物科技专利、复杂金融衍生品)直接放弃。
9. **小公司 vs 大公司** — 小公司更可能"涨 10 倍",大公司更可能"涨 1 倍"。
10. **季度跟踪比初次买入更重要** — 故事还在轨吗?基本面在改善还是恶化?

## 九、故事脚本最小模板(每只股票一份)

```markdown
🎯 一句话:[行业] 龙头 / 早期玩家 / 反转标的,[公司] 凭借 [核心优势] 在 [市场空间] 中占据 [位置]

✅ 验证证据(从财报派生 + 行业/管理层信号):
• [量化指标 1] — 如:营收 5y CAGR 18%(高速扩张)
• [量化指标 2] — 如:ROE 32% 长期保持(资本效率)
• [质性信号]   — 如:管理层 2026 Q1 提价但销量不降(定价权)

❌ 不会发生的事 / 卖出信号:
• [核心反证 1] — 如:连续 2 季营收 YoY < 20%
• [核心反证 2] — 如:资产负债率 > 50%
• [核心反证 3] — 如:管理层减持超 5%
```
