# 04 · 周期型 🔄 ABCD/12345 评估

> 公司质量 A/B/C/D × 周期位置 1/2/3/4/5 双维评分体系 — 适用于盈利受行业周期主导的公司(择时重于选股)。
>
> **核心目标**:在行业周期的特定位置,挑选出具有财务韧性的公司,以合理的价格进行投资,并在行业过热时坚决离场。
>
> **林奇语录**:"投资周期股不是投资公司,而是投资周期。在绝望中买入,在希望中卖出。"

---

## 第一部分:量化评分标准(基础分各 100 分)

### 一、"好公司"评分表(基础分 100 分)

**核心:评估公司在景气低谷中生存并在复苏时弹性的能力。**

| 类别 | 评价指标 | 评分标准(满分)| 评分细则(关键数据点)|
|------|----------|------------------|------------------------|
| **1. 财务安全与韧性(50 分)** |  |  |  |
|  | 偿债安全边际 | 25 分 | (现金及等价物 + 经营现金流) / 总债务:**>0.4** 得 25 分;0.3-0.4 得 18 分;0.2-0.3 得 10 分;<0.2 得 0 分 |
|  | 经营现金流质量 | 15 分 | 近 2 年行业低谷期,经营现金流 / 净利润:均 >1.0 得 15 分;偶尔 <1 得 8 分;常为负得 0 分 |
|  | 债务结构与成本 | 10 分 | 短期债务 / 总债务:<30% 得 10 分;30%-50% 得 6 分;>50% 得 0 分。有长期低息债务是加分项 |
| **2. 行业结构与成本优势(35 分)** |  |  |  |
|  | 成本竞争力 | 20 分 | 公司主要产品的单位现金成本处于行业内最低的 20%:20 分;前 40%:12 分;中等:5 分;成本高企:0 分 |
|  | 市场份额与地位 | 15 分 | 公司核心业务市场份额在 5 年内是否稳定或提升:行业前三且份额稳定得 15 分;前五得 10 分;份额持续流失得 0 分 |
| **3. 管理层周期管理能力(15 分)** |  |  |  |
|  | 低谷期战略 | 10 分 | 在上一个行业低谷期,是否进行了有效的成本削减、产能优化或战略收缩:是得 10 分;无所作为或错误扩张得 0 分 |
|  | 资本开支纪律 | 5 分 | 公司资本开支是否具有反周期性(在景气高点克制、低点审慎投资):是得 5 分;顺周期大肆扩张得 0 分 |

(**基础分合计:100 分**)

### 二、"好价格"评分表(基础分 100 分)

**核心:识别周期位置 — 在"过冷"时买入,在"过热"时卖出。**

| 类别 | 评价指标 | 评分标准(满分)| 评分细则(关键数据点)|
|------|----------|------------------|------------------------|
| **1. 行业景气度与先行指标(60 分)** |  |  |  |
|  | 产品价格/利润位置 | 30 分 | 主要产品价格:处于 5 年历史分位 **<20%**:30 分;20%-40%:20 分;40%-60%:10 分;>60%:0 分(分位越低,价格越低)|
|  | 行业库存与产能 | 20 分 | 行业总库存/销售量比率:处于 2 年高位并开始下降:20 分;仍在上升:0 分。行业产能利用率低于长期均值 >5%:加 10 分 |
|  | 行业资本开支 | 10 分 | 行业内竞争对手未来 1-2 年无大规模新增产能计划:10 分;仍有大规模计划:0 分 |
| **2. 市场情绪与估值(40 分)** |  |  |  |
|  | 历史估值分位 | 20 分 | 公司**市净率(PB)**处于自身 5 年历史分位:**<10%** 得 20 分;10%-30% 得 15 分;30%-50% 得 8 分;>50% 得 0 分 |
|  | 市场情绪与关注度 | 20 分 | 机构持股比例与卖方分析师覆盖数:两者均低(关注度低)得 20 分;一方低得 12 分;两者均高(热门)得 0 分 |

(**基础分合计:100 分**)

> **🔑 关键认知**:周期股**用 PB 而不是 PE**评估估值。
> - **PE 反向解读**:周期股 PE 最低时(行业景气高点)反而是**顶部**;PE 最高时(亏损期 / 微利)往往是**底部**
> - **PB 顺向解读**:PB 在 5y 低位(<10%)= 周期底部 = 买入窗口

## 第二部分:周期股专属调整因子(±10 分)

### 一、"好公司"调整因子

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 困境中优化资产负债表 | **+8 分** | 在行业下行期,成功通过资产出售、股权融资等方式,大幅降低杠杆,改善债务结构 |
|  | 获得低成本长期资金 | **+5 分** | 在行业低迷期,能以极低利率获得长期贷款或发行债券,为复苏储备"弹药" |
| **减分项** | 在景气高点进行巨额并购 | **-15 分** | 在行业市盈率创周期新高时,宣布以极高溢价进行大规模并购,明显"买在顶部" |
|  | 关键融资渠道枯竭 | **-10 分** | 在需要资金时,主要银行授信被削减或信用评级被下调,再融资能力受质疑 |

### 二、"好价格"调整因子

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 宏观经济政策强刺激 | **+10 分** | 针对该行业的强刺激政策(如汽车家电下乡、基建投资)出台,而股价尚未反应 |
|  | 行业出现标志性破产/整合 | **+8 分** | 行业内一家主要高成本竞争对手宣布破产或被收购,供给端出现实质性收缩 |
| **减分项** | 媒体与分析师集体唱多 | **-15 分** | 当主流媒体和经济学家一致宣称"超级周期到来"、"这次不一样"时 |
|  | 产品价格创周期新高但股价滞涨 | **-10 分** | 产品价格已远超上一轮周期高点,但公司股价上涨乏力,显示盈利无法传导 |

## 第三部分:等级划分标准(调整后总分)

### 一、公司质量最终等级(A/B/C/D)

| 等级 | 调整后总分范围 | 定性描述 |
|------|----------------|----------|
| **A 级** | **85 - 115 分** | **周期王者**:财务极稳健,成本领先,能在寒冬中存活并能在春天最先复苏 |
| **B 级** | 70 - 84 分 | **周期强者**:财务状况良好,有一定成本优势,能跟随行业复苏 |
| **C 级** | 55 - 69 分 | **周期跟随者**:财务有压力,成本无优势,生存依赖行业整体回暖 |
| **D 级** | < 55 分 | **周期挣扎者**:高负债、高成本,可能无法熬过下一个行业冬天 |

### 二、价格吸引力最终等级(1/2/3/4/5)

> 此处的"价格吸引力"特指"周期位置吸引力"。

| 等级 | 调整后总分范围 | 定性描述(对应的周期位置)|
|------|----------------|-----------------------------|
| **1 级** | **85 - 118 分** | **深度萧条期**:产品价格处于历史低位,全行业亏损,产能大幅出清,市场极度悲观。**最佳买入区域** |
| **2 级** | 70 - 84 分 | **复苏初期**:产品价格企稳或小幅反弹,部分企业开始扭亏,但市场将信将疑。**积极买入区域** |
| **3 级** | 55 - 69 分 | **景气上升期**:盈利快速增长,股价已部分反应,市场关注度提升。持有/观望区域 |
| **4 级** | 40 - 54 分 | **景气高点**:盈利创历史新高,估值不再便宜,市场情绪乐观。停止买入/减仓区域 |
| **5 级** | < 40 分 | **狂热泡沫期**:产品价格和公司市盈率均创历史新高,媒体狂热,分析师目标价不断上调。**坚决卖出区域** |

## 第四部分:二维矩阵投资决策

**横轴**:公司质量(A/B/C/D 级)
**纵轴**:周期位置吸引力(1/2/3/4/5 级)

| 公司质量 \\ 周期位置 | 1 级(深度萧条)| 2 级(复苏初期)| 3 级(景气上升)| 4 级(景气高点)| 5 级(狂热泡沫)|
|-----------------------|------------------|-------------------|-------------------|-------------------|-------------------|
| **A 级**(周期王者)| **全力出击** 最佳时机,可重仓布局 | **积极买入** 确定性高,适合建立核心仓位 | **持有** 享受业绩增长,但不再追加投资 | **分批减仓** 开始兑现利润 | **清仓卖出** 不留一股,绝不恋战 |
| **B 级**(周期强者)| **重点配置** 良好时机,可作为重要持仓 | **适度买入** 可以配置,但仓位应低于 A 级公司 | **持有/中性** 可跟随行业上涨,准备退出计划 | **大幅减仓** 风险增大,保留少量观察仓即可 | **坚决卖出** |
| **C 级**(周期跟随者)| **谨慎投机** 仅限极小的试探性仓位,赌行业反转 | **不参与或极小仓** 缺乏安全边际,风险过高 | **不参与** | **不参与** | **不参与** |
| **D 级**(周期挣扎者)| **坚决回避** 即便股价再低,也可能破产归零 | **坚决回避** | **坚决回避** | **坚决回避** | **坚决回避** |

## 第五部分:体系核心逻辑与应用要点

### 1. 核心理念

> 投资周期股不是投资公司,而是投资周期。
> "在你确认行业即将复苏时买入,在所有人确认行业繁荣时卖出。" — 林奇

### 2. 核心纪律

- **只投 A 级和 B 级公司**:确保投的是能在萧条中活下来的强者。C 级和 D 级公司是价值陷阱
- **只在 1 级和 2 级位置买入**:确保买在周期的底部区域。在 3 级及以后的位置,买入风险远大于收益
- **在 4 级和 5 级位置坚决卖出**:周期股**绝不做"长期持有"**,必须在市场狂热时兑现利润

### 3. 数据重点

**高度重视行业层面的先行指标**(产品价格、库存、产能),而非仅仅关注公司自身的季度财报:

| 行业 | 关键先行指标 |
|------|--------------|
| 钢铁 / 煤炭 | 产品现货价格 + 库存周转 + 行业开工率 |
| 化工 | 产品价差(产品价 − 原料价)+ 库存 |
| 航运 | BDI / CCFI 运价指数 |
| 汽车 / 家电 | 单月销量 + 渠道库存 |
| 房地产 | 销售面积同比 + 土地成交楼面价 |
| 半导体 | 费城半导体指数 + 设备订单(BB ratio)|

### 4. 与稳健/增长股体系的统一性

- 本体系依然使用 **A/B/C/D** 和 **1/2/3/4/5** 的通用范式,方便投资者将所有股票放入同一套决策语言中
- 通过调整评分标准和调整因子,完美适配了周期股**"择时重于选股、位置决定一切"**的特殊性

## 第九部分:常见误区警告

1. **❌ 用 PE 评估周期股** — 周期股 PE 最低时反而是顶部;一定用 PB
2. **❌ 长期持有周期股** — 周期股不是 stalwart,必须择时
3. **❌ 在媒体一致唱多时买入** — 这通常是 5 级狂热泡沫期
4. **❌ 投 C/D 级公司赌反转** — 即便周期来了,C/D 级公司可能先破产
5. **❌ 忽视行业先行指标只看财报** — 财报是滞后指标,先行指标是产品价格 + 库存
6. **❌ 把"价格便宜"等同于"低估"** — 价格便宜是结果,需要先看是否在周期底部位置
