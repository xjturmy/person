# 07 · PEG 估值速查

> 林奇成长投资体系的"估值锚",一个公式贯穿稳健 / 快速增长两类公司。
>
> **公式**:**PEG = PE-TTM ÷ (净利润 5y CAGR × 100)**(理杏仁同口径,与 [peg_curve.py](../../../.tools/dashboard/peg_curve.py) 一致)
>
> **林奇语录**:"如果一个公司 PE 等于增长率,它的价格是合理的。如果 PE 是增长率的两倍,你正在为它付太多。"

---

## 一、PEG 5 档评级表

| PEG 区间 | 评级 | 林奇判断 | 操作建议 |
|:---:|:---:|---|---|
| **< 0.5** | 🟢🟢 极度低估 | 价格远未跟上增长,严重错杀 | **重仓买入**(快速类核心仓 8-12%) |
| **0.5 - 1.0** | 🟢 合理偏低 | 林奇"甜蜜区",成长股黄金买点 | **建仓买入**(快速 6-10% / 稳健 8-12%) |
| **1.0 - 1.5** | 🟡 略贵 | 价格已跟上增长,无折扣无溢价 | **持有 / 谨慎新增** |
| **1.5 - 2.0** | 🔴 高估 | 已透支未来增长 | **减仓至跟踪仓 ≤ 2%** |
| **> 2.0** | 🔴🔴 严重高估 | 估值反转倒计时,泡沫信号 | **坚决卖出** |

> 💡 **类型微调**:
> - **稳健增长型 🛡️**:阈值更严 — PEG ≤ 0.8 才算"低估",1.7 即"高估"
> - **快速增长型 🚀**:容忍更高 — PEG ≤ 1.0 是"低估",2.0 才"高估"

---

## 二、PEG 不适用清单(关键!)

PEG **不适用** 的 4 类公司,强行计算会得出**严重误导**结论:

| 类型 | 为什么不适用 | 替代估值口径 |
|------|--------------|--------------|
| **🐢 缓慢增长型** | CAGR < 10% 时,PEG 数学失真(分母太小,小变化导致 PEG 大幅波动) | **股息率(≥4%)+ PE 绝对水平(<15)+ 派息历史 ≥10 年** |
| **🔄 周期型** | EPS 周期波动剧烈,顶部 PE 极低、底部 PE 极高(反过来),PEG 完全失真 | **PB 历史分位 < 30% + 产品价格 5y 分位** |
| **🔧 困境反转型** | EPS 为负或大幅波动,CAGR 无意义 | **EV/Sales + 现金/市值 + PB 历史分位** |
| **💰 隐蔽资产型** | EPS 与隐蔽资产价值脱钩,PEG 无信息含量 | **P/NAV(市值/净资产真实价值)+ 净流动资产(NCAV)** |

**判定流程**:遇到亏损 / 周期 / 隐蔽 → **直接跳过 PEG**,用替代口径。Dashboard [tabs/lynch_analysis.py](../../../.tools/dashboard/tabs/lynch_analysis.py) 第 4 步会自动识别并提示"不适用"。

## 三、PEG 公式详解(理杏仁口径)

### 3.1 标准口径

```
PEG = PE-TTM ÷ (净利润 5y CAGR × 100)

例:美的 2026-05
  PE-TTM        = 13.5
  净利 5y CAGR  = 10.2%
  PEG           = 13.5 ÷ (10.2 × 1) = 1.32  → 🟡 略贵
```

> ⚠️ **CAGR 是百分比 × 100 还是直接 × 1**:理杏仁口径是**直接用百分数**(10% → 10),所以 13.5 / 10.2 = 1.32。
> 而非 13.5 / 0.102 = 132(经常算错)。

### 3.2 CAGR 取数:倒数第二份年报

林奇的口径是 **TTM PE / (倒数第二份年报算出的 5y CAGR)**,不用最新一期 TTM CAGR。

**理由**:用 TTM CAGR 容易被最新季单一波动主导,降噪用倒数第二份年报口径。

```python
# 与 peg_curve.py 一致(memory: PEG 必须用理杏仁口径)
end_date = 倒数第二份年报日期(如 2024-12-31)
start_date = end_date - 5 年(2019-12-31)
np_end   = 净利润@2024-12-31
np_start = 净利润@2019-12-31
cagr = (np_end / np_start) ** (1/5) - 1
peg  = pe_ttm / (cagr * 100)
```

### 3.3 替代口径(常见变体)

| 口径名 | 公式 | 适用场景 |
|---|---|---|
| **林奇标准** | PE-TTM / (净利 5y CAGR × 100) | 默认 |
| **TTM CAGR** | PE-TTM / (净利 TTM YoY × 100) | 短期波动追踪(噪音大) |
| **未来 1 年 PEG** | PE-TTM / (一致预期未来 1 年增速 × 100) | 卖方研报常用,主观性强 |
| **3 年 PEG** | PE-TTM / (净利 3y CAGR × 100) | 行业周期较短(消费品)|
| **加权 PEG**(进阶)| PE-TTM / [(过去 5y CAGR × 0.5 + 未来 1y 预期 × 0.5) × 100] | 兼顾过去与未来 |

## 四、替代估值口径速查

### 4.1 PSG(Price-to-Sales Growth)— 适合亏损成长股

```
PSG = PS-TTM ÷ (营收 5y CAGR × 100)

例:某 SaaS 公司
  PS-TTM = 6.5
  营收 CAGR = 35%
  PSG = 6.5 / 35 = 0.19  → 极度低估
```

**评级**:< 0.3 极度低估 / 0.3-0.6 合理 / 0.6-1.0 偏贵 / > 1.0 高估

**适用**:
- 亏损中的高增长公司(SaaS / 生物科技 / 互联网早期)
- EPS 失真但营收增长清晰

### 4.2 EV / EBITDA — 适合资本结构差异大的行业

```
EV = 市值 + 净有息负债
EBITDA = 净利 + 利息 + 税 + 折旧 + 摊销

EV / EBITDA 评级:< 8 低估 / 8-12 合理 / > 15 高估
```

**适用**:
- 重资产行业(钢铁 / 化工 / 航运 / 电信)
- 高资本支出 / 高折旧的公司
- 跨国对比(避免不同税制扰动)

### 4.3 P / NAV — 适合隐蔽资产 / 投资控股公司

```
NAV = 现金 + 上市股权市值 + 不动产估值 + 其他可识别资产 - 总负债
P/NAV 评级:< 0.5 极致折扣 / 0.5-0.7 折扣明显 / > 1.0 无折扣
```

**适用**:
- 隐蔽资产型(老牌国企 / 投资控股公司 / 房地产)
- 多元化集团(SOTP — Sum Of The Parts)

### 4.4 股息率 + 派息历史 — 适合缓慢增长 / 防御性

```
评估三件套:
  股息率 ≥ 4%
  派息历史 ≥ 10 年连续
  派息率 30-60%(过低不持续,过高吃老本)
```

**适用**:
- 缓慢增长型(传统银行 / 公用事业 / 电信)
- 退休账户 / 现金流需求型组合

## 五、PEG 实战案例(本仓库)

### 5.1 美的集团 000333(2026-05)

```
PE-TTM        = 13.5
净利 5y CAGR  = 10.2%
PEG           = 1.32  → 🟡 略贵(稳健类阈值已超 0.8)
```

**判断**:稳健类高质量公司,但当前 PEG 1.32 已透支增长 — 持有/跟踪,等回调到 PEG ≤ 1.0 再加仓。

### 5.2 比亚迪 002594(2026-05)

```
PE-TTM        = 21.5
净利 5y CAGR  = 38.2%(2020-2024)
PEG (5y)      = 0.56  → 🟢 合理偏低

但!最新数据:
单季 YoY      = -11.8%(2026-Q1)
8 季 hits >20% = 4/8(快速边缘)
3y CAGR       = -5.1%

→ 5y 视角 PEG 仍便宜,但 3y / 单季视角已减速。**警告:故事可能在转折**
```

**判断**:不能只看 PEG 数字 — 必须结合 [8 季 YoY 连续性](../../../.tools/dashboard/lynch_classifier.py#L972) 看故事是否仍在轨。

### 5.3 茅台 600519(2026-05)

```
PE-TTM        = 22.8
净利 5y CAGR  = 13.5%
PEG           = 1.69  → 🔴 高估(稳健类阈值 1.7 已逼近)

最新单季      = +6.5%(已退化)
8 季 hits >10% = 4/8(已不达稳健铁律)
```

**判断**:类型从"稳健"向"缓慢"过渡中,PEG 已偏高 — **PEG 不再是核心估值锚**,转看股息率 + PE 历史分位。

### 5.4 恒瑞医药 600276(2026-05)

```
PE-TTM        = 32.4
净利 5y CAGR  = 11.8%
PEG           = 2.75  → 🔴🔴 严重高估

但!8 季 hits >10% = 7/8 满足稳健铁律
```

**判断**:增长稳健但估值过高 — 创新药行业溢价 + 当前估值已透支多年增长。**不入场**,等 PEG ≤ 1.5 再考虑。

## 七、林奇关于 PEG 的关键引语

> "如果一个公司 PE 等于增长率,它的价格是合理的。如果 PE 是增长率的两倍,你正在为它付太多;如果 PE 是增长率的一半,你捡到便宜了。" — 《One Up On Wall Street》

> "10% 增长 + 25 PE = 烧钱;25% 增长 + 25 PE = 合理;25% 增长 + 12 PE = 便宜得可怕。"

> "我从不相信 PEG = 0.3 这种'好得不像真的'的便宜,通常意味着市场知道我不知道的事。先查清原因再下手。"

## 八、PEG 常见陷阱

| 陷阱 | 原因 | 应对 |
|---|---|---|
| **PEG < 0.3 太便宜** | 市场可能预期增长不可持续 | 先核查 8 季 YoY 趋势 + 是否有故事破裂信号 |
| **CAGR 用 1 年增速** | 单年波动主导,失真 | 必须用 ≥3 年滚动 CAGR(理杏仁口径用 5 年) |
| **EPS 含一次性损益** | 扣非净利与归母净利差距大 | 用扣非净利重算 PEG(memory: 扣非占比 < 70% 重度依赖一次性) |
| **周期股算 PEG** | EPS 顶部低 PE,底部高 PE | 周期股不算 PEG,改用 PB 分位 |
| **金融股算 PEG** | 银行 PE 失真(账面利润 ≠ 经济利润) | 银行用 PB + ROE + 拨备覆盖 |
