# 08 · 六类公司估值口径速查表

> 一张表打齐:每一类林奇公司用什么估值法、什么阈值、什么卖出条件、什么故事检查项。
>
> **核心原则**:**类型决定一切** — 同一指标(如 PE)在不同类型下含义完全相反(周期股 PE 最高时反而是底部)。

---

## 一、6 类核心估值口径对照(主表)

| 维度 | 🐢 缓慢增长 | 🛡️ 稳健增长 | 🚀 快速增长 | 🔄 周期型 | 🔧 困境反转 | 💰 隐蔽资产 |
|---|---|---|---|---|---|---|
| **CAGR 区间** | < 10%(2-4%)| 10-20% | ≥ 20%(优 25-35%)| 不固定(随周期)| 短期亏损但改善 | 不重要 |
| **核心估值锚** | 股息率 + PE 绝对 | **PEG ≤ 1.0** | **PEG ≤ 1.5** + PSG | **PB 历史分位 < 30%** | EV/Sales + 现金/市值 | **P/NAV < 0.6** |
| **PEG 适用?** | ❌ 不用 | ✅ 是(金标准)| ✅ 是(可放宽到 1.5)| ❌ EPS 失真 | ❌ EPS 为负 | ❌ EPS 与资产无关 |
| **PE 解读方式** | 越低越好(<15)| 配合 PEG 看分位 | 配合 PEG 看分位 | **越高越底部!**(EPS 触底)| 不可用 | 不重要 |
| **PB 解读方式** | 一般 | 中等(配合 ROE)| 一般(高 ROE 公司可高 PB)| **越低越买**(<5y 30% 分位)| 越低越买(<10% 分位)| **越低越买**(<0.7)|
| **替代口径** | 派息率 + 派息历史 | PB ROE 匹配 | PSG / EV-EBITDA | PB 分位 + 产品价格分位 | EV/Sales + NCAV + 现金/市值 | NCAV + P/NAV + 资产分散度 |
| **股息要求** | **≥ 4%(必须)** | 稳定分红是加分 | 不要求 | 低谷期可有 | 不要求 | ≥ 4% 是加分(等待成本) |
| **持有周期** | 长期(滚动)| 多年(滚动)| 高速期(2-5 年)| 半周期(2-5 年)| 改善期(2-3 年)| 等催化剂(1-3 年) |
| **失败模式** | 股息中断 / 通胀蚕食 | 增长停滞 → 缓慢 | 增长失速 / PEG 反转 | 周期高位被套 | 反转破产 | 价值陷阱 / 资产被侵占 |

## 二、买入价格阈值(各类型 1 级"极致便宜"标准)

| 类型 | 1 级便宜的核心信号 | 阈值组合 |
|---|---|---|
| 🐢 缓慢增长 | 股息率历史高位 + PE 历史低位 | 股息率 ≥ 5% **且** 当前 PE 在 5y 自身 < 30% 分位 |
| 🛡️ 稳健增长 | PEG 低位 + 股息率 + 机构冷门 | **PEG ≤ 0.8** + 股息率 ≥ 3% + PE 在 5y 自身 < 30% 分位 |
| 🚀 快速增长 | PEG 低位 + 8 季 YoY 持续 | **PEG ≤ 1.0** + 8 季 hits >20% ≥ 6/8 + PE 在 5y 自身 < 50% 分位 |
| 🔄 周期型 | PB 极低 + 产品价格底部 + 库存出清 | **PB 在 5y < 20% 分位** + 行业产品价格在 5y < 20% 分位 + 行业库存在出清 |
| 🔧 困境反转 | 现金垫子厚 + 改善信号已现 + 极致 PB 折扣 | 现金/市值 ≥ 30% + 经营 CFO 转正 + PB 在 5y < 10% 分位 |
| 💰 隐蔽资产 | NAV 折扣极大 + 资产高流动性 | **P/NAV ≤ 0.4** + 资产含现金或上市股权 + NCAV/市值 ≥ 0.5 |

## 三、4 条通用卖出触发(无论类型)

> 任一触发即"严肃评估卖出",但**触发阈值随类型而变**。

### 3.1 估值反转

| 类型 | 卖出阈值 |
|---|---|
| 缓慢增长 🐢 | 股息率跌破 2% **或** PE > 25 |
| 稳健增长 🛡️ | **PEG > 1.7** |
| 快速增长 🚀 | **PEG > 2.0** |
| 周期型 🔄 | **PB 在 5y > 80% 分位** + 行业产品价格在 5y > 80% 分位 |
| 困境反转 🔧 | PB 涨回 5y > 70% 分位 |
| 隐蔽资产 💰 | **P/NAV ≥ 0.9**(折扣消失,价值兑现完成) |

### 3.2 增长断档

| 类型 | 卖出阈值 |
|---|---|
| 缓慢增长 🐢 | 不适用(本就低增长) |
| 稳健增长 🛡️ | **连续 2 季单季营收 YoY < 10%** |
| 快速增长 🚀 | **连续 2 季单季营收 YoY < 20%** |
| 周期型 🔄 | 不适用(增长本就周期波动) |
| 困境反转 🔧 | 改善信号停滞 ≥ 4 季 |
| 隐蔽资产 💰 | 主业 CFO 持续亏损 ≥ 4 季,资产被吞噬 |

### 3.3 财务护栏失守

| 类型 | 资产负债率上限 | 流动比率下限 | CFO/NI 下限 |
|---|---|---|---|
| 缓慢增长 🐢 | ≤ 60% | ≥ 1.2 | ≥ 0.8 |
| 稳健增长 🛡️ | **≤ 50%** | **≥ 1.5** | **≥ 0.9** |
| 快速增长 🚀 | **≤ 40%(林奇铁律)** | **≥ 1.5** | **≥ 1.0** |
| 周期型 🔄 | ≤ 55% | ≥ 1.5 | 低谷期 ≥ 1.0 |
| 困境反转 🔧 | ≤ 65% | ≥ 1.0 | 转正即可 |
| 隐蔽资产 💰 | ≤ 50% | ≥ 1.5 | 持平即可(资产为主) |

代码同源:[GUARDRAIL_THRESHOLDS](../../../.tools/dashboard/tabs/lynch_analysis.py#L84)

### 3.4 故事破裂

| 类型 | 故事兑现度阈值 | 主要检查项 |
|---|---|---|
| 缓慢增长 🐢 | 派息持续性 | 股息有无中断 / 派息率是否 30-60% |
| 稳健增长 🛡️ | 兑现度 ≥ 60% | 行业地位 / ROE 稳定 / 主业不被替代 |
| 快速增长 🚀 | 兑现度 ≥ 60% | 销量 / 海外占比 / 市占率 / 产品迭代 |
| 周期型 🔄 | 周期位置仍在底部 | 产品价格 / 库存 / 行业开工率 |
| 困境反转 🔧 | 改善信号仍在累加 | 现金流 / 营收止跌 / 管理层动作 |
| 隐蔽资产 💰 | 催化剂在路上 | 重组 / 分拆 / 大股东动作 |

## 四、故事脚本检查项(每只股票一份)

每个类型的"故事脚本"必填字段不同,跑完林奇分析法后写到 `02_companies/{N}_{name}/05_投资决策/林奇五步分析_{date}.md`:

| 类型 | 必填字段(超出基础"一句话 / 验证证据 / 卖出信号"模板) |
|---|---|
| 🐢 缓慢增长 | 股息可持续性证据 / 行业是否被技术替代 |
| 🛡️ 稳健增长 | 行业龙头地位 / ROE 长期稳定原因 / 与第 2、3 名差距 |
| 🚀 快速增长 | 增长来源(销量 / 提价 / 海外 / 市占)/ 8 季 YoY 趋势 / 产能与需求空间 |
| 🔄 周期型 | 周期位置(产品价格 / 库存 / 开工率分位)/ 反转催化剂 |
| 🔧 困境反转 | 改善信号清单(≥3 项)/ 主业是否还有救 / 现金垫子能撑多久 |
| 💰 隐蔽资产 | NAV 估算明细(三方资产) / 等什么催化剂 / 持有上限期 |

## 七、林奇关键引语:类型口径

> "我从不用单一的估值法看不同类型的公司 — 那是把同一把锤子用在所有钉子上。"

> "周期股的 PE 最高时反而是底部 — 因为 EPS 触底,PE 看起来吓人,但产品价格下一波上涨时,EPS 翻倍,PE 就回归了。"

> "稳健增长股最危险的不是高 PE,而是 PEG > 1.5 — 那意味着市场已经在为'增长加速'付费,但稳健公司就是稳健,不会突然加速。"

> "隐蔽资产股的最大风险不是 NAV 算错,而是控股股东不让你看到价值 — 选大股东声誉好的,等催化剂。"

> "困境反转股是双面刃 — 5 倍赔率背后是 50% 破产风险,所以单只仓位永远 ≤ 5%。"
