# 01 · 六类公司分类法

> 投资分析的起点 — 缓慢 / 稳健 / 快速 / 周期 / 困境反转 / 隐蔽资产 六类的速查表 + 量化阈值 + 分类决策树。
>
> **核心原则**:先定性、后定量。公司分类是所有投资分析的起点 — 不同类型的公司遵循完全不同的发展逻辑、风险特征和估值方法。**误判类型 = 灾难性决策**。
>
> **来源**:综合 PDF《01_彼得林奇六类公司判断》+《战胜华尔街》(Beating the Street, 1993) + 《彼得·林奇的成功投资》(One Up On Wall Street)。
>
> **配套**:本文档与 [.tools/dashboard/lynch_classifier.py](../../../.tools/dashboard/lynch_classifier.py) 一一对应(4 步法 + 阈值 + 财务验证完全同源)。

## 一、六类公司定义速查

| # | 类型 | Emoji | CAGR 区间 | 核心特征 | 典型行业 / 案例 | 投资目的 |
| --- | --- | --- | --- | --- | --- | --- |
| 1 | **缓慢增长型** Slow Grower | 🐢 | < 10%(通常 2-4%) | 大型成熟公司，主要靠股息 | 公用事业、传统银行、电信 | 高股息防御 |
| 2 | **稳健增长型** Stalwart | 🛡️ | 10-20%(可预测) | 行业龙头，ROE 长期稳定 | 消费、医药龙头(可口可乐、宝洁、伊利) | 组合「压舱石」 |
| 3 | **快速增长型** Fast Grower | 🚀 | ≥ 20%(优 ≥ 25-35%) | 中小型，扩张期，市场空间大 | 早期沃尔玛、家得宝、宁德时代 | 寻找十倍股 |
| 4 | **周期型** Cyclical | 🔄 | 不固定(随周期波动) | 盈利受行业供需主导 | 汽车、钢铁、航空、化工、航运 | 抓周期时机 |
| 5 | **困境反转型** Turnaround | 🔧 | 短期亏损但有改善 | 严重危机后复苏 | 克莱斯勒(1979)、福特(1980s) | 高风险高赔率 |
| 6 | **隐蔽资产型** Asset Play | 💰 | 不重要 | 市值显著低于资产真实价值 | 拥有土地/专利/牌照/子公司股权 | 等价值重估 |

**关键认知**：
- **类型决定一切** — 同一指标(如 PE)在不同类型下含义完全相反(周期股 PE 最高时反而是底部)
- **公司可有混合属性** — 主+辅标注，例如"稳健(主) + 快速增长特征(辅)"
- **类型会变化** — 至少每年重检一次(快速 → 稳健 → 缓慢是常见生命周期路径)

## 二、六类详细量化标准

### 1. 缓慢增长型 🐢

**核心定义**：增长率与 GDP 相当，主要追求稳定股息。

| 维度 | 量化阈值 | 说明 |
| --- | --- | --- |
| 营收/利润 CAGR | **< 10%**(通常 2-4%) | 接近 GDP 增速 |
| 股息率 | **≥ 4%** | 关键正向信号 |
| 股息支付历史 | **≥ 10 年连续派息** | 稳定性证据 |
| 资产负债率 | 较低，行业内健康水平 | 不应过度举债 |

**A 股代表**：传统银行、电信运营商、公用事业、成熟食品(如部分饮料公司)

**投资策略**：
- **买入信号**：股息率 ≥ 5% + 经营稳定 + 估值合理(PE < 15)
- **卖出信号**：股息中断/削减、基本面恶化、出现更高股息选择
- **不需要看 PEG**(增长不是核心)

**林奇警告**：缓慢增长型的最大风险是"通胀蚕食股息"和"行业被技术替代"(如电信 → 移动)

---

### 2. 稳健增长型 🛡️

**核心定义**：稳定可预测的增长，"组合压舱石"，最适合长期持有。

| 维度 | 量化阈值 | 说明 |
| --- | --- | --- |
| 营收/利润 CAGR | **10-20%** | 林奇核心区间 |
| 5 年扣非净利 CAGR | **≥ 12%** | 盈利稳定性证据 |
| ROE 长期 | **≥ 15%** | 优秀稳健龙头门槛(巴菲特同款) |
| 资产负债率 | **< 50%** | 财务结构稳健 |
| 经营现金流 / 净利润 | **> 1.0** | 盈利质量高 |
| 盈利稳定性 | **过去 5 年至少 4 年正增长** | 可预测增长 |

**边缘稳健**(本仓库新增条款，符合林奇"先定性后定量"精神)：
- CAGR 5-10%(略低于核心区间)**且** ROE ≥ 15% **且** 行业地位稳固 → 仍归稳健
- 典型：消费/家电/白酒龙头(如美的 8.7% / 五粮液 6.6% — ROE 18-20%)
- 信心略低(75% vs 标准稳健 85%)

**A 股代表**：贵州茅台、伊利股份(部分年份)、美的集团、五粮液、宝洁、可口可乐

**投资策略**：
- **买入信号**：**PEG ≈ 1**(理想)、股价回调到历史 PE 中位数以下、行业暂时性利空
- **持有原则**："如果你不打算持有 3 年，就不要持有 3 分钟"
- **卖出信号**：
  - PEG > 2(估值已透支增长)
  - 增速降到 < 5% 且 ROE 跌破 15%(转为缓慢增长)
  - 基本面恶化(连续 2 季度低于预期)

**林奇翻倍法则**：稳健龙头的"翻倍卖一半"策略 — 持有翻倍后卖一半锁定本金，剩余持有(适用于估值已合理但仍优质的稳健股)。

> 📊 **稳健股精细打分卡** — 见 [附录 A · 稳健增长股双维定量评估](#附录-a--稳健增长股双维定量评估)：100 分质量 + 100 分价格 + 调整因子 + 5×5 决策矩阵。

---

### 3. 快速增长型 🚀

**核心定义**：高速扩张期，"十倍股"主要来源，但风险也最大。

| 维度 | 量化阈值 | 说明 |
| --- | --- | --- |
| 营收 CAGR | **≥ 20%**(优 ≥ 25-35%) | 高速区间 |
| 单季营收同比 | **连续 ≥ 8 季度 > 20%** | 增长持续性证据 |
| ROE | **≥ 15%** | 增长质量证据 |
| 资产负债率 | **< 40%(铁律!)** | 林奇原话："超过 40% 我会立刻卖出" |
| 经营现金流 / 净利润 | **> 0.8** | 不能纯粹烧钱扩张 |
| 市场空间占比 | **< 1%** | 还有大量扩张余地 |
| PEG | **< 1**(便宜) / 1-2(合理) / > 2(贵) | 核心估值指标 |

**A 股代表**：早期立讯精密、中际旭创、宁德时代(扩张期)、家得宝、星巴克

**投资策略**：
- **买入信号**：**PEG < 1** + 财务铁律全过 + 单季增速保持
- **跟踪要点**：
  - 每季度看营收/利润是否维持高速
  - 警惕**减速信号**(单季增速降到 < 15% → 拉警报)
  - 关注扩张质量(同店销售 vs 新开店占比)
- **卖出信号**：
  - 增长率显著放缓(单季 < 15% 连续 2 季度)
  - PEG > 2(估值过度透支)
  - 财务铁律破坏(债 ≥ 40% 或 ROE < 15%)
  - 基本面被破坏(竞品突破/技术替代)

**林奇陷阱警告**：
- 不要投资"概念性"快速增长(没有持续盈利)
- 不要投资"热门行业中的热门公司"(预期已经透支)
- 不要在 PE > 40 时追高(回撤风险大)

---

### 4. 周期型 🔄

**核心定义**：盈利随宏观/行业供需剧烈波动，**时机决定一切**。

| 维度 | 关键判断 |
| --- | --- |
| **行业属性** | 产品价格/库存/产能利用率呈明显周期性 |
| **CAGR** | 不固定，可能某年 +50%、某年 -30% |
| **高点 ROE** | 可能 > 15%(此时反而是危险信号) |
| **PE 反向** | PE **最低**时往往是顶部，PE **最高**时(因盈利低)反而是底部 |

**典型行业**：汽车、钢铁、航空、有色金属、煤炭、化工、建材、航运、造纸

**周期判断框架**：

```
┌────────────────────────────────────────────────┐
│ 底部信号(买入时机)✅                           │
│   产能利用率 < 70%                              │
│   多数公司亏损或微利                            │
│   库存积压、价格低迷                            │
│   资本开支大幅减少                              │
│   破产、兼并增加                                │
│   PE 很高(因盈利低)甚至为负                   │
├────────────────────────────────────────────────┤
│ 顶部信号(卖出时机)⚠️                          │
│   产能利用率 > 90%                              │
│   全行业普遍盈利、利润率很高                    │
│   库存紧张、价格上涨                            │
│   大量新产能投资                                │
│   新竞争者进入                                  │
│   PE 很低,看起来"便宜"                         │
└────────────────────────────────────────────────┘
```

**A 股代表**：比亚迪(汽车)、三美股份(化工)、宝钢(钢铁)、中远海控(航运)

**投资策略**：
- **买入**：行业 1-2 级低点(看 PB、库存周期)
- **卖出**：行业 4-5 级高点(看产能扩张、盈利透支)
- **不要用 PEG**(周期股增长不可预测)
- **看 PB 替代 PE**(PB < 1.5 是底部参考线)

---

### 5. 困境反转型 🔧

**核心定义**：陷入严重危机但有明确复苏迹象 — 高风险高赔率。

**判定双重信号**(必须**同时满足**)：

1. **危机存在**：
   - 严重亏损(净利润 YoY < -30%)
   - 债务危机(资产负债率剧增)
   - 业务丑闻或管理层危机
   - 股价较历史高点跌幅 > 70%

2. **反转催化**：
   - **核心反转指标连续 2 季度改善**(营收/毛利/CFO)
   - 新管理层上任(管理变革)
   - 业务剥离非核心资产
   - 成本削减计划见效
   - PB 通常处于历史低位(< 1.5)

**林奇可逆 vs 不可逆判断**：

| ✅ 可逆困境(可投) | ❌ 不可逆困境(避开) |
| --- | --- |
| 流动性危机但核心业务有竞争力 | 技术被颠覆性替代(柯达、诺基亚) |
| 管理失误,新团队可纠正 | 结构性需求下降(传统媒体) |
| 一次性损失或行业暂时衰退 | 致命债务负担,接近破产 |
| 资产/品牌/客户基础完整 | 多次困境反转失败的"困境惯性" |

**A 股代表**：(谨慎对待 — A 股有退市风险、财务造假风险更高)
- 历史:格力(2008 危机后)、上汽(行业低谷反转)
- 美股经典:克莱斯勒(1979)、福特(1980s)

**投资策略**：
- **安全边际**:股价 < 历史高点 30%、PB < 1.5、债务 / EBITDA < 3
- **仓位控制**:单只 ≤ 5%(风险极高)
- **必看指标**:**经营性现金流是否仍正**(否则随时破产)
- **退出**:反转完成股价回到正常区间 → 重新分类(可能成为稳健或快速)

**林奇警告**：
- 不要投资"困境中的困境公司"(连续亏损 + 持续恶化)
- 一次困境反转可能成功,**连续困境往往万劫不复**

---

### 6. 隐蔽资产型 💰

**核心定义**：拥有被市场低估的巨额资产 — 增长率不重要，**资产重估**才是关键。

**常见隐蔽资产类型**：

| 资产类型 | 例子 | 重估方法 |
| --- | --- | --- |
| **土地** | 自有总部、厂房、商铺地段 | 评估当前市价 vs 账面值 |
| **现金** | 现金/总市值高(≥ 30%) | 净现金价值 |
| **专利** | 制药、科技专利 | 现金流贴现或交易可比 |
| **品牌** | 历史悠久但未充分变现 | 商誉评估 |
| **子公司股权** | 控股有市价的子公司 | 子公司市值占比 |
| **稀缺牌照** | 银行牌照、采矿权、电视牌照 | 牌照交易市价 |
| **税务亏损递延** | 历史亏损可抵税未来利润 | 税务规划价值 |

**判定标准**：
- 现金/总市值 **≥ 30%** 或 PB **< 1.0**
- 市值 **显著低于** 各类资产的真实市价之和
- 公司财报附注披露资产细节(需深入研读)

**A 股代表**：
- 中国中车(PB 0.98 < 1)
- 部分老牌商业地产(自有商铺地段)
- 部分有大额现金的二线消费

**投资策略**：
- **耐心是关键** — 等待催化(回购、分红、资产分拆、子公司上市)
- **不需要看 PEG / 增长率**
- **风险**:价值陷阱(便宜但永远不被发现)
- **退出信号**:
  - 大额回购/分红
  - 分拆催化兑现
  - 价值已经被发现(PB > 1.5)

## 三、定量化判定流程(4 步法)

> 与 [lynch_classifier.py:classify()](../../../.tools/dashboard/lynch_classifier.py) 完全对应。

### 步骤 1 · 行业属性优先

```
公司行业 ∈ {汽车 / 钢铁 / 航空 / 化工 / 航运 / 有色 / 煤炭 / 建材 / 地产}?
  └─ 是 → 周期型(主)。无视 CAGR(即便高也归周期)
       退出
  └─ 否 → 进步骤 2

公司行业 ∈ {银行 / 保险 / 证券}?
  └─ 是 → 金融业特殊处理:
       · CAGR ≥ 10% AND ROE ≥ 15%  → 稳健(优秀金融龙头)
       · 否则                       → 缓慢(传统银行/保险)
       · 不适用 PEG / 现金流 / 负债率铁律(行业特性)
       退出
  └─ 否 → 进步骤 2
```

### 步骤 2 · CAGR 三档分级

```
计算近 3-5 年营收/利润 CAGR

CAGR ≥ 20%
  └─ 初判:快速增长型(信心 80%, ≥ 25% 时 90%)

CAGR ∈ [10%, 20%)
  └─ 初判:稳健增长型(信心 85%)

CAGR ∈ [5%, 10%) AND ROE ≥ 15%
  └─ 初判:边缘稳健(信心 75%) — 消费/家电/白酒龙头特例

其他(CAGR < 5% OR ROE < 15%)
  └─ 初判:缓慢增长型(信心 80%)

数据缺失
  └─ 暂归"未分类",信心 20%(等数据补齐)
```

### 步骤 3 · 财务特征验证

> 知识库 PDF P3 步骤二 + 战胜华尔街《25 条法则》P24

| 验证维度 | 稳健合格 | 快速合格 | 失败处理 |
| --- | --- | --- | --- |
| **盈利稳定性** | 5 年至少 4 年正增长 | 3 年持续高增长 | 增长波动大 → 可能周期/困境 |
| **财务安全** | 资产负债率 < 50% | **< 40%(铁律)** | 远高于阈值 → 风险过大,降级或剔除 |
| **现金流质量** | CFO/NI > 1 | CFO/NI > 0.8 | 持续为负 → 增长质量存疑 |
| **ROE** | 平均 ≥ 15% | 平均 ≥ 15% | 持续低迷 → 降级到缓慢或困境 |

**降级路径**(失败 → 修正)：
```
快速候选: 债 ≥ 40% AND ROE < 15% → 降级稳健(质量不达标)
稳健候选: ROE < 12% AND CAGR < 15% → 降级缓慢
缓慢候选: 股息率 ≥ 4% → 信心 +0.10(防御性强)
缓慢候选: 股息率 < 2% → 信心 -0.05(缺乏防御吸引)
```

### 步骤 4 · 特殊情况附加(辅助标签)

不改变主类,加 secondary tag:

```
np_yoy < -30% AND PB < 1.5
  → 困境反转特征(主类不是 fast_grower 时,主类升级为 turnaround)

cash/MC ≥ 30%
  → 隐蔽资产特征(若同时 PB < 1.5,主类升级为 asset_play)

PB < 1.0
  → 隐蔽资产特征(账面价值被低估)
```

**最终输出**：主类 + 信心 + 辅助标签 + 步骤二验证警告。

## 四、PEG 估值法(林奇核心公式)

> "理想状态是 PE = 增长率,即 PEG = 1。" — 彼得·林奇

```
PEG = PE-TTM ÷ (盈利增长率 × 100)

例:
  公司 A: PE 20, 增长率 25% → PEG = 20/25 = 0.80 ✅ 便宜
  公司 B: PE 30, 增长率 15% → PEG = 30/15 = 2.00 ⚠️ 偏贵
```

### PEG 评级表

| PEG 区间 | 评级 | 行动 |
| --- | --- | --- |
| **< 0.5** | 🟢 极度便宜 | 强买信号(警惕陷阱:增长不可持续?) |
| **0.5 - 1.0** | 🟢 便宜 | 买入候选 |
| **1.0 - 1.5** | 🟡 合理 | 观察(等回调或基本面改善) |
| **1.5 - 2.0** | 🟠 偏贵 | 谨慎,考虑减仓 |
| **> 2.0** | 🔴 透支增长 | 卖出信号(估值已严重透支) |

### PEG 适用 / 不适用

| 适用 | 不适用 |
| --- | --- |
| ✅ 快速增长型(核心) | ❌ 周期型(增长波动太大) |
| ✅ 稳健增长型(理想 PEG ≈ 1) | ❌ 困境反转型(增长不可预测) |
| ✅ 增长率有持续性证据 | ❌ 银行/保险(需用 PB-ROE / 内含价值) |
| ⚠️ 缓慢增长型(意义不大,主要看股息) | ❌ 隐蔽资产(看资产重估) |

### A 股 PEG 调整

- A 股成长股 PEG 系统性偏高于美股 → 阈值放宽 1 档
- **快速增长股 PEG < 1.5** 仍可视为合理(美股标准 < 1)
- **结合行业平均 PEG 比较**(医药行业 PEG 中位数 vs 制造业)

## 五、闲聊法(Scuttlebutt)调研框架

> 源自菲利普·费雪,林奇发扬光大。"业余投资者比专业投资者具有与生俱来的优势。"

│   行业协会:行业整体趋势?                    │
├─────────────────────────────────────────────┤
│ 第三层:企业内部调研(高级)                  │

## 六、25 条黄金法则精炼

> 战胜华尔街第三部分。每年至少复读一遍。

### 🧠 投资哲学(1-5)

1. **投资你了解的领域** — 1 分钟内向 12 岁孩子解释清楚
2. **业余投资者有时间优势** — 华尔街 3 个月才发现的,你商场 3 年前就看到了
3. **忽略市场预测** — 没人能预测市场,包括我自己
4. **投资是艺术而非科学** — 数学无法预测人性
5. **不要投资热门行业中的热门公司** — 预期已经透支

### 🎯 选股标准(6-12)

6. **PEG < 1 的成长股** — 理想 PE = 增长率
7. **账面价值为负可能是好事** — 回购/收购商誉的合理后果
8. **沙漠之花** — 在低迷行业中表现优异的公司
9. **机构持股少的公司机会更大** — 还没被发现 = 信息优势
10. **内部人买入是好信号** — 没有比自掏腰包更诚实的投票
11. **利润率提升是重要信号** — 成本控制或定价权增强
12. **利基市场垄断者** — 细分市场的隐形冠军

### ⏳ 持有策略(13-18)

13. **以年为单位** — 不打算持有 3 年,就不要持有 3 分钟
14. **不因下跌恐慌卖** — 基本面没变,下跌是机会
15. **不因上涨兴奋买** — 估值透支,上涨是卖出机会
16. **定期复盘但不频繁交易** — 每季度检查一次足够
17. **接受错误,及时止损** — 最初理由消失就卖
18. **股价最终跟随盈利** — 长期看价值

### ⚠️ 风险管理(19-25)

19. **分散但不过度分散** — 5-10 只深研 vs 50 只泛了解
20. **不投连续困境公司** — 一次反转可能成功,连续往往万劫不复
21. **警惕价值陷阱** — 便宜有便宜的原因
22. **避免最糟糕行业中的最好公司** — 行业衰退,谁也独善其身不了
23. **不追逐概念** — 没业绩支撑的概念终将破灭
24. **警惕高负债** — 债务是企业的天敌(尤其周期股)
25. **享受投资过程** — 投资应该是乐趣

## 八、A 股本土化注意事项

### 阈值调整

| 指标 | 美股原标准 | A 股调整 | 原因 |
| --- | --- | --- | --- |
| 持续派息年限 | 20 年 | **10 年** | A 股公司历史短 |
| 快速增长 PEG | < 1 | **< 1.5 仍可接受** | A 股成长股估值系统性偏高 |
| 困境反转 PB | < 1 | **< 1.5** | A 股市场效率较低 |
| 隐蔽资产现金/MC | ≥ 30% | **≥ 30%(同)** | 通用 |

### 行业差异

- **白酒**:可视为稳健龙头(高 ROE + 可预测) — 茅台/五粮液
- **银行**:核心资产质量(不良率 + 拨备覆盖率)替代林奇财务铁律
- **新能源**:汽车归周期 + 动力电池/光伏归快速,组合复杂
- **港股 vs A 股**:港股估值更便宜但流动性差,需更长持有期

### 风险加强

- **退市风险** — 警惕 ST/*ST,困境反转需更严格安全边际
- **财务造假** — 现金流验证(CFO/NI)、关联交易、应收账款增速 vs 收入增速
- **政策风险** — 行业政策可能颠覆林奇标准(如教培、互联网平台)

## 附录 A · 稳健增长股双维定量评估

> 这是稳健增长型(Stalwart)的精细化打分卡 — 把第二章 §2 的定性标准量化为可执行的双维评分体系 + 调整因子 + 5×5 决策矩阵。
>
> **适用范围**：仅稳健增长型(快速/缓慢/周期/困境/隐蔽各有专属评估,见各类型小节)。

### A.1 体系框架

```
                  ┌──────────────────────────────────┐
公司分析  →       │  好公司评分(100 分,质量维度)     │
                  │  好价格评分(100 分,估值维度)     │
                  │  ↓ 各自加调整因子(±)             │
                  │  调整后总分 → 5 级分级            │
                  │  ↓                                │
                  │  公司 5 级 × 价格 5 级 = 决策矩阵 │
                  │  ↓                                │
                  │  明确买卖指令 + 仓位建议          │
                  └──────────────────────────────────┘
```

### A.2 好公司评分(质量维度,基础满分 100 分)

### A.6 投资决策矩阵(质量 × 价格)

| 公司 \ 价格 | 1 级(严重低估) | 2 级(适度低估) | 3 级(合理估值) | 4 级(估值偏高) | 5 级(严重高估) |
| --- | --- | --- | --- | --- | --- |
| **A 级(卓越)** | 🟢 强力买入(可重仓) | 🟢 积极买入(分批建仓) | 🟡 持有 / 观望 | 🟠 停止买入 / 减仓 ≥ 30% | 🔴 坚决卖出 |
| **B 级(良好)** | 🟢 积极买入 | 🟡 适度买入(卫星配置) | 🟡 持有(跟踪) | 🟠 考虑减仓 20-50% | 🔴 卖出 |
| **C 级(一般)** | 🟡 谨慎买入(严控仓位) | 🟡 少量试探(见好就收) | ⚪ 观望 | ⚪ 回避 | 🔴 坚决回避 |
| **D 级(差)** | 🟠 投机性买入(高风险) | ⚪ 不参与 | ⚪ 不参与 | ⚪ 不参与 | ⚪ 不参与 |

### A.7 执行流程

```
1. 数据收集
   ↓
2. 计算基础分(好公司 100 + 好价格 100)
   ↓
3. 应用调整因子(决策日志录入新闻/公告/动态)
   ↓
4. 确定等级(A/B/C/D × 1/2/3/4/5)
   ↓
5. 矩阵定位(交叉查找)
   ↓
6. 决策执行(明确买卖指令 + 仓位)
```

### A.8 核心逻辑总结

- **好公司**:增长 + 健康 + 回报 三维度量化,满分 100
- **好价格**:绝对估值(PEG/股息/现金) + 相对估值(行业/历史) 两视角量化,满分 100
- **调整因子**:捕捉基本面边际变化与市场情绪错杀 / 过热
- **交叉决策**:公司质量 × 价格吸引力 → 明确买卖指令 + 仓位建议
- **林奇风格量化**:把"增长型公司 + 合理估值 + 逆向机会"的定性洞见,转化为可执行的打分卡体系
