# 05 · 困境反转型 🔧 ABCD/12345 评估

> 公司质量 A/B/C/D × 价格吸引力 1/2/3/4/5 双维评分体系 — 适用于"严重危机后改善信号已现"的高赔率高风险标的。
>
> **核心目标**:在公司经营触底但**改善信号已出现**(现金流转正 / 营收止跌 / 管理层换血)时介入,博取 3-10 倍反转赔率。
>
> **林奇语录**:"被宣告破产前夜的克莱斯勒,艾科卡上任 5 年后股价涨 30 倍。困境反转的关键不是看现状有多惨,而是看改善信号有多明确。"

|------|----------|------------------|--------------|

|  | 经营现金流拐点 | 15 分 | 最近 4 季 CFO 由负转正 + 持续 2 季正 得 15 分;转正但仅 1 季 得 10 分;仍为负但收窄 ≥50% 得 5 分;无改善 0 分 |

## 第二部分:困境反转专属调整因子

### 一、"好公司"调整因子(±20 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 行业整体出清(供给端永久性收缩) | **+10 分** | 行业产能 3 年内退出 ≥ 20%(如钢铁去产能 / 航运拆船) |
|  | 监管环境从恶化转改善 | **+8 分** | 政策从限制转支持(如教培 → 双减后转型成功的公司) |
|  | 关键诉讼/调查靴子落地 | **+5 分** | 重大法律风险已确定结果(可承受范围内) |
| **减分项** | 改善信号已 4 季无进一步进展 | **-15 分** | 拐点后停滞,反转故事可能失败 |
|  | 现金流再次转负 | **-20 分** | 反转破产风险显著上升 |
|  | 关键债务展期失败 | **-15 分** | 资金链风险升级 |
|  | 核心客户/供应商流失 | **-10 分** | 业务基础持续侵蚀 |

### 二、"好价格"调整因子(±15 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 短期错杀(财报后短期跌幅过度) | **+10 分** | 季报后 1 周跌 > 15%,但基本面未恶化 |
|  | 大股东 / 管理层 增持 | **+8 分** | 内部人(非定增)增持 ≥ 1% 流通股 |
| **减分项** | PB 涨回 5y > 50% 分位 | **-10 分** | 资产端折扣消失 |
|  | 现金 / 市值 < 10% | **-15 分** | 安全垫子已耗尽 |

## 第三部分:等级划分标准(调整后总分)

### 一、公司质量最终等级 A/B/C/D

| 等级 | 调整后总分 | 定性描述 |
|------|------------|----------|
| **A 级** | **80 - 120** | 高确定性反转:改善信号 ≥3 项 + 现金充裕,3-5 倍赔率可期 |
| **B 级** | 60 - 79 | 中等确定性:改善 1-2 项明确,但财务尚紧;1.5-3 倍赔率 |
| **C 级** | 40 - 59 | 低确定性:仅财报数字改善,无催化剂;赔率有限 |
| **D 级** | < 40 | 仍在恶化或反转失败:回避 |

### 二、价格吸引力最终等级 1/2/3/4/5

| 等级 | 调整后总分 | 定性描述 |
|------|------------|----------|
| **1 级** | **80 - 115** | 极致便宜:现金 > 市值 50% + PB 极低分位 — 安全边际充分 |
| **2 级** | 60 - 79 | 适度便宜:资产端打折明显 |
| **3 级** | 40 - 59 | 价格反映了改善预期(已涨过一段) |
| **4 级** | 20 - 39 | 估值偏贵:多数改善已 priced in |
| **5 级** | < 20 | 已透支反转预期 |

## 第五部分:核心纪律与林奇语录

### 1. 介入纪律

> **林奇:"我喜欢困境反转股,但只在改善信号已经出现之后介入 — 不当抄底英雄。"**

- 严禁"猜底" — 必须看到 ≥2 项改善信号(现金流转正 / 营收止跌 / 管理层换血)再介入
- 检查频率:**每月**复查季报披露,反转故事节奏快、容易破裂

### 2. 卖出触发(任一即严肃评估)

| # | 条件 | 阈值 |
|---|------|------|
| ① | 改善信号停滞 ≥ 4 季 | 反转故事失败 |
| ② | 现金流再次转负 | 财务恶化 |
| ③ | 关键债务展期失败 | 资金链风险升级 |
| ④ | PB 涨回 5y > 70% 分位 | 资产端折扣消失,反转预期已 priced in |
| ⑤ | 反转完成转向稳健 | **重新分类**为稳健/缓慢增长型,按新框架评估 |

### 3. 与其他类型的区别

| 维度 | 困境反转 🔧 | 周期型 🔄 | 缓慢增长 🐢 |
|------|-------------|------------|--------------|
| **盈利现状** | 严重亏损 / 大幅下滑 | 周期高低波动 | 稳定低增速 |
| **核心赌注** | 改善信号兑现 | 周期位置反转 | 股息持续 |
| **估值口径** | EV/Sales + PB + 现金 | PB 分位 | 股息率 + PE |
| **持有周期** | 改善期(2-3 年)→ 反转完成转类型 | 半周期(2-5 年)| 长期(滚动) |
| **失败模式** | 反转破产 | 周期高位被套 | 股息中断 |

### 4. 林奇关键引语

> "克莱斯勒在 1979 年被认为破产前夜,所有人都在卖。但艾科卡上任后,K-car 救了公司,股价 5 年涨 30 倍。" — 困境反转最大赔率往往在所有人都已绝望时

> "找寻'破产但不会破'的公司:看现金、看主营是否有救、看管理层有没有救亡的人。"

> "困境反转股不能买太多 — 一只占 5% 已经是上限,因为破产风险永远存在。"
