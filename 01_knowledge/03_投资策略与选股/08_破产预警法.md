# 08 破产预警法 — Altman Z-Score

> **大师**：Edward Altman（爱德华·阿特曼，1941-）
> **方法**：Z-Score —— 用 5 个财务比率合成的破产概率指标
> **出处**：1968 年论文《Financial Ratios, Discriminant Analysis and the Prediction of Corporate Bankruptcy》
> **定位**：**风险底线工具**，不是选股工具，是"绝对回避名单"过滤器

---

## 🎯 一句话核心

> "**Z 值告诉你这家公司未来 2 年破产的概率。**"
>
> —— Altman

Altman 用 1946-1965 年间 66 家美国制造业公司（33 家破产 + 33 家健康）做判别分析，
**Z-Score 在公司破产前 1 年的预测准确率 95%，前 2 年 72%。**

巴菲特名言"投资第一条规则：不要亏损"——Z-Score 是**避免踩雷的量化护栏**。

---

## 📐 公式（制造业上市公司原版）

```
Z = 1.2 × A + 1.4 × B + 3.3 × C + 0.6 × D + 1.0 × E
```

| 变量 | 计算 | 含义 | 数据来源 |
|------|------|------|---------|
| **A** | 营运资金 / 总资产 | 短期流动性 | `05_安全性分析` |
| **B** | 留存收益 / 总资产 | 累积盈利能力 | `02_盈利分析` |
| **C** | EBIT / 总资产 | 资产盈利效率 | `02_盈利分析` |
| **D** | 股票市值 / 总负债 | 市场对偿债的信心 | `01_估值分析` + `05_安全性分析` |
| **E** | 营业收入 / 总资产 | 资产周转效率 | `03_成长性分析` |

> 注：营运资金 = 流动资产 - 流动负债

---

## 🚦 Z 值阈值解读（制造业）

| Z 值区间 | 评级 | 含义 |
|----------|------|------|
| **Z > 2.99** | 🟢 安全区 | 未来 2 年破产概率极低 |
| **1.81 < Z < 2.99** | 🟡 灰色区 | 警惕，需深入研究 |
| **Z < 1.81** | 🔴 危险区 | 2 年内破产概率 > 80% |

---

## 🔄 三种变体（按公司类型选用）

### Z'-Score（非上市制造业）
```
Z' = 0.717 A + 0.847 B + 3.107 C + 0.420 D' + 0.998 E
```
其中 **D' = 账面权益 / 总负债**（替代市值）

### Z''-Score（非制造业 / 服务业 / 新兴市场）⭐ A 股推荐
```
Z'' = 6.56 A + 3.26 B + 6.72 C + 1.05 D'
```
**仅 4 项，去掉了营收/资产**（因服务业资产周转结构差异大）

| Z'' 值 | 评级 |
|--------|------|
| > 2.6 | 🟢 安全 |
| 1.1 ~ 2.6 | 🟡 警惕 |
| < 1.1 | 🔴 危险 |

## 🇨🇳 A 股适用性评估

### ✅ 高适用度（用 Z 或 Z''）

| 行业 | 公式 | 项目内公司 |
|------|------|------|
| 传统制造业 | Z（原版） | 中国中车 / 三美 / 三花 / 立讯 / 美的 |
| 消费品 | Z'' | 茅台 / 五粮液 / 伊利 / 蜜雪 |
| 医药 | Z'' | 恒瑞 |
| 新能源（成长期） | Z''（更稳） | 比亚迪 / 宁德 / 中际旭创 |

### ❌ 不适用

| 行业 | 原因 |
|------|------|
| 银行 | 资产负债结构完全不同（招商银行用 CAR 资本充足率代替） |
| 保险 | 浮存金特殊性（新华保险用偿付能力充足率） |
| 房地产 | 存货周转完全异常 |

## 🚨 风险信号组合（Altman + 其他大师）

A 股实践中，单一 Z 值会有"假阴性"。建议配合：

| 信号 | 阈值 | 含义 |
|------|------|------|
| Z'' < 1.1 | — | 财务危险 |
| **+** Piotroski F < 3 | — | 经营恶化中 |
| **+** 资产负债率 > 80% | — | 高杠杆 |
| **+** 经营现金流连续 2 年为负 | — | 失血 |

⚠️ **四项命中 ≥ 3 项 → 立即清出股票池**

## 🧮 项目数据映射（DuckDB 落地）

```sql
-- A 股推荐的 Z'' 公式
SELECT
    ticker,
    date,
    -- A: 营运资金 / 总资产
    (current_assets - current_liabilities) / total_assets AS a,
    -- B: 留存收益 / 总资产
    retained_earnings / total_assets AS b,
    -- C: EBIT / 总资产
    ebit / total_assets AS c,
    -- D': 账面权益 / 总负债
    book_equity / total_liabilities AS d_prime,
    -- Z''
    6.56 * (current_assets - current_liabilities) / total_assets
    + 3.26 * retained_earnings / total_assets
    + 6.72 * ebit / total_assets
    + 1.05 * book_equity / total_liabilities AS z_double_prime,
    -- 评级
    CASE
        WHEN 6.56 * (current_assets - current_liabilities) / total_assets
           + 3.26 * retained_earnings / total_assets
           + 6.72 * ebit / total_assets
           + 1.05 * book_equity / total_liabilities > 2.6 THEN 'safe'
        WHEN ... > 1.1 THEN 'gray'
        ELSE 'danger'
    END AS rating
FROM v_company_yearly;
```

## ⚠️ Altman 的局限

1. **不适合科技股**：高估值科技股 D 值（市值/负债）极高，Z 值常常虚高
2. **滞后于现金流**：现金流量表恶化往往**早于** Z 值发出警报
3. **不预测股价**：Z 值高的公司**不代表股价会涨**，仅代表破产概率低
4. **行业基准不同**：相同 Z 值在不同行业含义不同，**只能横向对比同行**
