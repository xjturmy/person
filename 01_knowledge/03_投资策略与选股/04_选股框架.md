## 一、评估体系概览

### 1.1 五维评估模型

**核心维度及权重：**

- 💰 **盈利能力** - 25%（核心指标）
- 📊 **估值水平** - 20%（安全边际）
- 📈 **成长性** - 20%（未来潜力）
- 🛡️ **安全性** - 15%（风险控制）
- 💸 **现金流 + 分红** - 20%（可持续性）

**综合评分 = 加权求和，满分 100 分**

### 3.2 ROE评分标准（杜邦分析）

```
ROE = 净利率 × 资产周转率 × 权益乘数

ROE评分：
ROE>20%且稳定 → 5分
ROE>15%且稳定 → 4分
ROE>12%且稳定 → 3分
ROE>8%且稳定 → 2分
ROE<8%或下滑 → 1分

质量检查：
□ ROE来自高利润率（优质）
□ ROE来自高周转（中等）
□ ROE来自高杠杆（需警惕）
```

### 3.3 盈利能力维度计算

```
盈利得分 = (ROE得分×2 + ROA得分 + ROIC得分×2 + 毛利率得分 + 净利率得分) / 7 × 25%
        = ___ / 7 × 0.25
        = ___ 分
```
