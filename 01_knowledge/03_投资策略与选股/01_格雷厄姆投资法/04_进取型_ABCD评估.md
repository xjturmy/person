# 04 · 进取型 ⚔️ ABCD/12345 评估

> 公司质量 A/B/C/D × 价格吸引力 1/2/3/4/5 双维评分体系 — 适用于"进取型投资者"(愿意花时间研究、追求 12-15% 年化)的核心持仓。
>
> **核心目标**:挑选**二三线优质 + 适度成长 + PEG ≤ 1.0 + 财务稳健**的"格雷厄姆式成长股",捕捉成长合理性的甜蜜点,3-5 年滚动持有。
>
> **格雷厄姆语录**:"进取型投资者会在以下三个领域寻找机会:相对不受欢迎的大公司 / 廉价证券 / 特殊情境。"

## 第二部分:进取型专属调整因子

### 一、"好公司"调整因子(±10 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 行业出现寡头化趋势 | **+8 分** | 行业 CR3 从 N 升至 N+10pp,公司在受益方 |
|  | 持续股票回购 | **+5 分** | 近 3 年回购金额 ≥ 净利润 5%,自由现金流覆盖 |
|  | ROIC 5 年持续提升 | **+8 分** | 5 年 ROIC 趋势向上(每年 +1pp 以上)|
|  | 业务结构升级 | **+5 分** | 高毛利业务占比从 N% 升至 N+10pp |
| **减分项** | 增长依赖加杠杆 | **-10 分** | 杜邦分解:权益乘数贡献 > 50% ROE 提升 |
|  | 新业务 ROIC < 老业务 | **-8 分** | 公司在 ROIC 较低的领域大额扩张(典型价值毁灭)|
|  | 大额商誉 / 频繁并购 | **-10 分** | 商誉 / 净资产 > 30%,且并购整合不及预期 |
|  | 关联交易激增 | **-8 分** | 关联交易 / 营收从 N% 升至 N+10pp,治理预警 |

### 二、"好价格"调整因子(±10 分)

| 类型 | 信号 | 调整分数 | 关键触发条件 |
|------|------|----------|--------------|
| **加分项** | 业绩超预期但股价滞涨 | **+10 分** | 季报营收/利润超预期 ≥ 15%,股价 1 个月涨幅 < 增长幅度 |
|  | 行业悲观情绪极端 | **+5 分** | 行业整体 PE 在自身历史分位 < 20%,但公司基本面优于行业 |
| **减分项** | PEG 已 > 1.5 | **-12 分** | 估值开始透支未来增长 |
|  | 卖方一致预期过高 | **-8 分** | 卖方给出的明年增长预期 > 25%,但历史 5 年 CAGR < 15%(易失望)|

## 第三部分:等级划分标准(调整后总分)

### 一、公司质量最终等级 A/B/C/D

| 等级 | 调整后总分 | 定性描述 |
|------|------------|----------|
| **A 级** | **85 - 110** | 真进取型优质:盈利质量好 + ROIC ≥ 15% + 增长可持续 — 组合 5-10% 核心仓 |
| **B 级** | 70 - 84 | 良好进取型:基本面达标但 1-2 项瑕疵(如 ROIC 12-15% / CFO/NI 0.7-0.9)|

| **D 级** | < 55 | 不达进取标准:或归类为深度低估(若 PB < 1)或回避 |

### 二、价格吸引力最终等级 1/2/3/4/5

| 等级 | 调整后总分 | 定性描述 |
|------|------------|----------|

## 第五部分:核心纪律与格雷厄姆语录

### 1. 持有纪律

> **格雷厄姆:"进取型投资者必须接受短期波动,但永远不放弃安全边际。"**

- 进取型持有 3-5 年(完整盈利兑现周期)
- 检查频率:每季度业绩 + 每年深度复评一次
- 单只仓位上限 10%(因风险高于防御型)

### 2. 卖出触发(任一即严肃评估)

| # | 条件 | 阈值 |
|---|------|------|
| ① | PEG > 1.7 | 估值反转 |
| ② | 连续 2 季 EPS YoY < 5% | 增长断档 |
| ③ | CFO/NI < 0.7 持续 2 年 | 利润含金量预警 |
| ④ | ROIC 连续 3 年下滑 | 资本效率破裂 |

### 3. 与防御型 / 林奇快速增长的区分线

| 维度 | 进取型 ⚔️ | 防御型 🛡️ | 林奇快速 🚀 |
|------|-----------|------------|-------------|
| EPS CAGR | ≥ 8% | ≥ 5% | ≥ 20% |
| 股息要求 | 当前有派息即可 | **连续 10 年(铁律)** | 不要求 |
| ROIC 要求 | ≥ 15% | ≥ 15%(隐含)| ≥ 20% |
| PE 容忍 | < 25 | < 15-25 | < 50 |
| PEG 要求 | **≤ 1.0(铁律)** | 不强制 | ≤ 1.5 |
| 风险敞口 | 中等 | 低 | 高 |

### 4. 格雷厄姆关键引语

> "我从不为成长支付任何溢价。如果一家公司增长 12%,我只愿意付 12 倍 PE — 一分不多。"
>
> "进取型投资的成功,90% 来自避免失败,10% 来自抓到大牛股。"
>
> "二线优质股的最佳买入时机,通常是行业老大老二打价格战、老三被市场抛弃时。"

### 示例 2:三花智控 002050(2026 年评估,假设案例)

**好公司评分**:
