# 01 · 四类价值分类法

> 格雷厄姆深度价值体系下,公司按"价值来源"分四类。**入手前必须先确定类型,否则估值口径会用错。**
>
> **来源**:《Security Analysis》第 6 篇资产分析 + 《The Intelligent Investor》第 14 章防御型投资 + 第 15 章进取型投资 + 巴菲特致股东信中"格雷厄姆三类买入决策"延伸。

---

## 一、四类速查表

| # | 类型 | Emoji | 价值核心 | 主选估值尺 | 安全边际目标 | 持有周期 | 典型代表(A 股)|
|---|------|-------|----------|-----------|--------------|---------|----------------|
| 1 | **深度低估型** | 🪙 | 净流动资产 / 清算价值 | PB < 1 / NCAV | ≥ 50% | 1-3 年(价值实现即卖)| 周期股低谷 / 困境地产 / 部分钢铁 |
| 2 | **防御型** | 🛡️ | 现有盈利 + 股息 + 资产 | PE × PB ≤ 22.5 | ≥ 33% | 5+ 年(长期持有)| 招行 / 长江电力 / 大秦铁路 |
| 3 | **进取型** | ⚔️ | 未来盈利 + 适度成长 | PEG ≤ 1.0 | ≥ 25% | 3-5 年(成长兑现期)| 美的 / 伊利 / 上汽集团 |
| 4 | **特殊情境** | 🎭 | 事件驱动 / NAV 重估 | EV/Sales / NAV | ≥ 30%(概率加权)| 1-3 年(事件兑现)| 重组分拆 / 私有化套利 / 困境反转 |

## 二、决策树:这家公司属于哪一类?

```
                          【输入:一家公司】
                                ↓
                    Q1: 公司是否持续盈利且稳定?
                       ├─ 否(亏损 / 微利 / 周期波动 ≥ 50%)
                       │      ↓
                       │   Q2: 资产是否扎实(NCAV 大于市值的 1.5 倍)?
                       │      ├─ 是 → 🪙 深度低估型
                       │      └─ 否 → Q3: 是否有特殊事件驱动?
                       │              ├─ 是 → 🎭 特殊情境(困境反转 / 重组 / 套利)
                       │              └─ 否 → ❌ 不投(无价值锚点)
                       │
                       └─ 是(连续 5 年盈利,EPS 稳定)
                              ↓
                          Q4: 公司规模是否够大,且分红稳定?
                              ├─ 是(市值 > 100 亿 + 连续 10 年分红 + DY ≥ 3%)
                              │      ↓
                              │   🛡️ 防御型(对应 Intelligent Investor 第 14 章)
                              │
                              └─ 否(中小盘 + 高成长 + 分红不稳)
                                     ↓
                                 Q5: PEG 是否合理(≤ 1.0)?
                                     ├─ 是 → ⚔️ 进取型(对应第 15 章)
                                     └─ 否 → ❌ 估值偏高,等待回落
```

## 三、🪙 深度低估型(NCAV / 清算价值法)

### 定义

公司的**当前市值低于其净流动资产的 2/3**,或者市净率(PB)显著低于 1。

**净流动资产(NCAV)= 流动资产 - 总负债**

### 特征清单

- ✅ **PB < 1.0**(优 < 0.7)
- ✅ **市值 < NCAV × 2/3**(格雷厄姆原版铁律)
- ✅ **流动比率 ≥ 2.0**(确保清算价值)
- ✅ **资产质量真实**(剔除商誉 / 无形资产 / 可疑应收)
- ⚠️ 通常是**周期低谷期**或**行业受冲击**的标的

### A 股实操难度

A 股牛市偏多,真正符合 NCAV 标准的标的极少 — 主要在:
- **熊市末期**(如 2018Q4 / 2024 跌深的中小盘)
- **强周期低谷**(钢铁 / 煤炭 / 化工 / 航运在景气底部)
- **政策受冲击行业**(地产 / 教培 / 平台经济 2021-2024)

### 风险与边界

| 风险 | 表现 | 防范 |
|------|------|------|
| 价值陷阱 | 资产是真,但持续亏损烧钱 | 看 5 年是否累计盈利,即使微利 |
| 资产虚胖 | 大量商誉 / 待摊费用 | 用净有形资产(NTA)替代净资产 |
| 流动性差 | 市值 < 50 亿,买卖难 | 设最低市值门槛 |
| 治理风险 | 大股东掏空 | 看 5 年是否分红 + 关联交易占比 |

详见 [02_深度低估型_ABCD评估.md](02_深度低估型_ABCD评估.md)。

## 四、🛡️ 防御型(防御型投资者七准则)

### 定义

格雷厄姆《Intelligent Investor》第 14 章为"**防御型投资者**"(目标年化 ≥ 7-10%、不愿耗时研究)设计的七条准则:

### 防御型七准则(原版)

| # | 准则 | 阈值 | A 股调整 |
|---|------|------|----------|
| 1 | **规模充分** | 工业:营收 ≥ $10 亿;公用事业:总资产 ≥ $20 亿 | 市值 ≥ 200 亿 |
| 2 | **财务足够稳健** | 流动比率 ≥ 2.0;长期负债 ≤ 净流动资产 | 同 |
| 3 | **盈利稳定** | 过去 10 年每年盈利,无亏损 | 过去 10 年至少 8 年盈利 |
| 4 | **股息稳定** | 过去 20 年连续分红 | 过去 10 年连续分红 |
| 5 | **盈利增长** | 10 年 EPS 增长 ≥ 33%(三年均值对比)| 10 年 EPS CAGR ≥ 3% |
| 6 | **PE 合理** | 当前 PE ≤ 过去 3 年平均 EPS 的 15 倍 | PE ≤ 过去 3 年平均的 15 倍,且 < 25 |
| 7 | **PB 合理** | 当前 PB ≤ 1.5,且 PE × PB ≤ 22.5(格氏数)| 软达标:< 22.5 满分,22.5-40 灰阶 |

### A 股符合度参考

| 公司 | 符合数(7 条)| 主要瑕疵 |
|------|--------------|---------|
| 招商银行(600036)| 6/7 | 格氏数偶超(PE×PB ≈ 6,远低于 22.5)— **真符合** |
| 长江电力(600900)| 6/7 | PE 长期 18-22 略高 |
| 贵州茅台(600519)| 4/7 | 格氏数严重超标(PE × PB > 50)|
| 美的集团(000333)| 5/7 | 防御性偏弱,更接近"进取型" |
| 伊利股份(600887)| 5/7 | 盈利稳定但 EPS 增长率不达标 |

> **A 股操作建议**:格雷厄姆原版 7 条对 A 股偏严苛,建议"软达标"模式(满足 5/7 即可视为防御型,但 PE×PB 仍是关键)。

详见 [03_防御型_ABCD评估.md](03_防御型_ABCD评估.md)。

## 五、⚔️ 进取型(进取型投资者补充准则)

### 定义

格雷厄姆《Intelligent Investor》第 15 章为"**进取型投资者**"(愿意花时间研究、追求 ≥ 12% 年化)设计的进取型选股准则:

### 进取型五准则

| # | 准则 | 阈值 | 实操要点 |
|---|------|------|----------|
| 1 | **财务结构** | 流动比率 ≥ 1.5;长期负债 ≤ 110% 净流动资产 | 金融除外 |
| 2 | **盈利稳定** | 过去 5 年盈利,无亏损 | 而非 10 年 |
| 3 | **股息历史** | 当前有派息(任何金额)| 比防御型放宽 |
| 4 | **增长** | 当前 EPS > 5 年前 EPS | EPS 不下滑即可 |
| 5 | **价格** | PE < 历史 3 年均值 EPS 的 9 倍 | A 股调整为 PE < 25 + PEG ≤ 1.0 |

### A 股进取型典型路径

进取型不是"买中小盘高成长",而是**"二线优质 + 估值合理 + 适度成长"**:
- 行业龙二/龙三(非龙一,因为龙一 PE 通常已贵)
- ROE 稳定 ≥ 12%(虽然不如防御型 18%)
- PEG ≤ 1.0(用 5 年 CAGR 算分母)
- 有专业人士跟踪研究的能力(因为需要更细颗粒度的判断)

### A 股符合度参考

| 公司 | 符合度 | 适用方法 |
|------|--------|----------|
| 美的集团 | ⭐⭐⭐⭐ | PEG ≈ 1.3,稍超但 ROE 22% 可接受 |
| 三花智控 | ⭐⭐⭐⭐ | PEG ≈ 1.5,二线龙头,符合定义 |
| 比亚迪 | ⭐⭐⭐ | 增长太快,接近林奇"快速增长型" |
| 中国中车 | ⭐⭐⭐⭐ | 防御型/进取型边界,看周期位置 |

详见 [04_进取型_ABCD评估.md](04_进取型_ABCD评估.md)。

## 六、🎭 特殊情境(Special Situations)

### 定义

格雷厄姆把"非常规价值机会"统称为**特殊情境**(Special Situations) — 这些机会不依赖企业持续经营的"常态"价值,而是依赖**事件本身**实现价值。

### 三大子类

#### 子类 A:困境反转

| 特征 | 描述 |
|------|------|
| 价值锚 | 反转后的盈利 / EV/Sales / 重置成本 |
| 触发 | 管理层换届 / 业务剥离 / 行业出清 / 政策放开 |
| 时间 | 12-36 个月 |
| 关键判定 | 反转可见性 ≥ 50% / 现金流转正 / 负债不至于强制重组 |
| 失败率 | 50-70%(高风险高赔率) |
| 仓位 | 单只不超过组合 3-5% |

#### 子类 B:重组 / 套利

| 特征 | 描述 |
|------|------|
| 价值锚 | 重组对价 / 私有化对价 / 分拆 NAV |
| 触发 | 公告 / 监管批文 / 股东大会决议 |
| 时间 | 3-12 个月 |
| 关键判定 | 监管通过概率 ≥ 80% / 价差 ≥ 15% / 时间成本可承受 |
| 失败率 | 10-30%(被否决 / 拖延)|
| 仓位 | 高确定性可重仓(单只 ≤ 10%) |

#### 子类 C:NAV 折价

| 特征 | 描述 |
|------|------|
| 价值锚 | 持有的子公司 / 物业 / 股权的市价加总 |
| 触发 | 控股公司 / 投资公司 / 房地产信托 |
| 时间 | 不可控(等价值重估)|
| 关键判定 | NAV 折价 ≥ 30% / 治理改善信号 |
| 失败率 | 折价持续(永久折价陷阱)|
| 仓位 | 低仓位(单只 ≤ 5%) |

### A 股案例

| 案例 | 子类 | 结果 |
|------|------|------|
| 平安私有化中国电信(2024)| 重组套利 | 6% 价差,3 个月兑现 |
| 万科 2025 困境(地产周期)| 困境反转 | 在判定中 — 看 2026 销售改善 |
| 紫金矿业控股紫金黄金 | NAV 折价 | 持续折价 ~25% |

详见 [05_特殊情境_ABCD评估.md](05_特殊情境_ABCD评估.md)。

## 七、四类的金融业适配

| 类型 | 银行 | 保险 |
|------|------|------|
| 🪙 深度低估 | PB < 0.6 + ROE > 8%(招行 2018-2019)| EV/NBV 长期分位 < 30%(新华 2024) |
| 🛡️ 防御型 | PB < 1.0 + DY > 4% + ROE > 13% + CET1 > 10% | EV 增长稳定 + DY > 3% + 偿付能力 > 200% |
| ⚔️ 进取型 | 二三线城商/农商,但 PB 折价 + ROE > 13% + NPL < 1.5% | 二线寿险,EV 增速 > 行业 + NBV margin 改善 |
| 🎭 特殊情境 | 经历 NPL 高峰后改善的(如 2014-2016 五大行)| 监管改革后调整完毕的(预期 2027+) |

> 实操中:
> - 银行 PE 不可比 → 看 PB × DY ≥ 0.04(代替式)+ CET1 / NPL
> - 保险用 EV(内含价值) 替代净资产 → P/EV < 0.5 + EV 增速 > 12%

## 八、跨类型实战:同一家公司在不同时期类型变化

| 公司 | 2014 | 2018 | 2024 | 2026 |
|------|------|------|------|------|
| 招商银行 | 防御型 | 防御型 | 防御型 | 防御型 |
| 美的集团 | 进取型 | 进取型 | 进取型 / 防御型 | 防御型 |
| 万科 A | 进取型 | 防御型 | **困境反转** | **特殊情境** |
| 长江电力 | 防御型 | 防御型 | 防御型 | 防御型 |
| 比亚迪 | 进取型 | 进取型 | 进取型 / 林奇快速 | 进取型 |
| 上证 50 整体估值 | 深度低估 | 防御 | 接近防御 | 防御-进取 |

**关键认知**:类型不是固定标签,**需每年复评**。万科从 2014 年的"进取型"演变到 2024 年的"困境反转",评估口径完全不同。
