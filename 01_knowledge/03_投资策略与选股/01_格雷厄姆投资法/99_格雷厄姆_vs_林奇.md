## 一、哲学起点对比

| 维度 | 格雷厄姆 1934 | 林奇 1989 |
|------|--------------|-----------|
| **价值定义** | 资产 + 已实现盈利 | 资产 + 已实现盈利 + **预期成长** |
| **核心理念** | 安全边际(Margin of Safety)| 故事 + 类型(Story + Category)|
| **第一问题** | "我亏不亏得起?" | "为什么这家公司会涨?" |
| **避险策略** | 资产打底 + 财务铁律 | 类型驱动财务护栏 + 故事兑现度 |
| **持有周期** | 1-5+ 年(类型分化) | 2-多年(类型分化) |
| **超额收益来源** | 价格 - 内在价值的折价 | 公司增长 + PEG 估值修复 |
| **代表作** | 《Security Analysis》《Intelligent Investor》 | 《One Up On Wall Street》《Beating the Street》 |
| **典型语录** | "用 0.5 美元买 1 美元" | "如果不能 3 句话说清,就别买" |

| **🛡️ 稳健增长** | — | ✅ 强对应 | ✅ 部分对应 | — |

## 三、四类公司在两套体系下的差异判定

### 案例 1:招商银行(600036)

| 维度 | 格雷厄姆视角 | 林奇视角 |
|------|------------|---------|
| 类型 | 🛡️ **防御型(A 级)** | 🛡️ **稳健增长 / 🐢 缓慢增长边界** |
| 核心估值 | 格氏数 PE×PB ≈ 6(远低于 22.5)| PEG ≈ 0.5(超低)|
| 决策一致性 | ✅ 两套都看好,深度低估 | ✅ 一致 |
| 持有周期 | 5+ 年(防御压舱石)| 多年滚动(稳健压舱石)|
| 仓位上限 | 15% | 15% |
| **建议组合标签** | "防御型 - 稳健增长重叠区"|

### 案例 2:贵州茅台(600519)

| 维度 | 格雷厄姆视角 | 林奇视角 |
|------|------------|---------|
| 类型 | 🛡️ 防御型质量 A 级,但**价格永远 5 级** | 🛡️ **稳健增长(A 级)** |
| 核心估值 | 格氏数 PE×PB > 100,严重不达标 | PE 历史 35-50%,PEG = 1.5-2.0 |
| 决策冲突 | ❌ 格雷厄姆**永远拒绝买入** | ✅ 林奇接受合理价格,A 级 + 3 级 = 持有 |
| 解释 | 格雷厄姆只买"明显便宜",茅台 PB 8 不可能买 | 林奇接受"伟大公司付合理价" |
| **建议** | 用林奇框架决策(格雷厄姆框架对茅台不适用)|

### 案例 3:比亚迪(002594)

| 维度 | 格雷厄姆视角 | 林奇视角 |
|------|------------|---------|
| 类型 | ⚔️ 进取型(B 级 — 现金流尚不稳定)| 🚀 **快速增长(A 级)** |
| 核心估值 | PEG ≈ 1.5(超出格雷厄姆容忍)| PEG ≈ 1.5(在林奇容忍区)|
| 决策冲突 | △ 格雷厄姆边界拒绝 | ✅ 林奇接受,A 级 + 2 级 = 配置 |
| 解释 | 格雷厄姆要求 PEG ≤ 1.0,比亚迪超出 | 林奇容忍 ≤ 1.5,在金标准区 |
| **建议** | 用林奇快速增长框架 + 设比亚迪为"成长股桶"|

### 案例 4:万科(000002)2025 评估

| 维度 | 格雷厄姆视角 | 林奇视角 |
|------|------------|---------|
| 类型 | 🎭 **特殊情境 - 困境反转** | 🔧 **困境反转** |
| 核心估值 | EV/历史峰值 < 0.5 + PB < 0.5 | 反转可见性 + 现金缓冲 |
| 决策一致性 | ✅ 两套都视为困境反转,逻辑一致 | ✅ 一致 |
| 仓位上限 | 5%(困境单只)| 5%(林奇困境也限仓)|
| 持有周期 | 18-24 月(设期限)| 18-24 月 |
| **建议组合标签** | "困境反转(双体系一致)"|

### 案例 5:招行 2014(历史回测)

| 维度 | 格雷厄姆视角 | 林奇视角 |
|------|------------|---------|
| 当时 PE = 4 / PB = 0.7 / DY = 7% |
| 类型 | 🪙 **深度低估(NCAV 接近)** | 🛡️ 稳健增长 + 估值修复 |
| 决策 | 1 级 + A 级 = 全力出击 | 1 级 + A 级 = 全力出击 |
| 一致性 | ✅ 完美一致 | ✅ 完美一致 |
| **2014-2021 实际回报** | **+350%** |
| **回顾**:两套体系都准确识别这是历史性机会 |

## 四、相同公司不同时期的类型迁移

### 万科 A 的时间轴

| 时间 | 格雷厄姆类型 | 林奇类型 | 关键事件 |
|------|------------|---------|----------|
| 2010-2014 | ⚔️ 进取型 A 级 | 🚀 快速增长 | 房地产黄金期 |
| 2015-2018 | 🛡️ 防御型 B 级 | 🛡️ 稳健增长 | 销售增速 30% → 10% |
| 2019-2021 | ⚔️ 进取型 B 级 | 🛡️ 稳健 / 🐢 缓慢边界 | 政策周期 |
| 2022-2024 | 🎭 困境反转 | 🔧 困境反转 | 三道红线 + 销售崩 |
| 2025 | 🎭 困境反转(深度)| 🔧 困境反转 | 仍未走出困境 |
| **关键**:万科类型在 15 年里迁移了 5 次,**评估口径必须每年复评** |

### 美的集团的时间轴

| 时间 | 格雷厄姆类型 | 林奇类型 | 关键事件 |
|------|------------|---------|----------|
| 2013(IPO 前一年)| ⚔️ 进取型 | 🚀 快速增长 | 上市前高速期 |
| 2015-2019 | ⚔️ 进取型 A | 🚀 快速增长 → 🛡️ 稳健 | 业务结构升级 |
| 2020-2024 | ⚔️ 进取型 / 🛡️ 防御边界 | 🛡️ 稳健增长 A 级 | 库卡整合期 |
| 2025+ | 🛡️ 防御型(B-C 边界)| 🛡️ 稳健 / 🐢 缓慢 | 增速回落到 5-10% |

## 五、何时用格雷厄姆,何时用林奇

### 格雷厄姆框架更适用的场景

| 场景 | 理由 |
|------|------|
| **熊市末端 / 估值低位** | 资产打底逻辑发挥最大作用 |
| **金融业(银行 / 保险)** | PB / 股息率比 PEG 更可比 |
| **公用事业 / 垄断行业** | 现金流稳定,符合防御型定义 |
| **强周期行业底部** | NCAV / PB 法 |
| **困境反转 / 重组套利** | 特殊情境理论是格雷厄姆原创 |
| **新手 / 不愿耗时研究的投资者** | 防御型 7 准则机械执行,门槛低 |

### 林奇框架更适用的场景

| 场景 | 理由 |
|------|------|
| **牛市中段 / 估值合理期** | 增长合理性比绝对低估更重要 |
| **消费 / 医药 / 科技** | 持续增长的故事可识别 |
| **小盘股 / 早期玩家** | "10 倍股"逻辑 |
| **周期股转折期** | 林奇有专门的周期型评估口径 |
| **故事驱动型公司** | 林奇"故事脚本"工具 |
| **追求高 alpha 的投资者** | 接受更高 PE / PEG 容忍度 |

## 六、组合层面的双体系叠加

### 推荐组合标签模板

每只标的同时给出两个标签 + 决策来源:

```yaml
公司: 招商银行
代码: 600036

格雷厄姆:
  类型: 防御型
  公司质量: A
  价格吸引力: 1
  决策: 全力出击 10-15%

林奇:
  类型: 稳健增长
  公司质量: A
  价格吸引力: 1
  决策: 全力出击 10-15%

决策来源: 双体系一致 → 高确定性
组合仓位: 12%
```

### 决策冲突时的处理

| 冲突类型 | 处理方法 |
|----------|---------|
| 格雷厄姆 5 级 / 林奇 1 级(茅台经典)| 取保守者(格雷厄姆)— 不买 |
| 格雷厄姆 1 级 / 林奇 5 级(罕见)| 取保守者(林奇)— 不买 |
| 格雷厄姆 D 级 / 林奇 A 级 | 看具体冲突原因:质量分歧通常是估值标准不同,选适合该行业的体系 |
| 类型完全不同(防御型 vs 快速增长)| 复评 — 通常公司在不同时期类型迁移 |

## 七、组合双体系比例建议

参考巴菲特 1965 致股东信 + 现代投资组合理论:

```text
┌─────────────────────────────────────┐
│ 推荐组合配置(按双体系标签)          │
├─────────────────────────────────────┤
│  防御型 + 稳健增长(双体系一致重叠区) │  60-70%
│  ────────────────────────────────── │
│  进取型 + 稳健/快速(双体系部分重叠)  │  15-25%
│  ────────────────────────────────── │
│  深度低估 + 周期型 / 困境反转          │  5-10%
│  ────────────────────────────────── │
│  特殊情境(套利 / 重组 / NAV)         │  5-10%
└─────────────────────────────────────┘
```

> **避免**:仓位 100% 在格雷厄姆 / 林奇任何单一类型 — 缺乏类型分散导致策略风险集中。
