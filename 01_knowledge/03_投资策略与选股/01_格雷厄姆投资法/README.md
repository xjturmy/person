## 🚨 4 条通用卖出触发(任一即严肃评估)

1. **当前价格 > 内在价值 × 1.33** — 安全边际反转
2. **三道防线任一被穿透** — 财务健康破裂(现金 / 经营造血 / 弹药)
3. **报表造假预警** — 应收增速 >> 营收 / 商誉 > 净资产 30% / 审计非标
4. **价值陷阱信号确认** — 行业衰退 / 主业被替代 / 管理层失能

## 🚀 入门路径建议

### 路径 A · 防御型投资者(目标 7-10% 年化,不愿耗时研究)

1. 读 [00_方法论总览.md](00_方法论总览.md)(30 分钟)
2. 读 [03_防御型_ABCD评估.md](03_防御型_ABCD评估.md)(30 分钟)
3. 用 7 准则筛选 10-15 只蓝筹组合
4. 每年深度复评 1 次,价格涨多减仓 / 跌多加仓
5. 股债比例 30-70%(动态平衡)

### 路径 B · 进取型投资者(目标 12-15% 年化,愿意花时间)

1. 读 [00_方法论总览.md](00_方法论总览.md) + [01_四类价值分类.md](01_四类价值分类.md)
2. 读 [03_防御型_ABCD评估.md](03_防御型_ABCD评估.md) + [04_进取型_ABCD评估.md](04_进取型_ABCD评估.md)
3. 读 [99_格雷厄姆_vs_林奇.md](99_格雷厄姆_vs_林奇.md) — 决定每只标的用哪套框架
4. 跑五步实战 [06](06_实战_商业模式.md) → [10](10_实战_深度审视.md)
5. 组合按双体系标签管理

### 路径 C · 特殊情境投资者(高赔率高门槛)

1. 完整读完所有方法论文档
2. 重点读 [05_特殊情境_ABCD评估.md](05_特殊情境_ABCD评估.md)
3. 读巴菲特 1965 致股东信(特殊情境占 35% 时间)
4. 单只仓位严格上限 5-10%
