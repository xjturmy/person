# 01 · 三大范式判定

> 鲁政委《保卫财富》核心框架 — **5+5+5=15 个信号矩阵**判定黄金当前主导身份。
>
> **核心命题**:任一时刻只一种身份主导。范式一(经济金融)激活 → 触发"避险或周期";范式二(技术革命)激活 → 解锁"抗通胀";范式三(大国博弈)激活 → 长期"周期"。
>
> **判定规则**:每个范式 5 个信号,通过 ≥3 个即"该范式激活"。三大范式投票 → 决定主导身份与配置权重。

---

## 一、为什么需要三大范式?

### 1.1 传统框架在 2022 年后失效

| 传统认知 | 2022 年后现实 | 失效原因 |
|---------|--------------|----------|
| 美元涨 → 黄金跌 | 美元 + 黄金同涨 | 去美元化 + 央行增持 |
| 实际利率为负 → 黄金涨 | 实际利率 +2% 时金价仍创新高 | 大国博弈 + 储备货币重构 |
| 信息化时代 → 黄金 20 年熊 | AI 革命期 → 黄金与科技股同涨 | AI 引发就业冲击 + 避险需求 |

→ **结论**:单一变量无法解释金价,需要"三大范式综合投票"。

### 1.2 三大范式的逻辑分层

```
┌──────────────────────────────────────────────────────────────────┐
│              三大范式分层(从短到长)                              │
├──────────────────────────────────────────────────────────────────┤
│                                                                    │
│  范式一 · 经济金融 ← 短期(月-季)                                  │
│   驱动:周期位置、风险偏好、流动性                                  │
│   输出:🛡️ 避险信号 / 🔄 周期信号                                  │
│                                                                    │
│  范式二 · 技术革命 ← 中期(年-十年)                                │
│   驱动:产业技术周期、生产力跃迁                                    │
│   输出:🔥 抗通胀 加成 / 削弱                                       │
│                                                                    │
│  范式三 · 大国博弈 ← 长期(十年-世代)                              │
│   驱动:储备货币变迁、央行行为、制裁工具化                          │
│   输出:🔄 周期定盘星(长期支撑或压制)                              │
│                                                                    │
└──────────────────────────────────────────────────────────────────┘
```

## 二、范式一 · 经济金融属性(5 信号)

**核心问题**:黄金当前处于经济周期的什么位置?

### 2.1 五个信号

| # | 信号 | 阈值 | 数据源 | 当前(2026-05) |
|---|------|------|--------|----------------|
| 1.1 | **康波周期位置** | 萧条期 = 激活 | 周金涛框架 / 手填 | 🟢 第五次萧条期 |
| 1.2 | **VIX 恐慌指数** | > 25 = 避险激活 | AkShare `index_global_em_dict` | ⚪ 18(待激活)|
| 1.3 | **美国 10Y 实际利率** | < 0 = 利好黄金 | 10Y - CPI YoY | 🟢 -1.2% |
| 1.4 | **避险资金流向** | 美债+黄金净流入 | SPDR 持仓 / TLT 流入 | 🟢 SPDR +12%(近 6 月)|
| 1.5 | **地缘风险事件** | ≥ 1 重大事件持续 | GPR Index / 手填 | 🟢 俄乌 + 中东 |

### 2.2 投票判定

- ✅ 激活 ≥ 3 → 范式一激活 → 触发 🛡️ 避险或 🔄 周期信号
- ⚪ 激活 < 3 → 范式一钝化 → 黄金缺乏周期支撑

### 2.3 当前判定(2026-05)

**4/5 激活** → 🟢 范式一已激活,主导短期"避险 + 周期"信号

## 三、范式二 · 技术革命属性(5 信号)

**核心问题**:当前技术周期是支持还是压制金价?

### 3.1 历史规律

| 技术革命 | 时间 | 美股表现 | 黄金表现 | 关系 |
|----------|------|----------|----------|------|
| 电气化 | 1870-1946 | 工业股牛市 | 相对弱势 | 反向 |
| 信息化 | 1991-2020 | 互联网+科技牛市 | 20 年熊市 | **强反向** |
| AI 革命 | 2020-? | 科技股大涨 | 同步大涨 | **同向(罕见)** |

→ **2026 年关键判断**:AI 革命破"信息化压金价"惯例,因 AI 引发**就业/经济冲击的隐忧 + 实际利率长期低位**。

### 3.2 五个信号

| # | 信号 | 阈值 | 数据源 | 当前(2026-05) |
|---|------|------|--------|----------------|
| 2.1 | **AI 商用化进度** | ChatGPT-5+ 应用规模 | 手填 / 行业报告 | 🟢 AI 应用未全面替代就业 |
| 2.2 | **美股科技股 vs 金价相关性** | 12 月相关性 > 0 = 同涨同跌 | 计算:NDX 与 GLD | 🟢 0.42(同向)|
| 2.3 | **生产力增速** | 美国非农生产力 YoY < 2% | BLS / FRED | 🟡 1.8%(临界)|
| 2.4 | **就业冲击隐忧** | AI 替代率新闻指数 | 手填 / 媒体跟踪 | 🟢 高位 |
| 2.5 | **能源革命矛盾** | 储能/光伏 vs 黄金需求竞争 | 行业产能 / 手填 | ⚪ 中性 |

### 3.3 投票判定

- ✅ 激活 ≥ 3 → 范式二激活 → 解锁 🔥 抗通胀 + 削弱"科技股压金价"
- ⚪ 激活 < 3 → 回到信息化时代逻辑(科技股牛 → 黄金弱)

### 3.4 当前判定(2026-05)

**3/5 激活** → 🟢 范式二刚刚激活,黄金抗通胀逻辑加成

## 四、范式三 · 大国博弈属性(5 信号)

**核心问题**:全球货币体系是否正在重塑?

### 4.1 黄金"再货币化"历史节点

| 年份 | 事件 | 影响 |
|------|------|------|
| 1971 | 尼克松关闭黄金兑换窗口 | 美元贬值 → 70s 黄金大牛 |
| 2008 | 次贷危机 | 全球量化宽松 → 黄金十年牛市 |
| 2022 | 俄乌制裁 + 美元储备冻结 | 央行增持 → 黄金"再货币化"新时代 |

### 4.2 五个信号

| # | 信号 | 阈值 | 数据源 | 当前(2026-05) |
|---|------|------|--------|----------------|
| 3.1 | **全球央行净购金** | 连续 ≥ 2 年净增持 | WGC 季报 / AkShare | 🟢 连续 3 年 |
| 3.2 | **美元储备占比** | 全球 < 60% = 激活 | IMF COFER | 🟢 58%(下行趋势)|
| 3.3 | **去美元化倡议** | 金砖+/上合 etc 推进 | 手填 / 新闻 | 🟢 BRICS+ 扩容持续 |
| 3.4 | **制裁工具化频率** | 美国 SDN list 增长 | OFAC 数据 / 手填 | 🟢 历史高位 |
| 3.5 | **新兴经济体储备需求** | 印度/中东央行黄金占比 ≥ 10% | WGC 国别报告 | 🟢 多国突破 10% |

### 4.3 投票判定

- ✅ 激活 ≥ 3 → 范式三激活 → 长期 🔄 周期定盘星建立
- ⚪ 激活 < 3 → 回归"美元一超"框架,黄金长期承压

### 4.4 当前判定(2026-05)

**5/5 全部激活** → 🟢 范式三长期激活,这是 2022 年后黄金牛市的根本逻辑

## 五、综合投票判定主导身份

### 5.1 判定矩阵

| 范式一(经济)| 范式二(技术)| 范式三(博弈)| 主导身份 | 配置建议 |
|---------------|---------------|---------------|----------|----------|
| ✅ 激活 | ⚪ 钝化 | ⚪ 钝化 | 🛡️ 避险(短期)| 临时 +5-10% |
| ⚪ 钝化 | ✅ 激活 | ⚪ 钝化 | 🔥 抗通胀(中期)| 战略 10-15% |
| ⚪ 钝化 | ⚪ 钝化 | ✅ 激活 | 🔄 周期(长期)| 战略 10-15% |
| ✅ 激活 | ✅ 激活 | ⚪ 钝化 | 🛡️ + 🔥 混合 | 战略 15-20% |
| ✅ 激活 | ⚪ 钝化 | ✅ 激活 | 🛡️ + 🔄 混合 | 战略 15-20% |
| ⚪ 钝化 | ✅ 激活 | ✅ 激活 | 🔥 + 🔄 混合 | 战略 15-20% |
| ✅ 激活 | ✅ 激活 | ✅ 激活 | **三身份全开** ⭐ | **战略 20-25%(高风险偏好可至 38%)** |
| ⚪ 钝化 | ⚪ 钝化 | ⚪ 钝化 | 黄金弱势期 | **降至 0-5%** |

### 5.2 当前判定流程示例(2026-05)

```text
范式一(经济金融):4/5 ≥ 3 ✅ 激活
范式二(技术革命):3/5 ≥ 3 ✅ 激活
范式三(大国博弈):5/5 ≥ 3 ✅ 激活
                              ↓
                    三范式全部激活 ⭐
                              ↓
              主导身份:🛡️ + 🔥 + 🔄 全开
                              ↓
              战略配置建议:20-25%
                              ↓
              高风险偏好客户:可至 38%(鲁政委震撼结论)
```

## 六、范式切换的早期信号

### 6.1 范式一切换信号(短期)

| 信号变化 | 含义 | 行动 |
|---------|------|------|
| VIX 从 > 30 跌回 < 15 | 避险情绪退潮 | 减仓 5% 战术仓 |
| 实际利率从 -1% 升至 +2% | 抗通胀逻辑失效 | 减仓至战略下限 |
| 地缘风险事件解决 | 短期推动力消失 | 仅持有战略仓 |

### 6.2 范式二切换信号(中期)

| 信号变化 | 含义 | 行动 |
|---------|------|------|
| AI 商用化全面落地 + 生产力大涨 | 类似 2000 年代互联网 | 警惕黄金转入熊市 |
| 美股科技股 vs 金价相关性转负 | 回到信息化时代逻辑 | 减仓战略仓至 5-10% |

### 6.3 范式三切换信号(长期)

| 信号变化 | 含义 | 行动 |
|---------|------|------|
| 央行连续 2 季净抛售黄金 | 储备需求逆转 | 战略减仓至 5% |
| 美元储备占比回升至 65%+ | 美元霸权重建 | 战略减仓至 0-5% |
| 主要经济体回归"美元锚" | 货币体系回稳 | 战略减仓至 0-5% |

## 七、信号阈值的代码化(yaml 雏形)

> Phase 2.4 将这些阈值落到 `.tools/rules/gold_paradigm.yaml` 让用户可调

```yaml
# .tools/rules/gold_paradigm.yaml(Phase 2.4 实现)
paradigms:
  economic_financial:
    threshold: 3   # 5 个信号中通过 ≥3 即激活
    signals:
      - id: kondratiev_phase
        name: 康波周期位置
        type: categorical
        activate_when: [萧条期, 萧条期中后段]
        source: manual
      - id: vix
        name: VIX 恐慌指数
        type: numeric
        activate_when: "> 25"
        source: akshare
      - id: real_rate
        name: 美国 10Y 实际利率
        type: numeric
        activate_when: "< 0"
        source: derived  # 10Y_YIELD - CPI_YoY
      - id: spdr_inflow
        name: SPDR 黄金 ETF 持仓 6 月变化
        type: percent
        activate_when: "> 5%"
        source: akshare
      - id: geopolitical_event
        name: 地缘冲突活跃事件数
        type: count
        activate_when: ">= 1"
        source: manual

  tech_revolution:
    threshold: 3
    signals:
      - id: ai_commercialization
        name: AI 商用化进度
        type: categorical
        activate_when: [早期, 中期]
        source: manual
      - id: tech_gold_correlation
        name: 美股科技 vs 金价相关性
        type: numeric
        activate_when: "> 0"
        source: derived
      # ...

  great_power_struggle:
    threshold: 3
    signals:
      - id: cb_net_purchase
        name: 全球央行连续净购金年数
        type: count
        activate_when: ">= 2"
        source: wgc
      # ...

# 主导身份判定逻辑
identity_rules:
  - condition: economic_financial.active && !tech_revolution.active && !great_power_struggle.active
    identity: safe_haven_short
    weight: "+5-10% 战术仓"
  - condition: economic_financial.active && tech_revolution.active && great_power_struggle.active
    identity: full_three_active
    weight: "20-25% 战略 / 高风险 38%"
```
