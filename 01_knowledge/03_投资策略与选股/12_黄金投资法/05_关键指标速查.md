# 05 · 关键指标速查

> 黄金投资的**4 张分位表**— 一屏看懂当前黄金的"贵贱热度"。
>
> **核心指标**:① 实际利率 / ② 金油比 / ③ 金银比 / ④ SPDR 持仓
>
> **用途**:
> - **战术择时**:这些指标的极端分位 = 短期反转/共振信号
> - **配合范式投票**:范式 + 分位 = 完整决策(范式定方向,分位定时机)
> - **类比林奇 PEG 速查**:一张表 → 一个分位 → 一个动作

## 二、指标 ① · 实际利率(美债 10Y - CPI)

### 2.1 公式与含义

```
实际利率 = 美国 10Y 名义利率 - 美国 CPI 同比
```

**经济含义**:持有黄金的**机会成本**。
- 实际利率 < 0 → 持现金会跑输通胀 → 黄金"零息劣势"消失
- 实际利率 > 0 → 持现金能跑赢通胀 → 黄金不如美债

### 2.2 分位表(2004-2024)

| 当前实际利率 | 历史分位 | 信号 | 黄金未来 12 月预期回报 |
|-------------|---------|------|---------------------|
| < -2% | < 5% | 🟢🟢🟢 极度低估 | +18% |
| -2% ~ -1% | 5-15% | 🟢🟢 强力利好 ⭐ | +12% |
| -1% ~ 0% | 15-35% | 🟢 利好 | +6% |
| 0% ~ +1% | 35-65% | ⚪ 中性 | +1% |
| +1% ~ +2% | 65-85% | 🟡 压制 | -3% |
| > +2% | > 85% | 🔴 严重压制 | -8% ⚠️(2022 后失效) |

### 2.3 当前位置(2026-05-07)

```
实际利率 -1.2%(分位 12%) → 🟢🟢 强力利好
                              ↓
                    黄金未来 12 月预期回报 +12%
                              ↓
                    战术行动:维持或加仓 5%
```

## 三、指标 ② · 金油比

### 3.1 公式与含义

```
金油比 = 黄金价格(美元/盎司) / WTI 原油价格(美元/桶)
```

**经济含义**:**避险资产 vs 风险资产**的相对价值。
- 金油比高 = 黄金相对油价偏强 = 经济衰退预期
- 金油比低 = 油价相对黄金偏强 = 经济繁荣

### 3.2 分位表(2000-2024)

| 当前金油比 | 历史分位 | 信号 | 含义 |
|-----------|---------|------|------|
| > 35 | > 90% | 🔴 黄金严重过热 | 经济衰退 / 减仓黄金 |
| 25-35 | 70-90% | 🟡 黄金偏强 ⭐ | 衰退期持续 / 持仓 |
| 15-25 | 30-70% | ⚪ 平衡 | 中性 |
| 10-15 | 10-30% | 🟢 油价偏强 | 经济繁荣 / 加仓黄金机会 |
| < 10 | < 10% | 🟢🟢 油价严重过热 | 加仓黄金 |

### 3.3 当前位置

```
金油比 25(分位 75%)→ 🟡 黄金偏强
                            ↓
                  含义:经济衰退/萧条期延续
                  行动:持仓但警惕短期回撤
                  危险信号:若 > 35 应主动减仓
```

### 3.4 历史关键时点

| 时间 | 金油比 | 经济背景 | 黄金 12 月后 |
|------|-------:|---------|-----------|
| 2008-12(危机)| 23 | 雷曼后避险 | +27% |
| 2016-02(油价崩)| 44 | 油价跌破 30 | -13%(回吐) |
| 2020-04(疫情)| 89 | 负油价 | -2% |
| 2024-12 | 25 | 萧条期延续 | (待观察) |
| **2026-05** | **25** | 萧条期 + AI 革命 | (当前) |

## 四、指标 ③ · 金银比

### 4.1 公式与含义

```
金银比 = 黄金价格(美元/盎司) / 白银价格(美元/盎司)
```

**经济含义**:**避险货币 vs 工业金属**的相对价值。
- 金银比高 = 黄金相对工业属性偏强 = 避险情绪强
- 金银比低 = 白银工业需求强 = 经济繁荣

### 4.2 分位表(2000-2024)

| 当前金银比 | 历史分位 | 信号 | 含义 |
|-----------|---------|------|------|
| > 100 | > 95% | 🔴 黄金严重过热 | 减仓黄金 |
| 85-100 | 75-95% | 🟡 黄金偏强 ⭐ | 持仓 |
| 65-85 | 30-75% | ⚪ 平衡 | 中性 |
| 50-65 | 10-30% | 🟢 白银偏强 | 加仓黄金/白银 |
| < 50 | < 10% | 🟢🟢 经济繁荣 | 加仓黄金机会 |

### 4.3 当前位置

```
金银比 88(分位 80%)→ 🟡 黄金偏强
                            ↓
                  含义:避险情绪偏强,工业需求弱
                  行动:持仓但避免短期追高
                  考虑:若 > 100 应主动减仓
```

### 4.4 历史关键时点

| 时间 | 金银比 | 背景 | 后续 |
|------|-------:|------|------|
| 2011-04(白银暴涨)| 32 | 白银投机 | 金银比飙至 60+ |
| 2020-04(疫情)| 124 | 极端避险 | 白银爆涨,金银比 65 |
| 2024-09 | 92 | AI + 萧条 | 持续 |
| **2026-05** | **88** | 萧条 + AI | (当前) |

### 4.5 金银对冲组合

> 如果金银比 > 95% 分位,可考虑"卖金买银"对冲:
> - 卖出黄金 5%
> - 买入白银 5%(白银 ETF 或实物)
> - 当金银比回归到 70 附近(中位)时,回换

SPDR Gold Shares(GLD)是**全球最大黄金 ETF**,持仓量是机构资金对黄金态度的风向标。

## 八、组合分位的"3 红 1 绿"反转规则

> **逆向投资纪律**:当三个指标进入红色(过热),一个指标仍在绿色,就是一次黄金"短期顶"信号。

| 情景 | 实际利率 | 金油比 | 金银比 | SPDR | 行动 |
|------|---------|--------|--------|------|------|
| 三红一绿 | 🔴(>+2)| 🔴(>35)| 🔴(>100)| 🟢(回流) | 减仓战术仓 |
| 三绿一红 | 🟢(<-1)| 🟢(<15)| 🟢(<60)| 🔴(撤离) | 加仓抄底 ⭐ |
| 全绿 | 🟢🟢 | 🟢 | 🟢 | 🟢 | 共振加仓 ⭐⭐ |
| 全红 | 🔴 | 🔴 | 🔴 | 🔴 | 共振减仓 ⚠️ |

> "**多空相对一致 = 加减仓机会;多空背离 = 持仓观望**"

### 9.1 顶部分位仪表盘

```python
# tabs/gold_analysis.py 顶部 banner
from gold_metrics import compute_percentile

metrics = {
    "实际利率": -1.2,
    "金油比": 25.0,
    "金银比": 88.0,
    "SPDR 持仓": 870.0,
}

cols = st.columns(4)
for i, (name, value) in enumerate(metrics.items()):
    pct = compute_percentile(name, value)  # 计算 20Y 分位

```

```python
# .tools/db/fetch_gold_ratios.py
def compute_ratios():
    gold = read("gold.price_history")['lbma_pm']
    silver = read("gold.silver_price")['lbma_pm']
    oil = read("gold.oil_price")['wti']

    df['gold_oil'] = gold / oil
    df['gold_silver'] = gold / silver

    return df

# 每周计算分位
def compute_percentiles(metric_series, window_years=5):
    return metric_series.rolling(window_years*252).rank(pct=True)
```
