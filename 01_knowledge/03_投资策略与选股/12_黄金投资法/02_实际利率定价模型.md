# 02 · 实际利率定价模型

> 黄金的"经典锚"— **实际利率 = 名义利率 - 通胀预期**。
>
> **核心逻辑**:黄金不生息,持有黄金的机会成本 = 实际利率。
> - 实际利率 ↓ → 持有黄金机会成本 ↓ → 金价 ↑
> - 实际利率 ↑ → 持有黄金机会成本 ↑ → 金价 ↓
>
> **2022 后注意**:这套模型解释力下降但仍是**核心定量锚**。当三大范式投票全开时,实际利率是择时关键。当模型失效时,反而提示"大国博弈/技术革命范式主导"。

---

## 一、模型公式

### 1.1 核心公式

```
实际利率(Real Rate) = 名义利率(Nominal Rate) - 通胀预期(Expected Inflation)
```

### 1.2 三种实测口径

| 口径 | 公式 | 数据源 | 优缺点 |
|------|------|--------|--------|
| **A · 美债 TIPS 收益率** | 直接读取 10Y TIPS 收益率 | FRED `DFII10` | ⭐⭐⭐⭐⭐ 市场预期最权威 |
| **B · 名义 - 实际 CPI** | 10Y 名义 - 美国 CPI YoY | macro.duckdb / AkShare | ⭐⭐⭐⭐ 数据易得,有滞后 |
| **C · 名义 - SPF 通胀预期** | 10Y 名义 - 费城联储调查 | FRED `EXPINF10YR` | ⭐⭐⭐ 调查月频 |

**preson 默认采用口径 B**(数据齐全,与现有 `macro.duckdb` 集成):
```sql
SELECT
    n.date,
    n.value AS nominal_10y,
    c.cpi_yoy,
    n.value - c.cpi_yoy AS real_rate
FROM macro.cn_us_10y_yield n
JOIN macro.us_cpi_yoy c USING (date)
ORDER BY date DESC LIMIT 1;
```

## 二、实际利率与金价的历史关系

### 2.1 长周期(1971-2025)

```
┌──────────────────────────────────────────────────────────────────┐
│         实际利率(蓝)反向叠加金价(金) — 50 年视角                │
├──────────────────────────────────────────────────────────────────┤
│                                                                    │
│ 实际利率(右轴)                                       金价(左轴) │
│ 8% ──┐                                              ─── 4000 USD │
│      │                                                            │
│ 6% ──┤  ┌─70s 滞胀─┐                                              │
│      │  │ 实际利率 │  ┌─2008 危机─┐    ┌─2020 量宽─┐    ┌──2022 ─│
│ 4% ──┤  │ -3% to  │  │ 实际利率   │    │ 实际利率   │    │  反常 │
│      │  │ -8%     │  │ -2% to 0%  │    │ -1% to -2% │    │  +2%  │
│ 2% ──┤  │ 金 +1300%│  │ 金 +200%   │    │ 金 +60%    │    │  金继 │
│      │  └─────────┘  └────────────┘    └────────────┘    │  续涨 │
│ 0% ──┼────────────────────────────────────────────────────┴──────│
│      │                                                            │
│-2% ──┤                                                            │
│      │                                                            │
│-4% ──┘                                              ─── 0 USD     │
│                                                                    │
│   1971  1980  1990  2000  2010  2020  2025                        │
│                                                                    │
└──────────────────────────────────────────────────────────────────┘
```

**关键观察**:
1. **1971-1980** 实际利率 -3% 到 -8% → 金价从 35 → 850(+2300%)
2. **2008-2012** 实际利率 -2% 到 0% → 金价从 700 → 1900(+170%)
3. **2020-2021** 实际利率 -2%(2020 11 月低点)→ 金价 1500 → 2070(+38%)
4. **2022-2024 反常** 实际利率 +2% 但金价继续创新高 → **范式二/三主导**

### 2.2 散点图回归(2004-2024)

| 实际利率区间 | 金价 12 月平均涨幅 | 相关性 |
|------|-----------|--------|
| < -2% | +18% | -0.62 |
| -2% ~ 0% | +9% | -0.51 |
| 0% ~ +1% | +1% | -0.32 |
| +1% ~ +2% | -3% | -0.22 |
| > +2% | +5%(2022 后异常) | **+0.18** ⚠️ |

**核心发现**:
- 实际利率 < +1% → 模型有效,金价与实际利率强反向
- 实际利率 > +2% → **2022 后模型失效**,需查看范式二/三是否激活

## 四、与三大范式的联动

### 4.1 实际利率作为范式一的核心信号

实际利率是 [01_三大范式判定.md](01_三大范式判定.md) 范式一的 1.3 信号 —— 阈值 < 0% 即激活。

### 4.2 当模型失效(2022 反常)时如何处理?

**判定流程**:
```text
实际利率 < 0%?
    ├── YES → 模型有效 → 加仓信号(强度由实际利率绝对值决定)
    └── NO(实际利率 > +1%)→ 模型可能失效
              ↓
              检查范式二是否激活?
              ├── YES → AI 革命破信息化逻辑,黄金仍可持有
              └── NO  → 检查范式三是否激活?
                          ├── YES → 大国博弈支撑,长期持有但减仓战术仓
                          └── NO  → 黄金弱势期,降至战略下限
```

### 4.3 历史失效期回顾

| 时期 | 实际利率 | 金价表现 | 失效原因 |
|------|---------|----------|----------|
| 2022 H2 | +1.5% | -8%(下跌) | 模型有效 |
| 2023 全年 | +2.0% | +14%(上涨) | **范式三激活** — 央行增持 + 去美元化 |
| 2024 H1 | +2.2% | +21%(上涨) | **范式三 + 范式二** — AI 引发就业隐忧 |

→ **教训**:模型不能孤立使用,必须**与三大范式联合判定**

## 五、实际利率的"四象限"决策矩阵

```
                通胀高(CPI > 4%)
                       │
        ┌──────────────┼──────────────┐
        │              │              │
        │  象限 II     │  象限 I      │
        │  滞胀型      │  通胀繁荣    │
        │              │              │
        │  实际利率<0  │  实际利率<0  │
        │  名义利率低  │  名义利率高  │
        │              │              │
        │  → 黄金强力  │  → 黄金强    │
        │  ⭐⭐⭐⭐⭐ │  ⭐⭐⭐⭐    │
        │              │              │
        │  例:1970s   │  例:2021    │
名义─────┼──────────────┼──────────────┼─── 名义高(10Y>4%)
        │              │              │
        │  象限 III    │  象限 IV     │
        │  通缩衰退    │  通胀回落    │
        │              │              │
        │  实际利率>0  │  实际利率>0  │
        │  名义利率低  │  名义利率高  │
        │              │              │
        │  → 黄金中性  │  → 黄金弱    │
        │  ⭐⭐⭐      │  ⭐⭐        │
        │              │              │
        │  例:2009    │  例:2024    │
        │              │  ← 模型失效  │
        └──────────────┼──────────────┘
                       │
                通胀低(CPI < 4%)
                       │
                    通缩
```

| 象限 | 实际利率 | 名义利率 | 通胀 | 黄金强度 | 配置 |
|------|---------|----------|------|----------|------|
| I | < 0 | 高 | 高 | ⭐⭐⭐⭐ | 15-20% |
| II | < 0 | 低 | 高 | ⭐⭐⭐⭐⭐ | 20-25% |
| III | > 0 | 低 | 低 | ⭐⭐⭐ | 5-10% |
| IV | > 0 | 高 | 低 | ⭐⭐ | 0-5%(除非范式二/三激活)|

**当前(2026-05)**:实际利率 -1.2% + 名义 4.3% + CPI 5.5% → **象限 II(滞胀型)** → 黄金强度 ⭐⭐⭐⭐⭐ → 配置 20-25%

## 六、与林奇 PEG 思想的类比

| 林奇 PEG | 黄金实际利率 |
|---------|-------------|
| PEG ≤ 1 = 估值合理 | 实际利率 < 0 = 黄金合理 |
| PEG ≤ 0.5 = 严重低估 | 实际利率 < -2% = 黄金严重低估 |
| PEG > 2 = 严重高估 | 实际利率 > +2% = 黄金严重高估(失效期除外) |

**思想对齐**:**用一个简洁的数字判定"贵不贵"**,但都需要多维校验。

## 八、数据需求(Phase 2.2)

| 表 | 字段 | 来源 | 频率 | 备注 |
|----|------|------|------|------|
| `macro.us_10y_yield` | date, value | AkShare / FRED | 日 | 已有(macro.duckdb)|
| `macro.us_cpi_yoy` | date, value | AkShare `macro_usa_cpi_monthly` | 月 | **待建** |
| `gold.real_rate_history` | date, nominal, cpi, real_rate | 派生 | 日 | **Phase 2.2** |
| `gold.price_history` | date, lbma_pm, sge_au | AkShare | 日 | **Phase 2.2** |

派生脚本 `fetch_real_rate.py`:
```python
def compute_real_rate():
    nominal = duckdb.read("macro.us_10y_yield")  # 日频
    cpi = duckdb.read("macro.us_cpi_yoy")         # 月频
    # 月频 CPI forward-fill 到日频
    cpi_d = cpi.set_index("date").resample("D").ffill()
    df = nominal.merge(cpi_d, on="date")
    df["real_rate"] = df["nominal"] - df["cpi_yoy"]
    return df
```
