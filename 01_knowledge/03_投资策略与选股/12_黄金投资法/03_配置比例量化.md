# 03 · 配置比例量化

> 黄金配置的**双层框架** — 战略层(康波周期)+ 战术层(范式投票/实际利率)。
>
> **战略层**:康波四阶段决定基础配置区间(0-25%)
> **战术层**:三大范式投票 + 实际利率决定具体微调(±5%)
> **风险偏好层**:低/中/高风险偏好对配置权重的乘数
>
> **核心公式**:
> ```
> 目标黄金配置 = 战略基础(康波) × 风险偏好乘数 + 战术调整(范式 + 实际利率)
> ```

---

## 一、战略层:康波周期定基础区间

### 1.1 康波四阶段配置表

| 阶段 | 时间特征 | 主要资产 | 黄金配置区间 | 依据 |
|------|---------|----------|--------------|------|
| **回升期** | 新技术商用化 / 经济复苏 | 股票 | **5-10%** | 风险偏好回归,黄金边缘化 |
| **繁荣期** | 经济过热 / 通胀温和 | 商品 / 股票 | **0-5%** | 实际利率上行,黄金弱势 |
| **衰退期** | 危机爆发 / 流动性紧 | 现金 | **5-10%** | 危机时黄金阶段性反弹 |
| **萧条期** | 旧动能衰竭 / 新动能未起 | **黄金** | **15-25%** ⭐ | 战略价值最高 |

**当前(2026-05)**:第五次康波萧条期中后段 → 战略基础区间 **15-25%**

### 1.2 康波切换的过渡逻辑

```
萧条期 → 回升期(关键切换)
                            ↓
              判定标志(三选二):
              ① AI 商用化全面落地(类 2003 互联网)
              ② 第一批新动能产业进入盈利周期(光伏/储能/生物)
              ③ 实际利率连续 2 年正常化(0-2%)
                            ↓
              确认切换 → 黄金战略权重降至 5-10%
```

预计切换时间:**2030-2035**(周金涛预测)

## 二、战术层:范式投票 + 实际利率微调

### 2.1 战术微调矩阵(±5% 战术仓)

| 三大范式投票 | 实际利率 | 战术调整 | 实际效果 |
|-------------|----------|----------|----------|
| 3 个全部激活 | < -1% | **+5%** | 战略上限突破 |
| 2 个激活 | < 0 | **+3%** | 偏向战略上限 |
| 1 个激活 | -1% ~ 1% | **0%** | 战略中位 |
| 0 个激活 | > 1% | **-3%** | 偏向战略下限 |
| 0 个激活 | > 2% | **-5%** | 战略下限突破 |

### 2.2 当前战术调整(2026-05)

- 三大范式:3/3 全部激活 ✅
- 实际利率:-1.2%(< -1%)✅
- → **战术调整 +5%**

→ **当前总配置 = 战略中位(20%)+ 战术 (+5%) = 25%**

```

### 3.2 鲁政委结论的颠覆性

| 传统建议 | 鲁政委回测 | 颠覆点 |
|---------|-----------|--------|
| 高风险组合黄金 0-5% | 高风险组合黄金 38% | **7-8 倍差距** |
| 黄金主要为防御 | 黄金对冲股票波动 | 黄金是**主动配置工具**,不只是防御 |
| 牛市黄金减仓 | 牛市黄金加仓 | 因为**股票占比上升 → 对冲需求上升** |

### 3.3 推导背后逻辑

**为什么高风险组合需要更多黄金?**

```text
┌─────────────────────────────────────────┐
│ 股票:高夏普但高回撤(中等-高方差)         │
│ 债券:中夏普低回撤(低方差)                │
│ 黄金:中夏普高回撤(中等方差,与股低相关)  │
└─────────────────────┬───────────────────┘
                      │
                      ▼
       股票占比 ↑ → 组合方差由股票主导
                      │
                      ▼
       此时配黄金:夏普稍降,但**回撤大幅降**
                      │
                      ▼
       夏普 = (收益 - 无风险)/方差
       方差降幅 > 收益降幅 → 夏普反而升 ⭐
```

→ **结论**:黄金不是"低风险资产",而是"低相关性资产"。在权益占比高时,加黄金能**通过相关性对冲降方差**。

## 四、综合配置公式

### 4.1 三层联动公式

```text
┌──────────────────────────────────────────────────┐
│ 第一步:确定康波阶段 → 战略基础区间                │
│   萧条期:15-25% / 衰退期:5-10% / 回升期:5-10% / 繁荣期:0-5%  │
│                                                  │
│ 第二步:风险偏好乘数                              │
│   低风险偏好(股票 < 10%):基础区间 × 0.1     │
│   中风险偏好(股票 20-40%):基础区间 × 0.7    │
│   高风险偏好(股票 ≥ 50%):基础区间 × 1.6 ⭐ │
│                                                  │
│ 第三步:战术微调(范式 + 实际利率)                │
│   全开 + 实际利率 < -1%:+5%                  │
│   2 激活 + 实际利率 < 0:+3%                  │
│   钝化 + 实际利率 > 2%:-5%                   │
│                                                  │
│ 最终配置 = (基础 × 乘数)+ 战术调整              │
└──────────────────────────────────────────────────┘
```

### 4.2 计算示例(高风险偏好,2026-05)

```python
# 当前康波萧条期 → 战略基础 15-25% → 取中位 20%
# 高风险偏好(股票 55%)→ 乘数 1.6 → 20% × 1.6 = 32%
#   注:乘数封顶在鲁政委 38% 上限
# 三范式全开 + 实际利率 -1.2% → 战术 +5%
# → 最终配置:32% + 5% = 37%(接近 38% 上限)

# 低风险偏好(股票 4%)→ 乘数 0.1
# 20% × 0.1 = 2% + 战术 0% = 2%(对应鲁政委 1-2% 区间)

# 中风险偏好(股票 20%)→ 乘数 0.7
# 20% × 0.7 = 14% + 战术 +3% = 17%(对应鲁政委 10-17% 区间)
```

→ 三个示例分别对应鲁政委回测结果的中段:**高 38% / 中 17% / 低 2%** ✅

## 五、再平衡机制(执行层)

### 5.1 再平衡频率与触发

| 触发类型 | 频率 | 阈值 | 行动 |
|----------|------|------|------|
| **定期再平衡** | 季度末 | — | 调回目标比例 ±2% 范围 |
| **偏离再平衡** | 任意 | 偏离 > ±5% | 立即调回目标 ±2% |
| **范式切换** | 任意 | 三大范式投票变化 | 重新计算目标 |
| **康波切换** | 多年 | 进入新阶段 | 战略基础区间切换 |

### 5.2 再平衡示例

**情景**:目标黄金 20%,股票牛市后股票升至 60%,黄金降至 15%。

```text
当前实际:股 60% / 债 25% / 黄金 15%
目标比例:股 55% / 债 25% / 黄金 20%
偏离度:股 +5% / 黄金 -5%

行动:卖出股票 5pp,买入黄金 5pp,回到目标
触发条件:股票偏离 +5% 已达阈值,必须再平衡
```

### 5.3 战术加仓 vs 战略再平衡

| 类型 | 触发 | 持续时间 | 仓位变化 |
|------|------|----------|----------|
| **战略再平衡** | 偏离阈值 / 季度末 | 永久 | 调回目标 |
| **战术加仓** | 范式切换 / 实际利率突变 | 1-3 季 | 临时 +5% |
| **事件性加仓** | 地缘冲突 / 危机爆发 | 1-3 月 | 临时 +5-10% |

**纪律提示**:战术 + 事件性加仓不超过总仓位的 **+10%**;超过 +10% 必须降回战略目标。

## 六、与组合其他资产的关系

### 6.1 黄金与股债的相关性

| 资产对 | 长期相关性(10Y) | 危机时(回撤期) |
|--------|------------------|------------------|
| 黄金 vs S&P 500 | -0.05 ~ +0.15(几乎无相关) | -0.30 ~ -0.50(强反向) |
| 黄金 vs 10Y 美债 | +0.20 ~ +0.40(弱正相关) | +0.40 ~ +0.60(同涨) |
| 黄金 vs 美元指数 | -0.40 ~ -0.60(传统反向) | 2022 后趋于 0 |

→ **黄金是组合的"低相关性 + 危机对冲"角色**,不是单纯的债券替代。

## 八、风险提示

| 风险 | 说明 | 应对 |
|------|------|------|
| **回撤风险** | 黄金可短期回撤 20%+(2013) | 长期持有,不被短期波动吓到 |
| **无息资产** | 不产生现金流 | 配置占比 ≤ 风险偏好乘数 × 战略基础 |
| **政策风险** | 中国对实物金交易有限制 | 优先 ETF |
| **基差风险** | ETF 跟踪误差 0.1-0.3% | 选 518880 等流动性好的 |
| **货币风险** | 国内 ETF 是人民币计价 | 不对冲美元波动,需理解 |
