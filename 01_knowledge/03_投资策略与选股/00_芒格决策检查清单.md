# 查理·芒格投资决策检查清单

> **原则**：在任何投资决策前使用本清单。这不是简单的打勾，而是深度反思过程。

---

## 📋 核心原则检查

### 1️⃣ 能力圈 - 我真的理解这个生意吗？

- [ ] **业务模式清晰** - 公司如何赚钱，能用一句话解释吗？
- [ ] **竞争优势明显** - 有什么别人模仿不了的？（品牌/技术/成本/网络效应）
- [ ] **持续多久** - 这个优势能持续 5 年以上吗？
- [ ] **管理层我信任** - 管理层过往记录如何？有没有诚信问题？

**📌 如果无法清晰回答上述问题，PASS。**

---

### 2️⃣ 反向思维 - 这家公司会怎样失败？

- [ ] **行业风险** - 行业是否在衰退？有颠覆性风险吗？
- [ ] **竞争加剧** - 竞争对手在蚕食市场份额吗？
- [ ] **管理层问题** - 管理层有更换风险吗？创始人依赖度太高吗？
- [ ] **财务风险** - 负债率是否过高？现金流是否恶化？
- [ ] **法律与监管** - 有政策风险吗？行业是否面临新监管？

**危险信号**：
- ⚠️ 最近几年营收或利润下滑
- ⚠️ 管理层频繁更换
- ⚠️ 债务快速增长
- ⚠️ 行业景气度下行

**📌 如果有 3 个以上危险信号，PASS。**

---

### 3️⃣ 多元思维 - 我从多个角度考虑了吗？

使用不同学科的思维方式分析：

| 学科 | 关键问题 | 检查 |
|------|---------|------|
| **经济学** | 市场规模有多大？增速多快？ | ☐ |
| **心理学** | 人们为什么需要这个产品？会一直需要吗？ | ☐ |
| **历史** | 这类企业过去的表现如何？有类似案例吗？ | ☐ |
| **数学** | 利润增长率现实吗？不会涉及会计技巧吧？ | ☐ |
| **生物学** | 公司是否在适应环境变化（进化）？ | ☐ |
| **系统论** | 公司在整个生态系统中的地位如何？ | ☐ |

**📌 至少从 3 个角度确认后再决策。**

---

### 4️⃣ 安全边际 - 价格足够便宜吗？

**格雷厄姆式安全边际：**

- [ ] **内在价值计算**
  - 历史平均 PE：________
  - 行业平均 PE：________
  - 合理 PE 范围：________ 至 ________
  - 当前 PE：________
  - **PE 溢价**：是否超过 20%？☐ 是  ☐ 否

- [ ] **PB 检查**
  - 历史平均 PB：________
  - 行业平均 PB：________
  - 当前 PB：________

- [ ] **股息率检查**
  - 当前股息率：________%
  - 是否高于债券收益率（5%）？☐ 是  ☐ 否

**📌 安全边际清单：**
- ✅ PE 低于历史中位数 20% 以上
- ✅ 或者 PB 低于 1.5 倍
- ✅ 或者股息率高于 3%
- **⚠️ 至少满足其中一个条件**

### 5️⃣ 长期价值创造 - 3年后这个生意会怎样？

- [ ] **收入增长**
  - 过去 3 年复合增速：________%
  - 预期未来 3 年增速：________%
  - 增速是否在放缓？☐ 是  ☐ 否

- [ ] **利润率**
  - 过去 3 年毛利率：________ → ________
  - 过去 3 年净利率：________ → ________
  - 趋势是改善还是恶化？☐ 改善  ☐ 恶化  ☐ 平稳

- [ ] **市场地位**
  - 市场份额在增长吗？
  - ROE/ROIC 是否高于 15%？☐ 是  ☐ 否
  - 相对竞争对手的地位在改善吗？

**📌 3年后，公司应该是更强而非更弱。**

---

### 6️⃣ 心理学陷阱 - 我是否陷入了常见偏差？

**自检你的决策动机：**

| 偏差 | 症状 | 自检 |
|------|------|------|
| **确认偏差** | 只看有利信息，忽视不利信息 | ☐ 是  ☐ 否 |
| **从众效应** | "这个股票很多人买，所以我也买" | ☐ 是  ☐ 否 |
| **近因偏差** | 最近涨得好，所以我买 | ☐ 是  ☐ 否 |
| **过度自信** | "我知道未来会怎样" | ☐ 是  ☐ 否 |
| **情感驱动** | 贪心或恐惧，而非理性 | ☐ 是  ☐ 否 |
| **沉没成本** | 因为之前买的亏了，所以继续拿 | ☐ 是  ☐ 否 |

**📌 如果勾了"是"，暂停。重新审视你的决策。**

---

### 7️⃣ 护城河与竞争优势 - 这家公司是否有"城堡"？

**护城河的 5 种类型：**

- [ ] **品牌护城河** - 消费者愿意溢价购买吗？（如茅台、苹果）
- [ ] **技术护城河** - 有专利或技术垄断吗？（如高端芯片、医疗设备）
- [ ] **网络效应护城河** - 用户越多价值越大吗？（如微信、支付宝）
- [ ] **成本领先** - 成本结构上有优势吗？（如海底捞的供应链）
- [ ] **转换成本** - 客户难以切换吗？（如企业软件）

**📌 至少有 1 个明显的护城河。否则易被竞争打败。**

---

### 8️⃣ 管理层质量 - 我在为谁工作？

**芒格对管理层的评价标准：**

- [ ] **诚信** - 过往有没有欺骗股东或监管部门？
- [ ] **能力** - 过往决策是否明智？产品创新能力如何？
- [ ] **对股东的态度** - 有没有大股东套现？高管薪酬是否合理？
- [ ] **资本分配能力** - 如何用现金（投资、回购、分红、并购）？
- [ ] **长期思维** - 管理层是否看 5 年以上的未来？

**危险信号：**
- ⚠️ 频繁的管理层变更
- ⚠️ 高管大量减持
- ⚠️ 薪酬与业绩不符
- ⚠️ 关联交易频繁且价格不透明

**📌 如果管理层让你不信任，PASS。**

---

### 9️⃣ 价格与价值 - 这是便宜货还是落阱？

**关键财务指标检查：**

| 指标 | 自己的估算 | 行业水平 | 历史中位 | 当前 | 评价 |
|------|---------|---------|---------|------|------|
| PE | | | | | |
| PB | | | | | |
| PEG | | | | | |
| PS | | | | | |
| 股息率 | | | | | |

**估值方法（选 2 种以上）：**

- [ ] **PE 估值法** - 目标价 = 净利润 × 合理 PE
- [ ] **PB 估值法** - 目标价 = 净资产 × 合理 PB  
- [ ] **DCF 现金流折现** - 未来 5 年现金流折现
- [ ] **相对估值** - 与同行龙头对比

**📌 用多种方法得到的目标价相近，更有说服力。**

---

### 🔟 风险评估 - 最坏情况会怎样？

**下行风险分析：**

- [ ] **乐观情景** - 公司发展顺利，目标价：¥________
- [ ] **基础情景** - 按历史增速发展，目标价：¥________
- [ ] **悲观情景** - 增速放缓或行业衰退，目标价：¥________

**关键问题：**
- [ ] 如果行业衰退 50%，公司还赚钱吗？
- [ ] 如果竞争对手推出颠覆性产品，会怎样？
- [ ] 如果政策改变了，会有多大影响？

**📌 确保即使在悲观情景下，也不会亏本。**

## 🔄 使用方式

### 1. 构建阶段
在考虑买入前填写本清单（15-30 分钟思考）

### 2. 复核阶段
每季度复核一次，确保持仓理由仍然成立

### 3. 卖出阶段
如果任何关键指标恶化（如管理层质量、护城河弱化），立即重新评估
