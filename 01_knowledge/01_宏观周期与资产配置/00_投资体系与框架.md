# 投资体系框架与纪律准则

> 我的投资哲学、体系框架与操作纪律

---

## 一、投资哲学

### 1.1 核心信条

**我的 7 条投资信念：**

1. **买股票就是买公司的一部分**  
   → 关注企业内在价值，而非股价波动

2. **安全边际是永恒的护城河**  
   → 以折扣价买入，为错误预留空间

3. **能力圈原则**  
   → 只投资自己能理解的企业

4. **市场先生是服务者而非指导者**  
   → 利用情绪波动，不被其左右

5. **时间是优质企业的朋友，劣质企业的敌人**  
   → 长期持有优质资产，避免频繁交易

6. **分散投资，但不过度分散**  
   → 5-10 个核心持仓，深入了解每个标的

7. **避免重大亏损比追求高收益更重要**  
   → 活下去，活得久，胜过一时的暴利

### 1.2 能力圈定义

**✅ 我理解的行业：**
- 消费（家电、食品饮料、医药）
- 制造（机械、汽车零部件、电子）
- 金融（银行、保险）

**🎓 学习中的行业：**
- 科技（半导体、软件）
- 能源（石油、煤炭）- 部分理解
- 周期（有色、化工）- 谨慎参与

**❌ 我不碰的领域：**
- 概念炒作股
- ST 和退市风险股
- 财务造假嫌疑股
- 过度复杂无法理解的企业
- 管理层不诚信的企业

## 二、投资体系框架

### 2.1 三层筛选体系

**第一层：筛选（排除法）**
- ✅ 连续 3 年 ROE > 10%
- ✅ 资产负债率 < 70%
- ✅ 经营现金流为正
- ✅ 无重大财务风险
- ✅ 业务可理解
- 📊 *淘汰 90% 的标的*

⬇️

**第二层：分析（定量 + 定性）**

*定量分析：*
- 盈利能力（ROE、利润率）
- 成长性（增速、持续性）
- 安全性（现金流、负债）

*定性分析：*
- 商业模式（可理解、可持续）
- 竞争优势（护城河强度）
- 管理团队（诚信、能力）

⬇️

**第三层：决策（估值 + 纪律）**

*估值分析：*
- 安全边际（溢价 20% 以内）
- 合理 PE/PB 范围
- 预期收益率（>10%）

*纪律检查：*
- 仓位控制（<10%）
- 买卖纪律（严格执行）
- 风控规则（止损明确）

### 2.2 策略配置框架

| 策略类型 | 配置比例 | 特征 | 代表标的 |
|----------|----------|------|----------|
| **深度价值** | 30-40% | 低PE/PB、高股息、稳健 | 银行、保险、基建 |
| **稳健成长** | 40-50% | 中等增速、优质ROE、行业龙头 | 消费龙头、制造业龙头 |
| **积极成长** | 10-20% | 高增速、高波动、高预期 | 科技成长、新能源 |
| **现金** | 5-10% | 等待机会、降低波动 | 货币基金、短债 |

### 2.3 仓位管理原则

**总体仓位配置**

| 市场估值 | 建议仓位 | 操作方向 |
|---------|---------|---------|
| 极低（PE<12） | 90% | 积极加仓 |
| 偏低（PE 12-14） | 70% | 标准配置 |
| 合理（PE 14-16） | 50% | 中性配置 |
| 偏高（PE 16-18） | 30% | 逐步减仓 |
| 极高（PE>18） | 20% | 大幅减仓 |

**个股仓位上限**

- 深度研究标的：最高 **10%**
- 标准研究标的：最高 **5%**
- 试探性持仓：最高 **3%**

**行业仓位限制**

- 单一行业不超过 **30%**
- 相关行业合计不超过 **40%**

## 三、买入纪律

### 3.1 买入前检查清单

**📊 定量检查（必须全部满足）**

- [ ] ROE > 10%（过去 3 年平均）
- [ ] 资产负债率 < 70%
- [ ] 经营现金流连续 3 年为正
- [ ] 营收增速 > 0% 或有合理复苏预期
- [ ] 无 ST、退市风险
- [ ] 估值处于合理或偏低区间

**🔍 定性检查（至少满足 4 项）**

- [ ] 商业模式简单可理解
- [ ] 具有明确竞争优势（护城河）
- [ ] 行业地位领先（前 3 名）
- [ ] 管理层诚信且能干
- [ ] 成长空间清晰
- [ ] 财务报表健康透明

**💰 估值检查（必须满足）**

- [ ] 留有安全边际（PE < 历史平均或行业平均）
- [ ] 预期年化收益率 > 10%
- [ ] 下行风险可控（最大回撤 < 30%）

**📈 组合检查**

- [ ] 该行业仓位 < 30%
- [ ] 买入后总仓位 < 90%
- [ ] 与现有持仓重叠度合理

**💭 心理检查**

- [ ] 不是因为股价涨了才想买
- [ ] 不是因为别人推荐才想买
- [ ] 能够接受持有 3 年以上
- [ ] 能够接受股价下跌 30% 不动摇

### 3.2 买入价位策略 - 三档建仓法

| 档位 | 仓位占比 | 估值条件 | 执行时机 |
|------|---------|---------|---------|
| **第一档（试探仓）** | 30% | 合理偏低估 | 开始建立观察仓 |
| **第二档（主力仓）** | 50% | 明显低估 | 加仓至核心持仓 |
| **第三档（重仓）** | 20% | 极端低估 | 重大机会时满仓 |

**具体示例：**  
如目标仓位为 8%，分三档建仓：

- **第一档**：2.4% 仓位 @ 合理估值偏低估
- **第二档**：4.0% 仓位 @ 明显低估
- **第三档**：1.6% 仓位 @ 极端低估

> 💡 分批建仓的优点：降低风险，平滑成本，增加灵活性

### 3.3 禁止买入情形

**❌ 绝对禁止：**

- 连续亏损且无好转迹象
- 资产负债率 > 80% 且持续恶化
- 经营现金流连续 2 年为负
- 涉嫌财务造假或诚信问题
- 股价处于历史高位（PE > 80%分位）且追高
- 借款炒股或杠杆买入

**⚠️ 谨慎买入（需特别评估）：**

- 处于强周期高点
- 过度依赖单一客户（> 50% 收入）
- 行业面临重大政策利空
- 管理层频繁变动
- 估值处于历史高位

## 四、卖出纪律

### 4.1 必须卖出情形（硬性纪律）

**📉 基本面恶化（立即卖出）**

- ❌ **买入逻辑被证伪** → 无论盈亏立即纠正
- ❌ **连续两季 ROE 大幅下滑（> 20%）** → 盈利能力恶化
- ❌ **负债率上升至危险水平（> 75%）** → 财务风险不可控
- ❌ **经营现金流转负** 且看不到改善 → 利润质量存疑
- ❌ **财务造假或重大诚信问题** → 零容忍
- ❌ **管理层重大失误** → 失去信任基础

**📈 估值过高（逐步减仓）**

- 📊 PE 升至历史 90%分位以上 → 泡沫阶段，逐步减仓
- 📊 市值达到内在价值上限 → 收割利润
- 📊 发现明显更好的替代机会 → 优化配置，换仓

### 4.2 考虑减仓情形

```
□ 基本面边际恶化
  - 毛利率连续两季下滑
  - 营收增速明显放缓
  - 竞争格局恶化

□ 估值进入偏高区间
  - PE>历史70%分位
  - 预期收益率降至<8%

□ 仓位过重需要再平衡
  - 单一标的因涨幅>15%仓位
  - 单一行业>35%仓位

□ 资金需求
  - 发现更好的机会
  - 个人资金需求
```

### 4.3 不卖出情形（避免错误）

```
□ 不因短期股价下跌而卖出（除非基本面恶化）
□ 不因市场恐慌而卖出（除非系统性风险）
□ 不因获利了结心理而过早卖出优质股
□ 不因需要资金而被迫卖出（保持适当现金）
□ 不因他人观点而动摇（除非有新的事实）
```

## 五、持有纪律

### 5.1 定期复盘制度

| 频率 | 任务 |
|------|------|
| **每日** | 浏览持仓重大公告（不超过 30 分钟）/ 关注持仓新闻 / 避免频繁看盘 |
| **每周** | 更新持仓估值数据 / 记录异常波动原因 / 检查遗漏 |
| **每月** | 深度阅读公司公告 / 更新财务数据表 / 评估持仓逻辑 |
| **每季** | 深度分析季报 / 重新审视估值 / 调整仓位 |
| **每年** | 全面复盘投资绩效 / 总结得失 / 更新体系 |

### 5.2 持有期间关注清单

```
正常持有信号：
□ 季度业绩符合或超预期
□ 估值处于合理区间
□ 行业地位稳固
□ 无重大负面事件
→ 继续持有，无需操作

需要关注信号：
△ 业绩低于预期但逻辑未变
△ 估值升至偏高区间
△ 竞争格局出现变化苗头
△ 管理层变动
→ 加强跟踪，暂不操作，准备应对预案

准备减仓信号：
⚠ 连续两季业绩低于预期
⚠ 竞争格局明显恶化
⚠ 估值升至高位且无业绩支撑
⚠ 出现替代性技术或产品
→ 制定减仓计划，执行纪律
```

## 六、风控纪律

### 6.1 风险识别清单

```
【个股风险】
□ 财务风险：负债过高、现金流差、应收过大
□ 经营风险：客户集中、技术替代、竞争恶化
□ 治理风险：大股东侵占、关联交易、诚信问题
□ 估值风险：PE过高、预期透支、情绪过热

【行业风险】
□ 政策风险：监管变化、补贴退坡、环保要求
□ 周期风险：需求波动、价格周期、库存周期
□ 技术风险：技术路线变化、颠覆性创新
□ 竞争风险：新进入者、价格战、份额流失

【市场风险】
□ 系统性风险：经济衰退、流动性收紧、地缘政治
□ 风格风险：价值/成长风格切换
□ 情绪风险：恐慌、贪婪、从众
```

### 6.2 风险控制措施

```
【事前控制】
- 深入研究，了解企业本质
- 坚持安全边际，不追高
- 分散投资，控制单一标的比例
- 保持现金储备，应对突发事件

【事中控制】
- 定期跟踪，及时发现风险信号
- 设定止损纪律（基本面止损）
- 避免杠杆，不上杠杆
- 情绪化时不做决策

【事后控制】
- 及时复盘，总结教训
- 调整能力圈边界
- 完善投资体系
```

### 6.3 极端情况应对

```
【个股暴雷】
情形：持仓公司突发重大负面
应对：
1. 24小时内收集全部信息
2. 评估影响程度和持续性
3. 若影响致命，立即清仓
4. 若影响可控，评估是否加仓摊低成本
5. 记录教训，完善研究框架

【市场系统性下跌】
情形：大盘下跌30%+，持仓普遍回撤
应对：
1. 检查持仓基本面是否恶化
2. 若基本面未变，属于情绪恐慌→加仓
3. 若杠杆或资金压力，被迫减仓
4. 避免在最低点割肉
5. 历史经验：优质资产终将恢复

【行业政策利空】
情形：持仓行业遭遇重大政策打击
应对：
1. 评估政策持续性和深度
2. 区分行业性影响与公司个体差异
3. 若行业逻辑被摧毁，果断换仓
4. 若只是短期扰动，逢低加仓
5. 关注政策受益方向，考虑调仓
```

## 七、行为纪律

### 7.1 克服认知偏差

| 偏差 | 表现 | 对策 |
|------|------|------|
| 确认偏差 | 只寻找支持观点的证据 | 强制阅读看空研报 |
| 损失厌恶 | 亏损股不愿卖，盈利股过早卖 | 设定客观卖出条件 |
| 锚定效应 | 被买入价锚定 | 关注内在价值，忽略成本 |
| 禀赋效应 | 对自己持仓估值过高 | 假设未持有，重新评估 |
| 从众心理 | 跟风买卖 | 独立思考，逆向思维 |
| 过度自信 | 高估自己判断 | 保持谦逊，概率思维 |
| 近因效应 | 被近期事件过度影响 | 拉长视角，看长期趋势 |

### 7.2 情绪管理

```
【贪婪时】
- 市场狂热，人人谈股票
- 新开户数激增
- 估值处于历史高位
→ 逐步减仓，保留胜利果实

【恐惧时】
- 市场恐慌，人人逃离
- 成交量萎缩
- 优质资产被错杀
→ 克服恐惧，逆向加仓

【无聊时】
- 市场横盘，没有明显机会
-  tempted to trade for excitement
→ 保持耐心，等待机会
→ 研究新标的，提升能力圈
```

### 7.3 交易纪律

```
□ 不在开盘30分钟内做重大决策
□ 不在情绪波动时做交易决策
□ 严格执行预设的买卖计划
□ 不临时改变既定的投资纪律
□ 交易前再次确认基本信息
□ 记录每笔交易的理由
□ 定期复盘交易决策质量
```
