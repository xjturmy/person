# 双周期动态调整框架

> 长周期（康波）定战略仓位 + 短周期（格雷厄姆指数）定战术调整
> 制定日期：2026年4月15日

---

## 三、第二层：短周期战术调整（格雷厄姆指数）

### 3.1 格雷厄姆指数（Graham Index）定义

**公式**：
```
格雷厄姆指数 = 股票盈利收益率 / 无风险利率
           = (1 / PE) / 国债收益率
           = 股票收益率倍数
```

**意义**：
- 衡量股票相对于无风险资产的吸引力
- >2：股票显著低估，应加仓权益
- 1.5-2：股票合理估值，维持配置
- <1.5：股票高估，应减仓权益
- <1：股票严重高估，应大幅减仓

### 3.2 格雷厄姆指数计算示例

```
假设当前（2026年4月）：
• 沪深300 PE = 12倍
• 10年期国债收益率 = 2.5%

计算：
股票盈利收益率 = 1/12 = 8.33%
格雷厄姆指数 = 8.33% / 2.5% = 3.33

判断：
3.33 > 2 → 股票显著低估，应加仓权益
```

### 3.3 格雷厄姆指数等级与战术调整

| 等级 | 格雷厄姆指数 | 市场状态 | 战术调整 | 权益仓位调整 |
|------|--------------|----------|----------|--------------|
| **极度低估** | >3.0 | 恐慌期 | 大幅加仓权益 | +10-15% |
| **显著低估** | 2.0-3.0 | 低迷期 | 适度加仓权益 | +5-10% |
| **合理区间** | 1.5-2.0 | 平衡期 | 维持配置 | 0% |
| **轻度高估** | 1.0-1.5 | 乐观期 | 适度减仓权益 | -5-10% |
| **极度高估** | <1.0 | 狂热期 | 大幅减仓权益 | -10-15% |

## 四、双周期叠加的实际配置计算

### 4.1 配置计算框架

```
实际配置 = 战略配置（康波）+ 战术调整（格雷厄姆指数）

约束条件：
• 战术调整幅度不超过±10%（防止过度交易）
• 进攻权益始终不低于10%（不错过反弹）
• 进攻权益始终不超过30%（萧条期上限）
• 债券类始终不低于25%（保持底线）
```

### 4.2 情景计算示例

#### 情景A：当前（2026年4月，萧条期+显著低估）

```
【战略配置（康波萧条期）】

黄金      15%
防御权益  30%
进攻权益  20%  ← 基础战略配置

【战术调整（格雷厄姆指数=3.33，显著低估）】
• 格雷厄姆指数>3.0，属于"极度低估"
• 建议加仓权益+10%
• 进攻权益：20% + 10% = 30%（触及萧条期上限）
• 债券类：35% - 10% = 25%（触及底线）

【最终实际配置】
债券类    25%  （-10%，降至底线）
黄金      15%  （维持）
防御权益  30%  （维持）
进攻权益  30%  （+10%，达到上限）
         ===
         100%
```

#### 情景B：假设2027年市场反弹后（格雷厄姆指数=1.8，合理区间）

```
【战略配置（康波萧条期）】
债券类    35%
黄金      15%
防御权益  30%
进攻权益  20%

【战术调整（格雷厄姆指数=1.8，合理区间）】
• 格雷厄姆指数1.5-2.0，属于"合理区间"
• 建议维持配置，不调整

【最终实际配置】
债券类    35%
黄金      15%
防御权益  30%
进攻权益  20%
         ===
         100%
```

#### 情景C：假设2028年市场狂热（格雷厄姆指数=0.9，极度高估）

```
【战略配置（康波萧条期）】
债券类    35%
黄金      15%
防御权益  30%
进攻权益  20%

【战术调整（格雷厄姆指数=0.9，极度高估）】
• 格雷厄姆指数<1.0，属于"极度高估"
• 建议减仓权益-15%
• 但进攻权益不能低于10%

防御权益  35%  （+5%，部分防御权益也减仓）

备注：此时虽然战略配置建议进攻权益20%，但市场极度高估，
     战术调整为10%，保留了10%的敞口不错过最后的疯狂，
     同时大幅降低风险。

## 五、动态调整操作流程

### 5.1 月度评估流程（短周期）

```
每月第一个周末执行：

Step 1：计算格雷厄姆指数
□ 查询当前沪深300 PE（理杏仁/同花顺）
□ 查询当前10年期国债收益率（中国债券信息网）
□ 计算：格雷厄姆指数 = (1/PE) / 国债收益率
□ 判断等级（极度低估/显著低估/合理/高估/极度高估）

Step 2：评估战术调整需求
□ 对比上月格雷厄姆指数变化
□ 判断是否触发调整阈值（变化>0.5才调整）
□ 计算建议调整幅度（±5-10%）

Step 3：检查约束条件
□ 进攻权益是否在10-30%区间内？
□ 债券类是否≥25%？
□ 调整幅度是否≤±10%？

Step 4：制定调整计划
□ 确定买入/卖出标的
□ 确定分批执行计划（至少分2批）
□ 设定执行时间表（本月内完成）
```

### 5.2 季度评估流程（长短周期叠加）

```
每季度末执行：

Step 1：评估长周期（康波）
□ 回顾康波周期相关数据：
  - AI商业化进展
  - 新能源产能周期
  - 可控核聚变突破
  - 地缘政治变化
□ 判断：是否接近2030年观察节点？
□ 评估：是否需要调整战略配置？

Step 2：评估短周期（格雷厄姆指数）
□ 计算本季度平均格雷厄姆指数
□ 对比上季度变化趋势
□ 判断市场估值方向

Step 3：综合调整决策
□ 长周期战略配置是否需要调整？（通常不需要，除非重大转折）
□ 短周期战术调整幅度确定
□ 最终配置比例计算

Step 4：执行再平衡
□ 卖出超出目标比例的资产
□ 买入低于目标比例的资产
□ 记录调整逻辑和预期
```

### 5.3 年度评估流程（长周期重点）

```
每年年末执行：

Step 1：深度评估康波周期位置
□ 回顾年度康波相关事件
□ 评估第六次康波技术进展：
  - AI：ChatGPT等大模型商业化数据
  - 能源：光伏成本、核聚变进展
  - 生物：基因编辑临床应用
□ 判断：距离2030年观察节点还有多远？
□ 预判：回升期可能到来的时间窗口

Step 2：回顾格雷厄姆指数年度走势
□ 绘制年度格雷厄姆指数走势图
□ 识别极端低估/高估时点
□ 评估战术调整的有效性

Step 3：优化调整参数
□ 格雷厄姆指数阈值是否需要微调？
□ 调整幅度±10%是否合适？
□ 约束条件是否需要调整？

Step 4：制定下一年战略
□ 确认长周期战略配置
□ 设定格雷厄姆指数监控目标
□ 制定再平衡触发条件
```

## 六、关键阈值与纪律

### 6.1 格雷厄姆指数监控阈值

```
┌─────────────────────────────────────────────────────────────┐
│              格雷厄姆指数监控阈值设置                          │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  【买入阈值】                                                │
│  3.0以上：极度低估，积极买入（加仓进攻权益）                 │
│  2.5-3.0：显著低估，适度买入                                 │
│                                                              │
│  【持有阈值】                                                │
│  1.5-2.5：合理区间，维持配置                                 │
│                                                              │
│  【卖出阈值】                                                │
│  1.0-1.5：轻度高估，适度减仓                                 │
│  <1.0：极度高估，大幅减仓（减仓进攻权益）                    │
│                                                              │
│  【调整幅度规则】                                            │
│  格雷厄姆指数变化>0.5：触发调整评估                          │
│  格雷厄姆指数变化>1.0：执行调整                              │
│                                                              │
│  【不行动区间】                                              │
│  格雷厄姆指数在1.5-2.0之间，且变化<0.5：不调整              │
│  （避免过度交易）                                            │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```

### 6.2 约束条件（红线）

| 约束项 | 下限 | 上限 | 理由 |
|--------|------|------|------|
| 进攻权益 | 10% | 30% | 萧条期纪律，防风险+不错过机会 |
| 债券类 | 25% | 50% | 保持底线流动性，极端恐慌时增加 |
| 黄金 | 10% | 25% | 基础保险10%，地缘风险时增至25% |
| 单季度调整 | - | ±10% | 避免过度交易，平滑调整 |
| 现金（货币基金） | 15% | - | 随时可动用的弹药 |

## 九、总结

### 核心原则

```
┌─────────────────────────────────────────────────────────────┐
│                                                              │
│   长周期（康波）定战略：                                     │
│   • 当前萧条期 → 进攻权益15-25%                              │
│   • 2030年若回升期确认 → 进攻权益35-45%                      │
