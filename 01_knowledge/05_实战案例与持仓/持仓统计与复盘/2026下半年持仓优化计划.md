# 2026下半年持仓优化计划

> 记录日期：2026-07-08  
> 定位：先确认持仓，再复盘操作，最后寻找好标的并建立观察仓。  
> 原则：所有买入、加仓、减仓、清仓动作，都必须能回到本计划和决策日志中复盘。

---

## 一、下一步总目标

2026 下半年的优化重点，不是扩大研究范围，而是把投资动作从“看到机会再判断”改成“先有计划，再等待机会”。

核心顺序：

1. 明确当前真实持仓：知道自己到底持有什么、为什么持有、仓位是否合理。
2. 总结上半年操作经验：把买对、买错、错过、冲动、犹豫都沉淀成规则。
3. 寻找值得长期跟踪的好标的：先建观察仓或观察清单，不急于重仓。
4. 等待好的时机买入：价格、基本面、仓位三者同时满足，才进入执行。

---

## 二、阶段一：现有持仓确认

### 2.1 必须先补全的持仓字段

在 [当前持仓总览.md](当前持仓总览.md) 中补全以下字段，作为下半年所有判断的起点。

| 字段 | 要求 | 目的 |
|------|------|------|
| 股票/ETF | 名称和代码必须完整 | 避免同名、港股/A股混淆 |
| 买入时间 | 至少精确到月份 | 判断买入是否来自当时的计划 |
| 数量/成本价 | 以真实账户为准 | 计算真实盈亏和仓位压力 |
| 当前价/当前值 | 每次复盘时更新 | 判断是否需要操作 |
| 占比 | 按当前市值计算 | 控制单一标的风险 |
| 收益率 | 含分红时单独备注 | 区分股价收益和分红收益 |
| 持仓理由 | 2-5 句 | 复盘时判断逻辑是否仍成立 |
| 失效条件 | 必须可观察 | 防止跌了以后临时改逻辑 |

### 2.2 每个持仓给出明确结论

每个持仓都要归入以下五类之一：

| 分类 | 含义 | 下一步动作 |
|------|------|------------|
| 核心持仓 | 长期逻辑仍强，仓位可承受波动 | 持有，按季复查 |
| 观察持仓 | 逻辑未完全证实，仓位应小 | 限制仓位，等待验证 |
| 机会持仓 | 估值或周期位置吸引，但确定性一般 | 只按计划加仓 |
| 待处理持仓 | 买入理由不清或逻辑已变化 | 重新写决策，不通过则减仓 |
| 应退出持仓 | 核心假设失效或仓位不再合理 | 制定退出节奏 |

### 2.3 仓位上限约束

在没有完成持仓确认前，不新增重仓。

建议先使用以下约束：

| 类型 | 单只仓位上限 | 组合内数量 | 说明 |
|------|--------------|------------|------|
| 核心持仓 | 8%-12% | 3-5 只 | 只给高确定性、可长期跟踪的公司 |
| 普通持仓 | 3%-6% | 5-10 只 | 有逻辑，但需要持续验证 |
| 观察仓 | 0.5%-2% | 3-8 只 | 用于逼迫自己跟踪，但不能影响心态 |
| 主题/周期仓 | 0%-5% | 按主题控制 | 防止同一产业链风险叠加 |

## 三、阶段二：上半年操作经验总结

### 3.1 复盘时间范围

复盘范围：2026-01-01 至 2026-06-30。

所有买入、加仓、减仓、清仓、开始观察但未买入的动作，都要进入复盘。

### 3.2 复盘表

| 日期 | 标的 | 动作 | 当时理由 | 结果 | 归因 | 可沉淀规则 |
|------|------|------|----------|------|------|------------|
| 待补 | 待补 | 买入/加仓/减仓/清仓/观察 | 待补 | 待补 | 判断/运气/估值/纪律 | 待补 |

### 3.3 必答问题

每一笔操作至少回答：

1. 当时是否有明确买入理由，还是因为价格上涨/下跌带来的情绪反应？
2. 是否提前写过失效条件？
3. 仓位是否和确定性匹配？
4. 如果现在重新做一次，会不会做同样的操作？
5. 盈亏主要来自基本面判断、估值变化、市场情绪，还是仓位管理？

### 3.4 上半年可能沉淀的经验类型

| 类型 | 需要总结的内容 | 转化为规则 |
|------|----------------|------------|
| 买对 | 为什么能买对，是否可复制 | 保留该判断流程 |
| 买错 | 哪个假设错了，是否提前可见 | 加入失效条件 |
| 卖早 | 是目标价错误，还是心态问题 | 调整分批卖出规则 |
| 卖晚 | 是否忽视了基本面恶化 | 增加强制减仓触发器 |
| 错过 | 是价格没到，还是研究不够 | 建立观察仓和提醒机制 |
| 冲动 | 是否没有经过决策日志 | 买入前强制冷静期 |

## 四、阶段三：好标的观察仓计划

### 4.1 候选标的来源

下半年优先从已有资料库中筛选，不盲目扩圈。

优先来源：

1. 当前真实持仓：先判断是否值得继续持有或加仓。
2. 核心公司库：新华保险、三美股份、中国中车、贵州茅台、美的集团、立讯精密、招商银行、伊利股份、恒瑞医药、宁德时代等。
3. 已有投资决策分析的公司：三美股份、中国中车、立讯精密、新华保险。
4. 候选池中基本面长期优秀、估值进入合理区间的公司。

### 4.2 观察仓准入条件

一个标的进入观察仓前，至少满足 4 条：

| 条件 | 要求 |
|------|------|
| 能力圈 | 能用自己的话说清楚公司怎么赚钱 |
| 主要矛盾 | 能写出看多和看空各 2 条 |
| 估值位置 | 当前估值不处于明显高估区，或有足够长期成长解释 |
| 财务质量 | ROE、现金流、负债结构没有明显硬伤 |
| 触发条件 | 知道什么价格/事件出现后才买 |
| 失效条件 | 知道什么情况出现后停止跟踪或退出 |

### 4.3 观察仓分层

| 层级 | 含义 | 仓位 | 复查频率 |
|------|------|------|----------|
| A类观察 | 高质量公司，主要等价格 | 1%-2% | 每月 |
| B类观察 | 逻辑有吸引力，但仍需验证 | 0.5%-1% | 每月/季报 |
| C类观察 | 周期或主题机会，波动大 | 0%-1% | 事件触发 |
| 仅观察清单 | 研究不够或价格不合适 | 0% | 季度 |

### 4.4 现有资料可优先复核的标的

| 标的 | 已有资料结论摘要 | 下半年动作 |
|------|------------------|------------|
| 中国中车 | 低估值、周期回暖，但长期成长和盈利能力一般 | 可作为低估值/高股息类候选，先复核最新估值和订单 |
| 立讯精密 | 成长性较强，ROE较好，但客户集中、毛利率低 | 可作为成长类候选，重点复核苹果依赖和汽车电子进展 |
| 三美股份 | 制冷剂周期弹性强，但价格回落风险高 | 只适合小仓位观察或周期验证，不宜因低PE直接重仓 |
| 新华保险 | 当前 Dashboard 上下文指向该公司，已有林奇/格雷厄姆分析 | 应优先形成一页最终结论：买入/观察/持有/回避 |

## 五、阶段四：买入时机与执行规则

### 5.1 买入前检查

任何买入或加仓前，必须回答：

1. 这是按计划买，还是因为短期上涨/下跌产生的冲动？
2. 买入后单只仓位是多少，是否超过上限？
3. 如果买入后继续下跌 20%，是否还愿意持有？
4. 当前买入理由是否至少包含 2 个独立证据？
5. 退出条件是否写清楚？
6. 这笔操作是否已经记录到决策日志？

### 5.2 分批买入规则

| 动作 | 条件 | 仓位 |
|------|------|------|
| 建观察仓 | 研究基本完成，但价格/业绩仍需验证 | 0.5%-2% |
| 第一次正式买入 | 价格进入计划区间，且核心逻辑未破坏 | 2%-4% |
| 第二次加仓 | 价格更便宜，或业绩验证明显增强 | +2%-3% |
| 停止加仓 | 估值不再便宜、仓位到上限、逻辑变弱 | 0 |
| 减仓 | 价格过高、逻辑变弱、仓位过重 | 分批执行 |

### 5.3 禁止动作

下半年尽量避免：

- 没写决策日志就买入。
- 因为“跌了很多”直接补仓。
- 因为“涨了很多”追入。
- 在没有确认真实仓位前新增大仓位。
- 同一行业或同一风险因子过度集中。
- 把观察仓亏损当成必须摊平的理由。

## 八、最终验收标准

到 2026 年底，本计划是否有效，用以下问题检验：

1. 当前持仓是否每一只都有明确理由和失效条件？
2. 上半年操作是否总结出至少 3 条可执行规则？
3. 是否有一份分层观察清单，而不是临时看行情选股？
4. 是否至少有 1 条真实买卖决策被完整记录并复盘？
5. 是否减少了冲动买入、无计划补仓和模糊持有？

如果以上问题能回答“是”，说明下半年的优化方向是有效的。
