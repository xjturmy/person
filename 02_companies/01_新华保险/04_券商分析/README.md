# 券商分析模块说明（新华保险）

## 1. 模块目标

用于持续跟踪券商对公司的一致预期、评级与目标价变化，辅助验证市场预期与基本面是否匹配。

## 2. 建议文件

- `01_盈利预测与一致预期.csv`
- `02_评级与目标价.csv`
- `03_预期修正与分歧跟踪.csv`
- `04_券商观点摘要.md`

## 3. 数据口径建议

### 3.1 盈利预测与一致预期

建议字段：

- 统计日期
- 机构覆盖数
- EPS 一致预期（FY1/FY2/FY3）
- 净利润一致预期（FY1/FY2/FY3）
- 收入一致预期（FY1/FY2/FY3）
- 预期标准差（或变异系数）

### 3.2 评级与目标价

建议字段：

- 报告日期
- 券商名称
- 评级（买入/增持/中性/减持/卖出）
- 当前目标价
- 前次目标价
- 调整方向（上调/下调/维持）
- 主要理由

### 3.3 预期修正与分歧跟踪

建议字段：

- 统计日期
- 近30天/90天 EPS 上修家数
- 近30天/90天 EPS 下修家数
- 一致预期变化幅度
- 评级净上调比例
- 预期分歧度

## 4. 更新频率

- 月度更新：一致预期、分歧度
- 周度更新：目标价和评级变化
- 事件更新：业绩预告、财报披露、重大政策后 48 小时内更新

## 5. 使用建议

- 与 `01_基本面数据/01_估值分析/` 联动：预期上修但估值分位低，通常具备性价比。
- 与 `22_成长性分析/` 联动：预期修正方向应与收入/利润趋势一致。
- 与 `50_策略分歧审视与综合决策.md` 联动：将券商分歧作为风险或机会证据。
