## 六、总结

### 6.1 投资结论

**立讯精密投资决策：**

| 维度 | 结论 |
|-----|------|
| **公司类型** | 快速增长型 + 稳健增长型 |
| **当前阶段** | 稳健增长期（增速15-20%） |
| **投资评级** | 适度看涨 |
| **目标仓位** | 3-5%（当前可建仓），5-8%（42元以下） |
| **核心风险** | 客户集中（苹果70%）、毛利率低 |
| **核心机会** | 极致ROE、汽车电子放量 |

### 6.2 关键成功因素

投资立讯精密成功的关键在于：

1. **跟踪大客户订单**
   - 密切关注iPhone出货量预测
   - 跟踪苹果供应链调整传闻
   - 关注苹果财报中的指引

2. **关注新业务进展**
   - 汽车电子订单和客户拓展
   - MR/VR等新品类进展
   - 安卓客户拓展情况

3. **监控盈利能力**
   - ROE是否维持在15%以上
   - 毛利率是否稳定在10%以上
   - 费用率控制情况

4. **保持纪律性**
   - 42元以下加仓，60元以上减仓
   - 设定明确的买卖纪律
   - 定期复盘修正
