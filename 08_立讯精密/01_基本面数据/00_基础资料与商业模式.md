## 四、战略挑战与风险（Strategic Challenges）

### 4.1 战略挑战矩阵

| 挑战类型 | 具体描述 | 影响程度 | 应对策略 |
|---------|---------|---------|---------|
| **客户集中** | 苹果收入占比70%+，依赖过重 | 高 | 拓展安卓+汽车电子 |
| **毛利率下滑** | 从20%+降至10%，盈利能力承压 | 高 | 提升自动化+高毛利业务 |
| **资本开支大** | 重资产模式，需持续投资 | 中 | 优化投资节奏 |
| **竞争加剧** | 精密制造行业竞争者增多 | 中 | 强化客户关系 |
| **地缘政治** | 苹果供应链分散化风险 | 中 | 海外产能布局 |

### 4.2 商业模式风险点

**1. 客户集中风险**
```
现状：苹果收入占比超过70%
风险：苹果订单波动或转移将重创业绩
案例：2021年苹果砍单导致立讯业绩下滑
应对：拓展安卓客户、汽车电子、自主品牌
```

**2. 毛利率下滑风险**
```
现状：毛利率从20%+降至10%，持续下滑
风险：盈利能力持续恶化
原因：低毛利组装业务占比提升
应对：提升自动化、拓展高毛利业务
```

**3. 技术替代风险**
```
现状：依赖现有精密制造技术
风险：新技术（如无线连接）可能减少连接器需求
应对：持续研发投入，跟踪技术趋势
```
